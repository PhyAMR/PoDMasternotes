This chapter introduces the R language and the small set of facilities that the rest of the book relies on: vectors and data frames, basic plotting, sampling from probability distributions, simple Monte-Carlo estimates, and a first look at the workflow we follow when running a Bayesian analysis. From Chapter 2 onward, code is hidden in the body and reproduced in the Appendix: Source code at the back of the book.

Remark. Why R. Built-in statistical functions, excellent plotting (base, ggplot2), and first-class Bayesian packages (rstan, rjags, brms, coda) make R the default language of the course. Everything in this chapter runs on a vanilla R installation; the Bayesian packages are introduced in their own chapters.

2.4 Probability distributions

Every standard distribution in R follows the d / p / q / r convention:

  • d…(x, …) — density (PDF) or mass function (PMF)
  • p…(q, …) — probability \(\Pr(X \le q)\) (CDF)
  • q…(p, …) — quantile (inverse CDF)
  • r…(n, …) — generate n random draws
Show code 
1dnorm(0)
2pnorm(1.96)
3qnorm(0.975)
4set.seed(1)
5rnorm(5, mean = 10, sd = 2)
1
Density of the standard normal at 0; equals \(1/\sqrt{2\pi}\approx 0.3989\).
2
The CDF at \(1.96\) — the familiar 97.5%-point of the standard normal.
3
The inverse of (2): \(z_{0.975} = 1.96\).
4
Always set a seed before sampling so the analysis is reproducible.
5
Draw five values from \(\mathcal{N}(10, 2^2)\). 
[1] 0.3989423
[1] 0.9750021
[1] 1.959964
[1]  8.747092 10.367287  8.328743 13.190562 10.659016

The same convention works for discrete laws: dbinom, dpois, dgeom, dnbinom, dhyper, dmultinom. For these, d returns the probability mass; otherwise the semantics are identical.

2.6 A first Monte-Carlo estimate

Monte-Carlo replaces a deterministic integral with the average of a function over a random sample. Here we estimate \(\mathbb{E}[X^2]\) for \(X \sim \mathcal{N}(0,1)\), which we know equals 1.

Show code 
1set.seed(42)
2N <- 10000
3x <- rnorm(N)
4mean(x^2)
5sd(x^2) / sqrt(N)
1
Reproducibility.
2
Number of draws.
3
Sample \(X_1, \ldots, X_N \sim \mathcal{N}(0,1)\).
4
The Monte-Carlo estimator \(\hat{\mathbb{E}}[X^2] = \frac{1}{N}\sum_i X_i^2\). It should be close to 1.
5
The standard error of the estimator — how much we trust the result. 
[1] 1.012334
[1] 0.0142808

The estimator converges at rate \(1/\sqrt{N}\) regardless of dimension; this is the property that makes MC the workhorse of Bayesian computation (Chapter 7).

2.7 A first Bayesian update

The simplest update we will meet in the book is the Beta–Binomial model: if the prior on the success probability is \(\theta \sim \text{Beta}(\alpha, \beta)\) and we observe \(k\) successes out of \(n\) trials, the posterior is \(\theta \mid k \sim \text{Beta}(\alpha+k,\, \beta+n-k)\).

Show code 
1alpha <- 2
2beta_param <- 2
3n <- 10; k <- 7
theta <- seq(0, 1, length.out = 400)
prior     <- dbeta(theta, alpha,     beta_param)
4posterior <- dbeta(theta, alpha + k, beta_param + n - k)

plot(theta, prior,     type = "l", lwd = 1, lty = 2,
     xlab = expression(theta), ylab = "density",
     ylim = c(0, max(posterior) * 1.05))
5lines(theta, posterior, lwd = 1.2, col = "#b8123e")
legend("topleft", c("Prior Beta(2,2)", "Posterior Beta(9,5)"),
       lty = c(2, 1), col = c("black", "#b8123e"),
       bty = "n", cex = 0.8)
1
The prior shape parameter \(\alpha\). Beta(2, 2) is a mild prior centred at 1/2.
2
The prior shape parameter \(\beta\) (renamed to avoid clashing with R’s beta function).
3
The data: 7 successes out of 10 trials.
4
The posterior is also Beta — this is conjugacy (formally introduced in Chapter 4).
5
Plot the two densities together; the data have shifted the mass to the right and concentrated it.

## A first Monte-Carlo estimate

Monte-Carlo replaces a deterministic integral with the average of a function over a random sample. Here we estimate $\mathbb{E}[X^2]$ for $X \sim \mathcal{N}(0,1)$, which we know equals 1.

::: {.example name="Monte-Carlo estimate of a Gaussian second moment"}

```{r}
set.seed(42)                       # <1>
N <- 10000                         # <2>
x <- rnorm(N)                      # <3>
mean(x^2)                          # <4>
sd(x^2) / sqrt(N)                  # <5>
```

1. Reproducibility.
2. Number of draws.
3. Sample $X_1, \ldots, X_N \sim \mathcal{N}(0,1)$.
4. The Monte-Carlo estimator $\hat{\mathbb{E}}[X^2] = \frac{1}{N}\sum_i X_i^2$. It should be close to 1.
5. The standard error of the estimator — how much we trust the result.

The estimator converges at rate $1/\sqrt{N}$ regardless of dimension; this is the property that makes MC the workhorse of Bayesian computation (Chapter 7).

:::

## A first Bayesian update

The simplest update we will meet in the book is the *Beta–Binomial* model: if the prior on the success probability is $\theta \sim \text{Beta}(\alpha, \beta)$ and we observe $k$ successes out of $n$ trials, the posterior is $\theta \mid k \sim \text{Beta}(\alpha+k,\, \beta+n-k)$.

::: {.example name="Beta–Binomial conjugate update"}

```{r}
#| fig-width: 4.5
#| fig-height: 3.0
#| fig-cap: "Beta prior and posterior for the success probability after observing 7 successes in 10 trials."
alpha <- 2                                     # <1>
beta_param <- 2                                # <2>
n <- 10; k <- 7                                # <3>
theta <- seq(0, 1, length.out = 400)
prior     <- dbeta(theta, alpha,     beta_param)
posterior <- dbeta(theta, alpha + k, beta_param + n - k)  # <4>

plot(theta, prior,     type = "l", lwd = 1, lty = 2,
     xlab = expression(theta), ylab = "density",
     ylim = c(0, max(posterior) * 1.05))
lines(theta, posterior, lwd = 1.2, col = "#b8123e")       # <5>
legend("topleft", c("Prior Beta(2,2)", "Posterior Beta(9,5)"),
       lty = c(2, 1), col = c("black", "#b8123e"),
       bty = "n", cex = 0.8)
```

1. The prior shape parameter $\alpha$. Beta(2, 2) is a mild prior centred at 1/2.
2. The prior shape parameter $\beta$ (renamed to avoid clashing with R's `beta` function).
3. The data: 7 successes out of 10 trials.
4. The posterior is also Beta — this is **conjugacy** (formally introduced in Chapter 4).
5. Plot the two densities together; the data have shifted the mass to the right and concentrated it.

:::
