3 Gaussian integrals

3.1 The simplest form

Let’s consider the Gaussian function

\[\begin{equation} p(x)=ce^{-\frac{a}{2}x^2}, a\in\mathbb{R}^+ \end{equation}\]

If \(p(x)\) is a pdf (probability density function) then it must necessarily be normalized over the entire interval \(\mathbb{R}\):

\[\begin{equation*} \int_{-\infty}^{+\infty}p(x)dx=1 \ \Longrightarrow \ c\int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2}dx=1 \end{equation*}\]

from which the value of \(c\) is obtained.

The integral has as its solution

\[\begin{equation} \boxed{\int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2}dx=\sqrt{\frac{2\pi}{a}}} \end{equation}\]

where \(a\) - let’s remember - must necessarily be real and positive.

3.2 The most general form

More generally, the Gaussian integral can be written as

\[\begin{equation*} \int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2+bx}dx \end{equation*}\]

To solve this integral it is useful to perform a change of variable starting from the maximum point of the function in the exponent, that is

\[\begin{equation*} \frac{d}{dx}\left(-\frac{a}{2}x^2+bx\right)=-ax+b=0 \ \Longrightarrow \ x=\frac{b}{a} \end{equation*}\]

Setting

\[\begin{equation*} y=x-\frac{b}{a} \end{equation*}\]

the exponent is written as

\[\begin{align*} -\frac{a}{2}x^2+bx&=-\frac{a}{2}\left(y^2+\frac{2b}{a}y+\frac{b^2}{a^2}\right)+by+\frac{b^2}{a}=\\ &=-\frac{a}{2}y^2-\cancel{by}-\frac{b^2}{2a}+\cancel{by}+\frac{b^2}{a}=-\frac{a}{2}y^2+\frac{b^2}{2a} \end{align*}\]

Rewriting the integral as a function of this new variable, we have

\[\begin{equation} \int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2+bx}dx=\int_{-\infty}^{+\infty}e^{-\frac{a}{2}y^2+\frac{b^2}{2a}}dy=\sqrt{\frac{2\pi}{a}}e^{\frac{b^2}{2a}} \end{equation}\]

In this formula, \(b\) can more generally belong to \(\mathbb{C}\).

3.2.1 Hubbard-Stratonovich transformation

Equation (eq:integrali-gaussiani-forma-generale) is often rewritten in the following way:

\[\begin{equation} \boxed{e^{\frac{b^2}{2a}}=\sqrt{\frac{a}{2\pi}}\int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2+bx}dx} \end{equation}\]

This is called the Hubbard-Stratonovich transformation and is useful as it allows the passage from a quadratic expression in \(b\) to a linear expression in \(b\). It is mainly used when \(b\) is a mathematical object such as a matrix, a field or a set of vectors, of which it would be difficult to calculate the square and it is therefore preferred to rewrite it in integral form, keeping \(b\) in linear form.

3.2.2 Calculation of the integral in the case \(b\in\mathbb{C}\)

Let’s assume that \(b=it\), with \(t\in\mathbb{R}\). The integral to be calculated becomes

\[\begin{equation} \varphi(t)=\sqrt{\frac{a}{2\pi}}\int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2}e^{itx}dx \end{equation}\]

Interpreting the variable \(t\) as temporal, the equation just written is none other than the Fourier transform of the Gaussian function (equation (eq:integrali-gaussiani-pdf-gaussiana)), which - as we will see later - is linked to the so-called characteristic function.

In order to calculate this integral, it is convenient to make the derivative of \(\varphi(t)\) explicit. Since this is in the form \(\int f(x,t)dx\), it is useful to recall the conditions that allow the derivation of \(\varphi(t)\) through that of \(f(x,t)\):

\[\begin{equation*} \frac{d}{dt}\int f(x,t)dx=\int\frac{\partial f(x,t)}{\partial t}dx \ \iff \ \begin{cases} f \ \text{and} \ \frac{\partial f}{\partial t} \ \text{continuous}\\ f \ \text{and} \ \frac{\partial f}{\partial t} \ \text{uniformly distributed} \end{cases} \end{equation*}\] \end{equation*}

The second condition, in particular, translates mathematically into

\[\begin{equation*} \begin{cases} \abs{f(x,t)}<A(x)\\ \abs{\frac{\partial f(x,t)}{\partial t}}<B(x) \end{cases} \ \ \ \text{where} \ \begin{cases} A(x)<\infty\\ B(x)<\infty \end{cases} \end{equation*}\] \end{equation*}

Assuming these conditions are valid, it will result

\[\begin{equation*} \varphi'(t)=i\sqrt{\frac{a}{2\pi}}\int_{-\infty}^{+\infty}xe^{-\frac{a}{2}x^2}e^{itx}dx \end{equation*}\]

By multiplying and dividing by \(-a\) we can write the integrand as a function of the derivative of the first exponent:

\[\begin{equation*} \varphi'(t)=-\frac{i}{a}\sqrt{\frac{a}{2\pi}}\int_{-\infty}^{+\infty}(-ax)e^{-\frac{a}{2}x^2}e^{itx}dx=-\frac{i}{\sqrt{2\pi a}}\int_{-\infty}^{+\infty}\frac{d}{dx}\left(e^{-\frac{a}{2}x^2}\right)e^{itx}dx \end{equation*}\]

It is natural, at this point, to proceed by integration by parts, where the term to be integrated is clearly the derivative of the exponential:

\[\begin{equation*} \varphi'(t)=-\frac{i}{\sqrt{2\pi a}}\left[e^{-\frac{a}{2}x^2}e^{itx}\bigg|_{-\infty}^{+\infty}-\int_{-\infty}^{+\infty}ite^{-\frac{a}{2}x^2}e^{ixt}dx\right] \end{equation*}\]

The first term is negligible as the dominant exponential, that is \(e^{-\frac{a}{2}x^2}\), tends to zero for \(x\to\pm\infty\). What remains is

\[\begin{equation*} \varphi'(t)=-\frac{t}{\sqrt{2\pi a}}\int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2}e^{ixt}dx \end{equation*}\]

Multiplying and dividing by \(\sqrt{a}\) follows:

\[\begin{equation*} \varphi'(t)=-\frac{t}{a}\cdot\sqrt{\frac{a}{2\pi}}\int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2}e^{ixt}dx=-\frac{t}{a}\varphi(t) \end{equation*}\]

Calculating the integral (eq:integrali-gaussiani-caso-complesso) therefore means solving the differential equation just written:

\[\begin{equation*} \frac{d\varphi}{dt}=-\frac{t}{a}\varphi \ \Longrightarrow \ \int\frac{d\varphi}{\varphi}=-\int\frac{t}{a}dt \ \Longrightarrow \ \log\varphi=-\frac{t^2}{2a}+c \ \Longrightarrow \ \varphi(t)=ke^{-\frac{t^2}{2a}} \end{equation*}\]

To set a boundary condition, it is useful to observe equation (eq:integrali-gaussiani-caso-complesso): we note that by setting \(t=0\) we find exactly the normalized Gaussian integral written in its simplest form, which therefore has a value equal to \(1\). We will therefore set \(\varphi(0)=1\), from which it follows that \(k=1\). We will thus have

\[\begin{equation} \boxed{\varphi(t)=e^{-\frac{t^2}{2a}}} \end{equation}\]

3.3 Characteristic functions

From the moment a continuous random variable \(x\) and a pdf function of this variable \(p(x)\) are introduced, it is possible to define a new function, called the characteristic function, very similar to the Fourier transform of \(p(x)\):

\[\begin{equation} \boxed{\varphi(k)=\int e^{ikx}p(x)dx=\left\langle e^{ikx}\right\rangle_p} \end{equation}\]

This is a useful tool as it allows us to derive all the properties of \(p(x)\) and is at the same time simpler to study.

A first important property that can be derived is the mean \(\langle x\rangle\). By deriving \(\varphi(k)\), in fact, we find

\[\begin{equation*} \frac{d\varphi}{dk}=\int ixe^{ikx}p(x)dx \ \Longrightarrow \ -i\frac{d\varphi}{dk}=\int xe^{ikx}p(x)dx \end{equation*}\]

By setting \(k=0\) we find precisely the mean of the distribution:

\[\begin{equation} -i\frac{d\varphi}{dk}\bigg|_{k=0}=\int xp(x)dx \ \Longrightarrow \ \boxed{\langle x\rangle=-i\frac{d\varphi}{dk}\bigg|_{k=0}} \end{equation}\]

From the series expansion of \(e^{ikx}\) a more general property is found:

\[\begin{equation*} e^{ikx}=\sum_{n=0}^\infty\frac{(ikx)^n}{n!} \ \Longrightarrow \ \varphi(k)=\left\langle e^{ikx}\right\rangle=\sum_{n=0}^\infty\frac{i^nk^n}{n!}\left\langle x^n\right\rangle \end{equation*}\]

By deriving \(n\) times with respect to \(k\), all the terms with a power of \(k\) less than \(n\) are canceled; what will remain is

\[\begin{equation*} \frac{d^n\varphi}{dk^n}=\frac{i^n\cancel{n!}}{\cancel{n!}}\langle x^n\rangle+\sum_{j=n+1}^\infty\frac{i^nk^{j-n}}{n!}\left\langle x^n\right\rangle \end{equation*}\]

By setting \(k=0\) again this time, all the terms of the summation will be zero, and we can thus derive the so-called moments of the distribution function:

\[\begin{equation} \frac{d^n\varphi}{dk^n}\bigg|_{k=0}=i^n\left\langle x^n\right\rangle \ \Longrightarrow \ \boxed{\left\langle x^n\right\rangle=(-i)^n\frac{d^n\varphi}{dk^n}\bigg|_{k=0}} \end{equation}\]

3.3.1 Characteristic function of the Gaussian

In the case of a Gaussian function we know that

\[\begin{equation*} p(x)=\sqrt{\frac{a}{2\pi}}e^{-\frac{a}{2}x^2} \end{equation*}\]

Setting \(\sigma^2=\frac{1}{a}\) it is rewritten as

\[\begin{equation*} p(x)=\frac{1}{\sqrt{2\pi\sigma^2}}e^{-\frac{x^2}{2\sigma^2}} \end{equation*}\]

The characteristic function is found directly from (eq:integrali-gaussiani-funzioni-caratteristiche-definizione):

\[\begin{equation} \varphi(k)=\frac{1}{\sqrt{2\pi\sigma^2}}\int_{-\infty}^{+\infty}e^{-\frac{x^2}{2\sigma^2}}e^{ikx}dx \end{equation}\]

From the Hubbard-Stratonovich transformation (equation (eq:integrali-gaussiani-trasformazione-di-hubbard-stratonovich)) we find the expression

\[\begin{equation} \boxed{\varphi(k)=e^{-\frac{\sigma^2}{2}k^2}} \end{equation}\]

We note that the pdf is an exponential with a second-degree polynomial in \(x\) as an exponent, and the same form is associated with the characteristic function \(\varphi(k)\) considering the variable \(k\): this is a peculiar property of the Gaussian function due to its characteristic symmetry.

It is useful now to calculate the mean of \(x\) and of its various powers \(x^n\).

For \(n=1\) the calculation is simple:

\[\begin{equation*} \langle x\rangle=-i\frac{d\varphi}{dk}\bigg|_{k=0}=-i\cdot\left(-\sigma^2k\right)e^{-\frac{\sigma^2}{2}k^2}\bigg|_{k=0}=0 \end{equation*}\]

In general, for odd \(n\) it is easy to show that the result will always be zero, since

\[\begin{equation} \left\langle x^n\right\rangle=\frac{1}{\sqrt{2\pi\sigma^2}}\int_{-\infty}^{+\infty}x^ne^{-\frac{x^2}{2\sigma^2}}dx=0 \end{equation}\]

since the integrand function is odd.

For the case of even \(n\), one must start from equation (eq:integrali-gaussiani-normalizzazione) and derive both members with respect to \(a\):

\[\begin{equation*} -\frac{1}{\cancel{2}}\int_{-\infty}^{+\infty}x^2e^{-\frac{a}{2}x^2}dx=-\frac{1}{\cancel{2}}\sqrt{\frac{2\pi}{a^3}} \ \Longrightarrow \ \left\langle x^2\right\rangle=\sqrt{\frac{a}{2\pi}}\int_{-\infty}^{+\infty}x^2e^{-\frac{a}{2}x^2}dx=\frac{1}{a} \end{equation*}\]

By deriving again, we find the case \(n=4\):

\[\begin{equation*} \frac{1}{\cancel{2}}\int_{-\infty}^{+\infty}x^4e^{-\frac{a}{2}x^2}dx=\frac{3}{\cancel{2}}\sqrt{\frac{2\pi}{a^5}} \ \Longrightarrow \ \left\langle x^4\right\rangle=\sqrt{\frac{a}{2\pi}}\int_{-\infty}^{+\infty}x^4e^{-\frac{a}{2}x^2}dx=\frac{3}{a^2} \end{equation*}\]

In general we will have

\[\begin{equation} \left\langle x^n\right\rangle=\frac{(n-1)!!}{a^{\frac{n}{2}}} \end{equation}\]

which can also be written as a function of \(\sigma\) as follows:

\[\begin{equation} \boxed{\left\langle x^n\right\rangle=(n-1)!!\sigma^n} \end{equation}\]

3.4 Multidimensional Gaussian integrals

Moving to a number of dimensions greater than one, it is useful to calculate the quantity

\[\begin{equation} Z(A)=\int_{\mathbb{R}^n}e^{-\frac{1}{2}\vec{x}^{\,T}A\vec{x}}d^nx \end{equation}\]

where \(x=\left(x_1,\dots,x_n\right)\) and \(A\) is a matrix of dimension \(n\times n\) that must respect two conditions:

If the matrix is diagonalizable then there exists an orthogonal matrix \(O\) such that

\[\begin{equation*} \Lambda=OAO^T \ \ \ \text{where} \ \Lambda= \begin{pmatrix} \lambda_{1} & &\\ & \ddots & \\ & & \lambda_{n} \end{pmatrix} \end{equation*}\]

At this point it is useful to perform a change of variable, setting \(\vec{y}=O\vec{x}\), so that in the exponent of (eq:integrali-gaussiani-multidimensione) it can be written

\[\begin{equation*} \vec{x}^{\,T}A\vec{x}=\vec{x}^{\,T}O^T\Lambda O\vec{x}=\vec{y}^{\,T}\Lambda\vec{y} \end{equation*}\]

By performing the change of variable we will then have

\[\begin{equation*} Z(A)=\int_{\mathbb{R}^n}e^{-\frac{1}{2}\vec{y}^{\,T}\Lambda\vec{y}}\norm{J}d^ny \end{equation*}\]

where \(\norm{J}\) is the determinant of the Jacobian:

\[\begin{equation*} J= \begin{pmatrix} \frac{\partial x_1}{\partial y_1} & \dots & \frac{\partial x_1}{\partial y_n}\\ \vdots & \ddots & \vdots\\ \frac{\partial x_n}{\partial y_1} & \dots & \frac{\partial x_n}{\partial y_n} \end{pmatrix} \end{equation*}\]

This matrix, in reality, coincides with \(O^T\), being

\[\begin{equation*} \vec{y}=O\vec{x} \ \iff \ \vec{x}=O^T\vec{y} \end{equation*}\]

Since \(O\) is an orthogonal matrix, \(O^T\) will in turn be orthogonal, so it will have a unit determinant by definition. It will then result

\[\begin{equation} \norm{J}=1 \ \Longrightarrow \ Z(A)=\int_{\mathbb{R}^n}e^{-\frac{1}{2}\vec{y}^{\,T}\Lambda\vec{y}}d^ny \end{equation}\]

Making the exponent explicit:

\[\begin{equation*} \sum_{i,j=1}^ny_i\Lambda_{ij}y_j=\sum_{i,j=1}^ny_i\lambda_{i}\delta_{ij}y_j=\sum_{i=1}^n\lambda_iy_i^2 \end{equation*}\]

From which, substituting in (eq:integrali-gaussiani-multidimensione-espressione-con-Lambda):

\[\begin{equation*} Z(A)=\int_{\mathbb{R}^n}e^{-\frac{1}{2}\sum\limits_{i=1}^n\lambda_iy_i^2}d^ny=\prod_{i=1}^n\int_{\mathbb{R}^n}e^{-\frac{1}{2}\lambda_i y_i^2}dy_i \end{equation*}\]

Each \(i\)-th integral is a Gaussian integral that is solved as we know from equation (eq:integrali-gaussiani-normalizzazione):

\[\begin{equation*} Z(A)=\prod_{i=1}^n\sqrt{\frac{2\pi}{\lambda_i}}=\sqrt{\frac{(2\pi)^n}{\lambda_1\lambda_2\dotso\lambda_n}}=\sqrt{\frac{(2\pi)^n}{\norm{\Lambda}}} \end{equation*}\]

Finally, since \(\norm{A}=\norm{\Lambda}\), we obtain the following result:

\[\begin{equation} \boxed{Z(A)=\sqrt{\frac{(2\pi)^n}{\norm{A}}}} \end{equation}\]

3.4.1 Calculation with the most general form

Analogously to what was done in section (sec:integrali-gaussiani-la-forma-piu-generale), it is possible to extend the reasoning to the most general form of the Gaussian integral, which in a space of generic dimension \(n\) is written as

\[\begin{equation} Z\left(A,\vec{b}\right)=\int_{\mathbb{R}^n}e^{-\frac{1}{2}\vec{x}^{\,T}A\vec{x}+\vec{x}^{\,T}\cdot\vec{b}}d^nx \end{equation}\]

In this case it is useful to translate the minimum of the exponential to the origin. First of all, therefore, let’s calculate this minimum:

\[\begin{equation} \frac{\partial}{\partial x_i}\left(-\frac{1}{2}\vec{x}^{\,T}A\vec{x}+\vec{x}^{\,T}\cdot\vec{b}\right)=0 \end{equation}\]

If the derivative of the second term is trivial, for the first it is advisable to make the summations explicit:

\[\begin{align*} \frac{\partial}{\partial x_i}\left(\vec{x}^{\,T}A\vec{x}\right)&=\frac{\partial}{\partial x_i}\sum_{j,k}x_jA_{jk}x_k=\sum_{j,k}\delta_{ji}A_{jk}x_k+\sum_{j,k}x_jA_{jk}\delta_{ki}=\\ &=\sum_{k}A_{ik}x_k+\sum_{j}x_jA_{ji} \end{align*}\]

Writing both summations with the same index results in:

\[\begin{equation} \frac{\partial}{\partial x_i}\left(\vec{x}^{\,T}A\vec{x}\right)=\sum_{j}(A_{ij}+A_{ji})x_j=\sum_{ij}2A_{ij}x_j=2A\vec{x} \end{equation}\]

having also assumed this time that the matrix is symmetric (\(A_{ij}=A_{ji}\)).

Returning to (eq:integrali-gaussiani-minimo-forma-generale), we have:

\[\begin{equation*} -A\vec{x}+\vec{b}=0 \ \Longrightarrow \ \vec{x}=A^{-1}\vec{b} \end{equation*}\]

Let’s now perform the translation

\[\begin{equation} \vec{y}=\vec{x}-A^{-1}\vec{b} \ \iff \ \vec{x}=\vec{y}+A^{-1}\vec{b} \end{equation}\]

Let’s substitute in the exponent:

\[\begin{align*} &-\frac{1}{2}\vec{x}^{\,T}A\vec{x}+\vec{x}^{\,T}\cdot\vec{b}=-\frac{1}{2}\left(\vec{y}^{\,T}+A^{-1}\vec{b}^{\,T}\right)A\left(\vec{y}+A^{-1}\vec{b}\right)+\left(\vec{y}^{\,T}+A^{-1}\vec{b}^{\,T}\right)\cdot\vec{b}=\\ &=-\frac{1}{2}\left(\vec{y}^{\,T}A\vec{y}+\vec{y}^{\,T}AA^{-1}\vec{b}+A^{-1}A\vec{b}^{\,T}\cdot\vec{y}+A^{-1}AA^{-1}\vec{b}^{\,T}\cdot\vec{b}\right)+\vec{y}^{\,T}\cdot\vec{b}+\vec{b}^{\,T}A^{-1}\vec{b} \end{align*}\]

Substituting \(AA^{-1}=A^{-1}A=\mathds{1}\):

\[\begin{equation*} -\frac{1}{2}\vec{x}^{\,T}A\vec{x}+\vec{x}^{\,T}\cdot\vec{b}=-\frac{1}{2}\left(\vec{y}^{\,T}A\vec{y}+\vec{y}^{\,T}\cdot\vec{b}+\vec{b}^{\,T}\cdot\vec{y}+\vec{b}^{\,T}A^{-1}\vec{b}\right)+\vec{y}^{\,T}\cdot\vec{b}+\vec{b}^{\,T}A^{-1}\vec{b} \end{equation*}\]

Due to commutativity it follows that \(\vec{y}^{\,T}\cdot\vec{b}=\vec{b}\cdot\vec{y}^{\,T}\), and also

\[\begin{equation*} \vec{b}\cdot\vec{y}^{\,T}=\left(\vec{b}\cdot\vec{y}^{\,T}\right)^{T}=\vec{b}^{\,T}\cdot\vec{y} \end{equation*}\]

being a scalar. Three terms therefore cancel out, giving a sum of zero, from which it follows:

\[\begin{equation*} -\frac{1}{2}\vec{x}^{\,T}A\vec{x}+\vec{x}^{\,T}\cdot\vec{b}=-\frac{1}{2}\vec{y}^{\,T}A\vec{y}-\frac{1}{2}\vec{b}^{\,T}A^{-1}\vec{b}+\vec{b}^{\,T}A^{-1}\vec{b}=-\frac{1}{2}\vec{y}^{\,T}A\vec{y}+\frac{1}{2}\vec{b}^{\,T}A^{-1}\vec{b} \end{equation*}\]

We can now substitute in the integral (eq:integrali-gaussiani-multidimensionale-forma-generale); let’s remember that, since this is also a simple translation, the determinant of the Jacobian matrix is unitary:

\[\begin{equation*} Z\left(A,\vec{b}\right)=\int_{\mathbb{R}^n}e^{-\frac{1}{2}\vec{y}^{\,T}A\vec{y}+\frac{1}{2}\vec{b}^{\,T}A^{-1}\vec{b}}d^ny \end{equation*}\]

From the formula (eq:integrali-gaussiani-multidimensionale-risultato-Z) we find the final result:

\[\begin{equation} \boxed{Z\left(A,\vec{b}\right)=\sqrt{\frac{(2\pi)^n}{\norm{A}}}e^{\frac{1}{2}\vec{b}^{\,T}A^{-1}\vec{b}}} \end{equation}\]

3.4.2 Calculation of the characteristic function

The result found in equation (eq:integrali-gaussiani-multidimensionale-risultato) is useful for calculating the characteristic function of a multidimensional Gaussian.

Equation (eq:integrali-gaussiani-multidimensionale-risultato-Z) tells us that the normalized Gaussian is

\[\begin{equation*} p\left(\vec{x}\right)=\sqrt{\frac{\norm{A}}{(2\pi)^n}}e^{-\frac{1}{2}\vec{x}^{\,T}A\vec{x}} \end{equation*}\]

From the definition of the characteristic function (equation (eq:integrali-gaussiani-funzioni-caratteristiche-definizione)) it follows:

\[\begin{equation*} \varphi\left(\vec{k}\right)=\int_{\mathbb{R}^n}e^{i\vec{k}\cdot\vec{x}}p\left(\vec{x}\right)d^nx=\sqrt{\frac{\norm{A}}{(2\pi)^n}}\int_{\mathbb{R}^n}e^{-\frac{1}{2}\vec{x}^{\,T}A\vec{x}+i\vec{k}\cdot\vec{x}}d^nx \end{equation*}\]

The result of the integral, as anticipated, is obtained directly from equation (eq:integrali-gaussiani-multidimensionale-risultato), setting \(\vec{b}=i\vec{k}\):

\[\begin{equation} \varphi\left(\vec{k}\right)=\sqrt{\frac{(2\pi)^n}{\norm{A}}}\cdot\sqrt{\frac{\norm{A}}{(2\pi)^n}}e^{\frac{1}{2}i\vec{k}^{\,T}A^{-1}i\vec{k}} \ \Longrightarrow \ \boxed{\varphi\left(\vec{k}\right)=e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}}} \end{equation}\]

3.5 Correlation function

Through the characteristic function we can calculate the moments of the multidimensional Gaussian.

Since in this case we are in the presence of multiple variables, we can choose some of these and obtain the following moment:

\[\begin{equation*} \left\langle x_ix_j\dots x_\ell\right\rangle \end{equation*}\]

where each variable can also be taken multiple times.

Extending the formula (eq:integrali-gaussiani-momenti) to a space of multiple dimensions, one obtains:

\[\begin{equation} \left\langle x_ix_j\dots x_\ell\right\rangle=(-i)^s\frac{\partial}{\partial k_i}\frac{\partial}{\partial k_j}\dots\frac{\partial}{\partial k_\ell}\varphi\left(\vec{k}\right)\bigg|_{\vec{k}=0} \end{equation}\]

where \(s\) is the total number of variables chosen, counting each one for each time it was taken. The function obtained from (eq:integrali-gaussiani-funzione-di-correlazione) is called the \(s\)-point correlation function.

As a simple case, we can calculate the two-point correlation function of the Gaussian:

\[\begin{equation} \left\langle x_ix_j\right\rangle=(-i)^2\frac{\partial}{\partial k_i}\frac{\partial}{\partial k_j}e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}}\bigg|_{\vec{k}=0} \end{equation}\]

Let’s first calculate the double derivative. As written in equation (eq:integrali-gaussiani-derivata-xAx), it results:

\[\begin{equation} \frac{\partial}{\partial k_j}e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}}=-\frac{1}{\cancel{2}}\cdot\cancel{2}A^{-1}\vec{k}e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}}=-A^{-1}\vec{k}e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}} \end{equation}\]

It remains to calculate the derivative of what has been obtained. Let’s therefore calculate the derivative of \(A^{-1}\vec{k}\):

\[\begin{equation*} \frac{\partial}{\partial k_i}A^{-1}\vec{k}=\frac{\partial}{\partial k_i}\sum_{r}\left(A^{-1}\right)_{jr}k_r=\sum_{r}\left(A^{-1}\right)_{jr}\delta_{ir}=\left(A^{-1}\right)_{ji}=\left(A^{-1}\right)_{ij} \end{equation*}\]

having always assumed the matrix \(A\) to be symmetric.

We can now derive what is written in (eq:integrali-gaussiani-calcolo-funzione-correlazione):

\[\begin{equation*} \frac{\partial}{\partial k_i}\frac{\partial}{\partial k_j}e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}}=-\left(A^{-1}\right)_{ij}e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}}+\left(A^{-1}\vec{k}\right)\cdot\left(A^{-1}\vec{k}\right)e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}} \end{equation*}\]

Returning to (eq:integrali-gaussiani-funzione-di-correlazione-due-punti-gaussiana), we will have to set \(\vec{k}=0\) for what has been calculated. In this way the second term will be canceled and in the first term the exponential will be identically equal to \(1\). We will therefore have

\[\begin{equation} \left\langle x_ix_j\right\rangle=(-i)^2\left[-\left(A^{-1}\right)_{ij}\right] \ \Longrightarrow \ \boxed{\left\langle x_ix_j\right\rangle=\left(A^{-1}\right)_{ij}} \end{equation}\]

We thus discover that

$\left(A^{-1}\right)_{ij}$ is the **two-point correlation function** of a pair of Gaussian random variables

If \(A^{-1}\) is diagonal, we can say that the two variables are uncorrelated with each other.

It is now easy to understand why the matrix must necessarily be symmetric, since the correlation function must not depend on the order of the variables in equation (eq:integrali-gaussiani-multidimensionale-funzione-correlazione-due-punti-gaussiana), and this justifies the assumption made initially on the form of the matrix.

Referring back to exercise (ex:integrali-gaussiani-due-variabili, verify that)

\begin{equation*}

\end{equation*}

Solution. Let’s calculate the inverse of \(A\):

\begin{equation*}
    A=
    \begin{pmatrix}
        3 & -1\\
        -1 & 3
    \end{pmatrix}
    \ \Longrightarrow \ A^{-1}=\frac{1}{8}
    \begin{pmatrix}
        3 & 1\\
        1 & 3
    \end{pmatrix}
\end{equation*}

By calculating the moments through equation (eq:integrali-gaussiani-multidimensionale-funzione-correlazione-due-punti-gaussiana) we actually find the results written in the assignment.

3.5.1 Wick’s theorem

Still in the case of Gaussian functions, it is clear that if \(s\) is odd, the correlation function will certainly be zero, due to the integration of an odd function over a symmetric interval, analogously to what was seen in paragraph (sec:integrali-gaussiani-funzione-caratteristica-della-gaussiana).

It remains to consider the case with even \(s\). Fortunately, it is not necessary to solve all the integrals that come out in the calculation each time, because Wick’s theorem simplifies the work for us:

Any correlation between an *even* number of Gaussian random variables can be rewritten as the **sum of all possible two-by-two products** of the two-point correlation functions

Since from the previous paragraph we know that the two-point correlation function for the Gaussian is given by \(\left(A^{-1}\right)_{ij}\), then in formulas the theorem can be rewritten:

\[\begin{equation} \boxed{\left\langle x_ix_j\dots x_nx_m\right\rangle=\sum_p\left(A^{-1}\right)_{i_pj_p}\dots\left(A^{-1}\right)_{n_pm_p}} \end{equation}\]

where the summation runs over all possible pairings of the \(s\) indices, which are equal to \(s!!=(s-1)\cdot(s-3)\dots3\cdot1\).

For example, in the case of four variables, their correlation function can be written as:

\[\begin{align*} \left\langle x_1x_2x_3x_4\right\rangle&=\left\langle x_1x_2\right\rangle\left\langle x_3x_4\right\rangle+\left\langle x_1x_3\right\rangle\left\langle x_2x_4\right\rangle+\left\langle x_1x_4\right\rangle\left\langle x_2x_3\right\rangle=\\ &=\left(A^{-1}\right)_{12}\left(A^{-1}\right)_{34}+\left(A^{-1}\right)_{13}\left(A^{-1}\right)_{24}+\left(A^{-1}\right)_{14}\left(A^{-1}\right)_{23} \end{align*}\]

*Returning to exercise (ex:integrali-gaussiani-due-variabili*, calculate the following moments:)

\begin{equation*}
    \left\langle x_1^2x_2^2\right\rangle \ \ \ \ \ \ \ \ \left\langle x_1^4\right\rangle \ \ \ \ \ \ \ \ \left\langle x_1x_2^2\right\rangle
\end{equation*}

For the first quantity, let's apply Wick's theorem:

\begin{align*}
    \left\langle x_1^2x_2^2\right\rangle&=\left\langle x_1^2\right\rangle\left\langle x_2^2\right\rangle+\left\langle x_1x_2\right\rangle\left\langle x_1x_2\right\rangle+\left\langle x_1x_2\right\rangle\left\langle x_1x_2\right\rangle=\\
    &=\left(A^{-1}\right)_{11}\left(A^{-1}\right)_{22}+\left(A^{-1}\right)_{12}\left(A^{-1}\right)_{12}+\left(A^{-1}\right)_{12}\left(A^{-1}\right)_{12}=\\
    &=\frac{3}{8}\cdot\frac{3}{8}+\frac{1}{8}\cdot\frac{1}{8}+\frac{1}{8}\cdot\frac{1}{8}=\frac{9}{64}+\frac{1}{64}+\frac{1}{64}=\frac{11}{64}
\end{align*}

Analogously for the second quantity:

\begin{equation*}
    \left\langle x_1^4\right\rangle=3\cdot\frac{3}{8}\cdot\frac{3}{8}=\frac{27}{64}
\end{equation*}

As for the third quantity, we have $s=3$, so since it is odd and we are dealing with a Gaussian function, the result is certainly zero.

\begin{equation*}

\end{equation*}

*Use* $x_1,x_2$ *to define two variables* $y_1,y_2$ *such that*

\begin{equation*}

\begin{equation*}

\end{equation*}

Solution. Let’s calculate \(\sigma_1^2\) and \(\sigma_2^2\):

\begin{align*}
    &\sigma_1^2=\left\langle y_1^2\right\rangle=a_1^2\left\langle x_1^2\right\rangle+b_1^2\left\langle x_2^2\right\rangle+2a_1b_1\cancel{\left\langle x_1x_2\right\rangle}=a_1^2+b_1^2\\

\end{align*}

By setting, for example, $a_1=b_1$ we would find:

\begin{equation*}
    \sigma_1^2=2a_1^2 \ \iff \ a_1=\frac{\sigma_1}{\sqrt{2}}
\end{equation*}

Since the same also applies to $\sigma_2$, we will have the following expressions for the variables we are looking for:

\begin{equation*}

    &=\frac{\sigma_1\sigma_2}{2}\cdot2=\sigma_1\sigma_2
\end{align*}

# Gaussian integrals

## The simplest form

Let's consider the **Gaussian function**

\begin{equation}
    p(x)=ce^{-\frac{a}{2}x^2},   a\in\mathbb{R}^+
\end{equation}

If $p(x)$ is a ***pdf*** (***p**robability **d**ensity **f**unction*) then it must necessarily be normalized over the entire interval $\mathbb{R}$:

\begin{equation*}
    \int_{-\infty}^{+\infty}p(x)dx=1 \ \Longrightarrow \ c\int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2}dx=1
\end{equation*}

from which the value of $c$ is obtained.

The integral has as its solution

\begin{equation}
    \boxed{\int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2}dx=\sqrt{\frac{2\pi}{a}}}
\end{equation}

where $a$ - let's remember - must necessarily be real and positive.

## The most general form

More generally, the Gaussian integral can be written as

\begin{equation*}
    \int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2+bx}dx
\end{equation*}

To solve this integral it is useful to perform a change of variable starting from the maximum point of the function in the exponent, that is

\begin{equation*}
    \frac{d}{dx}\left(-\frac{a}{2}x^2+bx\right)=-ax+b=0 \ \Longrightarrow \ x=\frac{b}{a}
\end{equation*}

Setting

\begin{equation*}
    y=x-\frac{b}{a}
\end{equation*}

the exponent is written as

\begin{align*}
    -\frac{a}{2}x^2+bx&=-\frac{a}{2}\left(y^2+\frac{2b}{a}y+\frac{b^2}{a^2}\right)+by+\frac{b^2}{a}=\\
    &=-\frac{a}{2}y^2-\cancel{by}-\frac{b^2}{2a}+\cancel{by}+\frac{b^2}{a}=-\frac{a}{2}y^2+\frac{b^2}{2a}
\end{align*}

Rewriting the integral as a function of this new variable, we have

\begin{equation}
    \int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2+bx}dx=\int_{-\infty}^{+\infty}e^{-\frac{a}{2}y^2+\frac{b^2}{2a}}dy=\sqrt{\frac{2\pi}{a}}e^{\frac{b^2}{2a}}
\end{equation}

In this formula, $b$ can more generally belong to $\mathbb{C}$.

### Hubbard-Stratonovich transformation

Equation (eq:integrali-gaussiani-forma-generale) is often rewritten in the following way:

\begin{equation}
    \boxed{e^{\frac{b^2}{2a}}=\sqrt{\frac{a}{2\pi}}\int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2+bx}dx}
\end{equation}

This is called the **Hubbard-Stratonovich transformation** and is useful as it allows the passage from a quadratic expression in $b$ to a linear expression in $b$. It is mainly used when $b$ is a mathematical object such as a matrix, a field or a set of vectors, of which it would be difficult to calculate the square and it is therefore preferred to rewrite it in integral form, keeping $b$ in linear form.

### Calculation of the integral in the case $b\in\mathbb{C}$

Let's assume that $b=it$, with $t\in\mathbb{R}$.\footnote{This assumption that $b$ is a pure imaginary is made for simple reasons of calculation; the latter, in fact, could also be performed if the real part is non-zero.} The integral to be calculated becomes

\begin{equation}
    \varphi(t)=\sqrt{\frac{a}{2\pi}}\int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2}e^{itx}dx
\end{equation}

Interpreting the variable $t$ as temporal, the equation just written is none other than the Fourier transform of the Gaussian function (equation (eq:integrali-gaussiani-pdf-gaussiana)), which - as we will see later - is linked to the so-called *characteristic function*.

In order to calculate this integral, it is convenient to make the derivative of $\varphi(t)$ explicit. Since this is in the form $\int f(x,t)dx$, it is useful to recall the conditions that allow the derivation of $\varphi(t)$ through that of $f(x,t)$:

\begin{equation*}
    \frac{d}{dt}\int f(x,t)dx=\int\frac{\partial f(x,t)}{\partial t}dx \ \iff \
    \begin{cases}
        f \ \text{and} \ \frac{\partial f}{\partial t} \ \text{continuous}\\
        f \ \text{and} \ \frac{\partial f}{\partial t} \ \text{uniformly distributed}
    \end{cases}
\end{equation*}

The second condition, in particular, translates mathematically into

\begin{equation*}
    \begin{cases}
        \abs{f(x,t)}<A(x)\\
        \abs{\frac{\partial f(x,t)}{\partial t}}<B(x)
    \end{cases}
    \ \ \ \text{where} \
    \begin{cases}
        A(x)<\infty\\
        B(x)<\infty
    \end{cases}
\end{equation*}

Assuming these conditions are valid, it will result

\begin{equation*}
    \varphi'(t)=i\sqrt{\frac{a}{2\pi}}\int_{-\infty}^{+\infty}xe^{-\frac{a}{2}x^2}e^{itx}dx
\end{equation*}

By multiplying and dividing by $-a$ we can write the integrand as a function of the derivative of the first exponent:

\begin{equation*}
    \varphi'(t)=-\frac{i}{a}\sqrt{\frac{a}{2\pi}}\int_{-\infty}^{+\infty}(-ax)e^{-\frac{a}{2}x^2}e^{itx}dx=-\frac{i}{\sqrt{2\pi a}}\int_{-\infty}^{+\infty}\frac{d}{dx}\left(e^{-\frac{a}{2}x^2}\right)e^{itx}dx
\end{equation*}

It is natural, at this point, to proceed by integration by parts, where the term to be integrated is clearly the derivative of the exponential:

\begin{equation*}
    \varphi'(t)=-\frac{i}{\sqrt{2\pi a}}\left[e^{-\frac{a}{2}x^2}e^{itx}\bigg|_{-\infty}^{+\infty}-\int_{-\infty}^{+\infty}ite^{-\frac{a}{2}x^2}e^{ixt}dx\right]
\end{equation*}

The first term is negligible as the dominant exponential, that is $e^{-\frac{a}{2}x^2}$, tends to zero for $x\to\pm\infty$. What remains is

\begin{equation*}
    \varphi'(t)=-\frac{t}{\sqrt{2\pi a}}\int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2}e^{ixt}dx
\end{equation*}

Multiplying and dividing by $\sqrt{a}$ follows:

\begin{equation*}
    \varphi'(t)=-\frac{t}{a}\cdot\sqrt{\frac{a}{2\pi}}\int_{-\infty}^{+\infty}e^{-\frac{a}{2}x^2}e^{ixt}dx=-\frac{t}{a}\varphi(t)
\end{equation*}

Calculating the integral (eq:integrali-gaussiani-caso-complesso) therefore means solving the differential equation just written:

\begin{equation*}
    \frac{d\varphi}{dt}=-\frac{t}{a}\varphi \ \Longrightarrow \ \int\frac{d\varphi}{\varphi}=-\int\frac{t}{a}dt \ \Longrightarrow \ \log\varphi=-\frac{t^2}{2a}+c \ \Longrightarrow \ \varphi(t)=ke^{-\frac{t^2}{2a}}
\end{equation*}

To set a boundary condition, it is useful to observe equation (eq:integrali-gaussiani-caso-complesso): we note that by setting $t=0$ we find exactly the normalized Gaussian integral written in its simplest form, which therefore has a value equal to $1$. We will therefore set $\varphi(0)=1$, from which it follows that $k=1$. We will thus have

\begin{equation}
    \boxed{\varphi(t)=e^{-\frac{t^2}{2a}}}
\end{equation}

## Characteristic functions

From the moment a continuous random variable $x$ and a *pdf* function of this variable $p(x)$ are introduced, it is possible to define a new function, called the **characteristic function**, very similar to the Fourier transform of $p(x)$:

\begin{equation}
    \boxed{\varphi(k)=\int e^{ikx}p(x)dx=\left\langle e^{ikx}\right\rangle_p}
\end{equation}

This is a useful tool as it allows us to derive all the properties of $p(x)$ and is at the same time simpler to study.

A first important property that can be derived is the mean $\langle x\rangle$. By deriving $\varphi(k)$, in fact, we find

\begin{equation*}
    \frac{d\varphi}{dk}=\int ixe^{ikx}p(x)dx \ \Longrightarrow \ -i\frac{d\varphi}{dk}=\int xe^{ikx}p(x)dx
\end{equation*}

By setting $k=0$ we find precisely the mean of the distribution:

\begin{equation}
    -i\frac{d\varphi}{dk}\bigg|_{k=0}=\int xp(x)dx \ \Longrightarrow \ \boxed{\langle x\rangle=-i\frac{d\varphi}{dk}\bigg|_{k=0}}
\end{equation}

From the series expansion of $e^{ikx}$ a more general property is found:

\begin{equation*}
    e^{ikx}=\sum_{n=0}^\infty\frac{(ikx)^n}{n!} \ \Longrightarrow \ \varphi(k)=\left\langle e^{ikx}\right\rangle=\sum_{n=0}^\infty\frac{i^nk^n}{n!}\left\langle x^n\right\rangle
\end{equation*}

By deriving $n$ times with respect to $k$, all the terms with a power of $k$ less than $n$ are canceled; what will remain is

\begin{equation*}
    \frac{d^n\varphi}{dk^n}=\frac{i^n\cancel{n!}}{\cancel{n!}}\langle x^n\rangle+\sum_{j=n+1}^\infty\frac{i^nk^{j-n}}{n!}\left\langle x^n\right\rangle
\end{equation*}

By setting $k=0$ again this time, all the terms of the summation will be zero, and we can thus derive the so-called **moments** of the distribution function:

\begin{equation}
    \frac{d^n\varphi}{dk^n}\bigg|_{k=0}=i^n\left\langle x^n\right\rangle \ \Longrightarrow \ \boxed{\left\langle x^n\right\rangle=(-i)^n\frac{d^n\varphi}{dk^n}\bigg|_{k=0}}
\end{equation}

### Characteristic function of the Gaussian

In the case of a Gaussian function we know that

\begin{equation*}
    p(x)=\sqrt{\frac{a}{2\pi}}e^{-\frac{a}{2}x^2}
\end{equation*}

Setting $\sigma^2=\frac{1}{a}$ it is rewritten as

\begin{equation*}
    p(x)=\frac{1}{\sqrt{2\pi\sigma^2}}e^{-\frac{x^2}{2\sigma^2}}
\end{equation*}

The characteristic function is found directly from (eq:integrali-gaussiani-funzioni-caratteristiche-definizione):

\begin{equation}
    \varphi(k)=\frac{1}{\sqrt{2\pi\sigma^2}}\int_{-\infty}^{+\infty}e^{-\frac{x^2}{2\sigma^2}}e^{ikx}dx
\end{equation}

From the Hubbard-Stratonovich transformation (equation (eq:integrali-gaussiani-trasformazione-di-hubbard-stratonovich)) we find the expression

\begin{equation}
    \boxed{\varphi(k)=e^{-\frac{\sigma^2}{2}k^2}}
\end{equation}

We note that the *pdf* is an exponential with a second-degree polynomial in $x$ as an exponent, and the same form is associated with the characteristic function $\varphi(k)$ considering the variable $k$: this is a peculiar property of the Gaussian function due to its characteristic symmetry.

It is useful now to calculate the mean of $x$ and of its various powers $x^n$.

For $n=1$ the calculation is simple:

\begin{equation*}
    \langle x\rangle=-i\frac{d\varphi}{dk}\bigg|_{k=0}=-i\cdot\left(-\sigma^2k\right)e^{-\frac{\sigma^2}{2}k^2}\bigg|_{k=0}=0
\end{equation*}

In general, for odd $n$ it is easy to show that the result will always be zero, since

\begin{equation}
    \left\langle x^n\right\rangle=\frac{1}{\sqrt{2\pi\sigma^2}}\int_{-\infty}^{+\infty}x^ne^{-\frac{x^2}{2\sigma^2}}dx=0
\end{equation}

since the integrand function is odd.

For the case of even $n$, one must start from equation (eq:integrali-gaussiani-normalizzazione) and derive both members with respect to $a$:

\begin{equation*}
    -\frac{1}{\cancel{2}}\int_{-\infty}^{+\infty}x^2e^{-\frac{a}{2}x^2}dx=-\frac{1}{\cancel{2}}\sqrt{\frac{2\pi}{a^3}} \ \Longrightarrow \ \left\langle x^2\right\rangle=\sqrt{\frac{a}{2\pi}}\int_{-\infty}^{+\infty}x^2e^{-\frac{a}{2}x^2}dx=\frac{1}{a}
\end{equation*}

By deriving again, we find the case $n=4$:

\begin{equation*}
    \frac{1}{\cancel{2}}\int_{-\infty}^{+\infty}x^4e^{-\frac{a}{2}x^2}dx=\frac{3}{\cancel{2}}\sqrt{\frac{2\pi}{a^5}} \ \Longrightarrow \ \left\langle x^4\right\rangle=\sqrt{\frac{a}{2\pi}}\int_{-\infty}^{+\infty}x^4e^{-\frac{a}{2}x^2}dx=\frac{3}{a^2}
\end{equation*}

In general we will have

\begin{equation}
    \left\langle x^n\right\rangle=\frac{(n-1)!!}{a^{\frac{n}{2}}}
\end{equation}

which can also be written as a function of $\sigma$ as follows:

\begin{equation}
    \boxed{\left\langle x^n\right\rangle=(n-1)!!\sigma^n}
\end{equation}

## Multidimensional Gaussian integrals

Moving to a number of dimensions greater than one, it is useful to calculate the quantity

\begin{equation}
    Z(A)=\int_{\mathbb{R}^n}e^{-\frac{1}{2}\vec{x}^{\,T}A\vec{x}}d^nx
\end{equation}

where $x=\left(x_1,\dots,x_n\right)$ and $A$ is a matrix of dimension $n\times n$ that must respect two conditions:

\begin{enumerate}
    \item it must be **symmetric** (and therefore **diagonalizable**)
    \item it must have **positive eigenvalues**
\end{enumerate}

If the matrix is diagonalizable then there exists an orthogonal matrix $O$ such that

\begin{equation*}
    \Lambda=OAO^T \ \ \ \text{where} \ \Lambda=
    \begin{pmatrix}
      \lambda_{1} & &\\
      & \ddots & \\
      & & \lambda_{n}
    \end{pmatrix}
\end{equation*}

At this point it is useful to perform a change of variable, setting $\vec{y}=O\vec{x}$, so that in the exponent of (eq:integrali-gaussiani-multidimensione) it can be written

\begin{equation*}
    \vec{x}^{\,T}A\vec{x}=\vec{x}^{\,T}O^T\Lambda O\vec{x}=\vec{y}^{\,T}\Lambda\vec{y}
\end{equation*}

By performing the change of variable we will then have

\begin{equation*}
    Z(A)=\int_{\mathbb{R}^n}e^{-\frac{1}{2}\vec{y}^{\,T}\Lambda\vec{y}}\norm{J}d^ny
\end{equation*}

where $\norm{J}$ is the determinant of the Jacobian:

\begin{equation*}
    J=
    \begin{pmatrix}
        \frac{\partial x_1}{\partial y_1} & \dots & \frac{\partial x_1}{\partial y_n}\\
        \vdots & \ddots & \vdots\\
        \frac{\partial x_n}{\partial y_1} & \dots & \frac{\partial x_n}{\partial y_n}
    \end{pmatrix}
\end{equation*}

This matrix, in reality, coincides with $O^T$, being

\begin{equation*}
    \vec{y}=O\vec{x} \ \iff \ \vec{x}=O^T\vec{y}
\end{equation*}

Since $O$ is an orthogonal matrix, $O^T$ will in turn be orthogonal, so it will have a unit determinant by definition. It will then result

\begin{equation}
    \norm{J}=1 \ \Longrightarrow \ Z(A)=\int_{\mathbb{R}^n}e^{-\frac{1}{2}\vec{y}^{\,T}\Lambda\vec{y}}d^ny
\end{equation}

Making the exponent explicit:

\begin{equation*}
    \sum_{i,j=1}^ny_i\Lambda_{ij}y_j=\sum_{i,j=1}^ny_i\lambda_{i}\delta_{ij}y_j=\sum_{i=1}^n\lambda_iy_i^2
\end{equation*}

From which, substituting in (eq:integrali-gaussiani-multidimensione-espressione-con-Lambda):

\begin{equation*}
    Z(A)=\int_{\mathbb{R}^n}e^{-\frac{1}{2}\sum\limits_{i=1}^n\lambda_iy_i^2}d^ny=\prod_{i=1}^n\int_{\mathbb{R}^n}e^{-\frac{1}{2}\lambda_i y_i^2}dy_i
\end{equation*}

Each $i$-th integral is a Gaussian integral that is solved as we know from equation (eq:integrali-gaussiani-normalizzazione):

\begin{equation*}
    Z(A)=\prod_{i=1}^n\sqrt{\frac{2\pi}{\lambda_i}}=\sqrt{\frac{(2\pi)^n}{\lambda_1\lambda_2\dotso\lambda_n}}=\sqrt{\frac{(2\pi)^n}{\norm{\Lambda}}}
\end{equation*}

Finally, since $\norm{A}=\norm{\Lambda}$, we obtain the following result:

\begin{equation}
    \boxed{Z(A)=\sqrt{\frac{(2\pi)^n}{\norm{A}}}}
\end{equation}

### Calculation with the most general form

Analogously to what was done in section (sec:integrali-gaussiani-la-forma-piu-generale), it is possible to extend the reasoning to the most general form of the Gaussian integral, which in a space of generic dimension $n$ is written as

\begin{equation}
    Z\left(A,\vec{b}\right)=\int_{\mathbb{R}^n}e^{-\frac{1}{2}\vec{x}^{\,T}A\vec{x}+\vec{x}^{\,T}\cdot\vec{b}}d^nx
\end{equation}

In this case it is useful to translate the minimum of the exponential to the origin. First of all, therefore, let's calculate this minimum:

\begin{equation}
    \frac{\partial}{\partial x_i}\left(-\frac{1}{2}\vec{x}^{\,T}A\vec{x}+\vec{x}^{\,T}\cdot\vec{b}\right)=0
\end{equation}

If the derivative of the second term is trivial, for the first it is advisable to make the summations explicit:

\begin{align*}
    \frac{\partial}{\partial x_i}\left(\vec{x}^{\,T}A\vec{x}\right)&=\frac{\partial}{\partial x_i}\sum_{j,k}x_jA_{jk}x_k=\sum_{j,k}\delta_{ji}A_{jk}x_k+\sum_{j,k}x_jA_{jk}\delta_{ki}=\\
    &=\sum_{k}A_{ik}x_k+\sum_{j}x_jA_{ji}
\end{align*}

Writing both summations with the same index results in:

\begin{equation}
    \frac{\partial}{\partial x_i}\left(\vec{x}^{\,T}A\vec{x}\right)=\sum_{j}(A_{ij}+A_{ji})x_j=\sum_{ij}2A_{ij}x_j=2A\vec{x}
\end{equation}

having also assumed this time that the matrix is symmetric ($A_{ij}=A_{ji}$).

Returning to (eq:integrali-gaussiani-minimo-forma-generale), we have:

\begin{equation*}
    -A\vec{x}+\vec{b}=0 \ \Longrightarrow \ \vec{x}=A^{-1}\vec{b}
\end{equation*}

Let's now perform the translation

\begin{equation}
    \vec{y}=\vec{x}-A^{-1}\vec{b} \ \iff \ \vec{x}=\vec{y}+A^{-1}\vec{b}
\end{equation}

Let's substitute in the exponent:

\begin{align*}
    &-\frac{1}{2}\vec{x}^{\,T}A\vec{x}+\vec{x}^{\,T}\cdot\vec{b}=-\frac{1}{2}\left(\vec{y}^{\,T}+A^{-1}\vec{b}^{\,T}\right)A\left(\vec{y}+A^{-1}\vec{b}\right)+\left(\vec{y}^{\,T}+A^{-1}\vec{b}^{\,T}\right)\cdot\vec{b}=\\
    &=-\frac{1}{2}\left(\vec{y}^{\,T}A\vec{y}+\vec{y}^{\,T}AA^{-1}\vec{b}+A^{-1}A\vec{b}^{\,T}\cdot\vec{y}+A^{-1}AA^{-1}\vec{b}^{\,T}\cdot\vec{b}\right)+\vec{y}^{\,T}\cdot\vec{b}+\vec{b}^{\,T}A^{-1}\vec{b}
\end{align*}

Substituting $AA^{-1}=A^{-1}A=\mathds{1}$:

\begin{equation*}
    -\frac{1}{2}\vec{x}^{\,T}A\vec{x}+\vec{x}^{\,T}\cdot\vec{b}=-\frac{1}{2}\left(\vec{y}^{\,T}A\vec{y}+\vec{y}^{\,T}\cdot\vec{b}+\vec{b}^{\,T}\cdot\vec{y}+\vec{b}^{\,T}A^{-1}\vec{b}\right)+\vec{y}^{\,T}\cdot\vec{b}+\vec{b}^{\,T}A^{-1}\vec{b}
\end{equation*}

Due to commutativity it follows that $\vec{y}^{\,T}\cdot\vec{b}=\vec{b}\cdot\vec{y}^{\,T}$, and also

\begin{equation*}
    \vec{b}\cdot\vec{y}^{\,T}=\left(\vec{b}\cdot\vec{y}^{\,T}\right)^{T}=\vec{b}^{\,T}\cdot\vec{y}
\end{equation*}

being a scalar. Three terms therefore cancel out, giving a sum of zero, from which it follows:

\begin{equation*}
    -\frac{1}{2}\vec{x}^{\,T}A\vec{x}+\vec{x}^{\,T}\cdot\vec{b}=-\frac{1}{2}\vec{y}^{\,T}A\vec{y}-\frac{1}{2}\vec{b}^{\,T}A^{-1}\vec{b}+\vec{b}^{\,T}A^{-1}\vec{b}=-\frac{1}{2}\vec{y}^{\,T}A\vec{y}+\frac{1}{2}\vec{b}^{\,T}A^{-1}\vec{b}
\end{equation*}

We can now substitute in the integral (eq:integrali-gaussiani-multidimensionale-forma-generale); let's remember that, since this is also a simple translation, the determinant of the Jacobian matrix is unitary:

\begin{equation*}
    Z\left(A,\vec{b}\right)=\int_{\mathbb{R}^n}e^{-\frac{1}{2}\vec{y}^{\,T}A\vec{y}+\frac{1}{2}\vec{b}^{\,T}A^{-1}\vec{b}}d^ny
\end{equation*}

From the formula (eq:integrali-gaussiani-multidimensionale-risultato-Z) we find the final result:

\begin{equation}
    \boxed{Z\left(A,\vec{b}\right)=\sqrt{\frac{(2\pi)^n}{\norm{A}}}e^{\frac{1}{2}\vec{b}^{\,T}A^{-1}\vec{b}}}
\end{equation}

### Calculation of the characteristic function

The result found in equation (eq:integrali-gaussiani-multidimensionale-risultato) is useful for calculating the characteristic function of a multidimensional Gaussian.

Equation (eq:integrali-gaussiani-multidimensionale-risultato-Z) tells us that the normalized Gaussian is

\begin{equation*}
    p\left(\vec{x}\right)=\sqrt{\frac{\norm{A}}{(2\pi)^n}}e^{-\frac{1}{2}\vec{x}^{\,T}A\vec{x}}
\end{equation*}

From the definition of the characteristic function (equation (eq:integrali-gaussiani-funzioni-caratteristiche-definizione)) it follows:\footnote{I believe that the exponential term with $\vec{k}$ should be rigorously written as $i\vec{x}^{\,T}\cdot\vec{k}$ as it is a row-by-column product, just as done in (eq:integrali-gaussiani-multidimensionale-forma-generale). I will however report the form given by the professor, which I imagine is equivalent (it would only be an abuse of notation).}

\begin{equation*}
    \varphi\left(\vec{k}\right)=\int_{\mathbb{R}^n}e^{i\vec{k}\cdot\vec{x}}p\left(\vec{x}\right)d^nx=\sqrt{\frac{\norm{A}}{(2\pi)^n}}\int_{\mathbb{R}^n}e^{-\frac{1}{2}\vec{x}^{\,T}A\vec{x}+i\vec{k}\cdot\vec{x}}d^nx
\end{equation*}

The result of the integral, as anticipated, is obtained directly from equation (eq:integrali-gaussiani-multidimensionale-risultato), setting $\vec{b}=i\vec{k}$:

\begin{equation}
    \varphi\left(\vec{k}\right)=\sqrt{\frac{(2\pi)^n}{\norm{A}}}\cdot\sqrt{\frac{\norm{A}}{(2\pi)^n}}e^{\frac{1}{2}i\vec{k}^{\,T}A^{-1}i\vec{k}} \ \Longrightarrow \ \boxed{\varphi\left(\vec{k}\right)=e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}}}
\end{equation}

## Correlation function

Through the characteristic function we can calculate the moments of the multidimensional Gaussian.

Since in this case we are in the presence of multiple variables, we can choose some of these and obtain the following moment:

\begin{equation*}
    \left\langle x_ix_j\dots x_\ell\right\rangle
\end{equation*}

where each variable can also be taken multiple times.

Extending the formula (eq:integrali-gaussiani-momenti) to a space of multiple dimensions, one obtains:

\begin{equation}
    \left\langle x_ix_j\dots x_\ell\right\rangle=(-i)^s\frac{\partial}{\partial k_i}\frac{\partial}{\partial k_j}\dots\frac{\partial}{\partial k_\ell}\varphi\left(\vec{k}\right)\bigg|_{\vec{k}=0}
\end{equation}

where $s$ is the total number of variables chosen, counting each one for each time it was taken. The function obtained from (eq:integrali-gaussiani-funzione-di-correlazione) is called the **$s$-point correlation function**.

As a simple case, we can calculate the two-point correlation function of the Gaussian:

\begin{equation}
    \left\langle x_ix_j\right\rangle=(-i)^2\frac{\partial}{\partial k_i}\frac{\partial}{\partial k_j}e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}}\bigg|_{\vec{k}=0}
\end{equation}

Let's first calculate the double derivative. As written in equation (eq:integrali-gaussiani-derivata-xAx), it results:

\begin{equation}
    \frac{\partial}{\partial k_j}e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}}=-\frac{1}{\cancel{2}}\cdot\cancel{2}A^{-1}\vec{k}e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}}=-A^{-1}\vec{k}e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}}
\end{equation}

It remains to calculate the derivative of what has been obtained. Let's therefore calculate the derivative of $A^{-1}\vec{k}$:

\begin{equation*}
    \frac{\partial}{\partial k_i}A^{-1}\vec{k}=\frac{\partial}{\partial k_i}\sum_{r}\left(A^{-1}\right)_{jr}k_r=\sum_{r}\left(A^{-1}\right)_{jr}\delta_{ir}=\left(A^{-1}\right)_{ji}=\left(A^{-1}\right)_{ij}
\end{equation*}

having always assumed the matrix $A$ to be symmetric.

We can now derive what is written in (eq:integrali-gaussiani-calcolo-funzione-correlazione):

\begin{equation*}
    \frac{\partial}{\partial k_i}\frac{\partial}{\partial k_j}e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}}=-\left(A^{-1}\right)_{ij}e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}}+\left(A^{-1}\vec{k}\right)\cdot\left(A^{-1}\vec{k}\right)e^{-\frac{1}{2}\vec{k}^{\,T}A^{-1}\vec{k}}
\end{equation*}

Returning to (eq:integrali-gaussiani-funzione-di-correlazione-due-punti-gaussiana), we will have to set $\vec{k}=0$ for what has been calculated. In this way the second term will be canceled and in the first term the exponential will be identically equal to $1$. We will therefore have

\begin{equation}
    \left\langle x_ix_j\right\rangle=(-i)^2\left[-\left(A^{-1}\right)_{ij}\right] \ \Longrightarrow \ \boxed{\left\langle x_ix_j\right\rangle=\left(A^{-1}\right)_{ij}}
\end{equation}

We thus discover that

::: {.definition}
    $\left(A^{-1}\right)_{ij}$ is the **two-point correlation function** of a pair of Gaussian random variables

:::

If $A^{-1}$ is diagonal, we can say that the two variables are **uncorrelated** with each other.

It is now easy to understand why the matrix must necessarily be symmetric, since the correlation function must not depend on the order of the variables in equation (eq:integrali-gaussiani-multidimensionale-funzione-correlazione-due-punti-gaussiana), and this justifies the assumption made initially on the form of the matrix.

### Wick's theorem

Still in the case of Gaussian functions, it is clear that if $s$ is odd, the correlation function will certainly be zero, due to the integration of an odd function over a symmetric interval, analogously to what was seen in paragraph (sec:integrali-gaussiani-funzione-caratteristica-della-gaussiana).

It remains to consider the case with even $s$. Fortunately, it is not necessary to solve all the integrals that come out in the calculation each time, because **Wick's theorem** simplifies the work for us:

::: {.theorem name="Wick's theorem"}
    Any correlation between an *even* number of Gaussian random variables can be rewritten as the **sum of all possible two-by-two products** of the two-point correlation functions

:::

Since from the previous paragraph we know that the two-point correlation function for the Gaussian is given by $\left(A^{-1}\right)_{ij}$, then in formulas the theorem can be rewritten:

\begin{equation}
    \boxed{\left\langle x_ix_j\dots x_nx_m\right\rangle=\sum_p\left(A^{-1}\right)_{i_pj_p}\dots\left(A^{-1}\right)_{n_pm_p}}
\end{equation}

where the summation runs over all possible pairings of the $s$ indices, which are equal to $s!!=(s-1)\cdot(s-3)\dots3\cdot1$.

For example, in the case of four variables, their correlation function can be written as:

\begin{align*}
    \left\langle x_1x_2x_3x_4\right\rangle&=\left\langle x_1x_2\right\rangle\left\langle x_3x_4\right\rangle+\left\langle x_1x_3\right\rangle\left\langle x_2x_4\right\rangle+\left\langle x_1x_4\right\rangle\left\langle x_2x_3\right\rangle=\\
    &=\left(A^{-1}\right)_{12}\left(A^{-1}\right)_{34}+\left(A^{-1}\right)_{13}\left(A^{-1}\right)_{24}+\left(A^{-1}\right)_{14}\left(A^{-1}\right)_{23}
\end{align*}
