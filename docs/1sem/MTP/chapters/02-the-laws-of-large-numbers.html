4.1 Joint Distribution

Let’s imagine we have two random variables \(x_1\) and \(x_2\) that are independent and also identically distributed, meaning they follow the same distribution law.

The joint distribution of these two variables, being independent, can be factored into the product of the individual distributions:

\[\begin{equation*} p\left(x_1,x_2\right)=p_1\left(x_1\right)p_2\left(x_2\right) \end{equation*}\]

Specifically, being identically distributed, we can write:

\[\begin{equation*} p_1(x)=p_2(x)=q(x) \ \Longrightarrow \ p\left(x_1,x_2\right)=q\left(x_1\right)q\left(x_2\right) \end{equation*}\]

4.2 Convolution

At this point, we can calculate the distribution of the sum \(x_1+x_2\) through the following reasoning: let’s imagine extracting \(x_1\) and \(x_2\) according to their distribution \(q(x)\); for the same outcome \(x\), different combinations of outcomes are possible, and all these outcomes must therefore be weighted according to the corresponding probability:

\[\begin{equation*} p(x)=\int\delta\left[x-\left(x_1+x_2\right)\right]q\left(x_1\right)q\left(x_2\right)dx_1dx_2\equiv\left\langle\delta\left(x-x_1-x_2\right)\right\rangle \end{equation*}\]

Rearranging the terms inside the delta function we have

\[\begin{equation*} p(x)=\int q\left(x_1\right)dx_1\int\delta\left[\left(x-x_1\right)-x_2\right]q\left(x_2\right)dx_2=\int q\left(x-x_1\right)q\left(x_1\right)dx_1 \end{equation*}\]

Choosing the final integral with the variable \(x_1\) or \(x_2\) is completely arbitrary, so we rename this variable \(y\), thus obtaining

\[\begin{equation} \boxed{p(x)=\int q(x-y)q(y)dy} \end{equation}\]

The expression obtained is called a convolution.

Calculate the distribution of \(x=x_0+x_1\) where \(x_0\) and \(x_1\) are two independent and identically distributed variables according to

\begin{enumerate}
    \item Since $x_1,x_2\in[0,1]$ then $x\in[0,2]$. Intuitively, we understand that $x$ cannot follow a uniform distribution, since the number of possible combinations for the extreme values ($0$ and $2$) is only one, while for the intermediate values there are certainly more.

    Referring to equation (eq:convoluzione), the functions $q(x-y)$ and $q(y)$ are non-zero only when both $x-y$ and $y$ belong to the interval $[0,1]$. Making the interval for the first variable explicit with respect to $y$:

    \begin{equation*}
        0\leqslant x-y\leqslant1 \ \iff \ -x\leqslant-y\leqslant1-x \ \iff \ x-1\leqslant y\leqslant x
    \end{equation*}

    We know that $x\in[0,2]$, so the lower limit will be $x-1\in[-1,1]$. We distinguish the negative case from the positive one:

    \begin{itemize}
        \item $x-1\in[-1,0] \ \iff \ x\in[0,1]$

        $y$ cannot be negative, so the lower limit can only be $0$; as for the upper limit, however, the entire interval of $x$ is valid:

        \begin{equation*}
            p(x)=\int_0^x1\cdot1dy=y|_0^x=x
        \end{equation*}

        \item $x-1\in(0,1] \ \iff \ x\in[1,2]$

        This time the interval for the lower limit is entirely valid, while for the upper limit only $1$ is valid, since $y$ cannot exceed $1$:

        \begin{equation*}
            p(x)=\int_{x-1}^11\cdot1dy=y|_{x-1}^1=1-x+1=2-x
        \end{equation*}
    \end{itemize}

    The final pdf, therefore, is the following:

    \begin{equation*}
        p(x)=
        \begin{cases}
            0 \ &x\in(-\infty,0)\cup(2,+\infty)\\
            x & x\in[0,1]\\
            2-x & x\in(1,2]
        \end{cases}
    \end{equation*}

    \item This time we have no limits for the variables since they are defined in all of $\mathbb{R}$.

    \begin{align*}

    \begin{equation*}
        p(x)=\frac{1}{2\pi\sigma^2}e^{-\frac{x^2-2x\mu+2\mu^2}{2\sigma^2}}\int_{-\infty}^{+\infty}e^{-\frac{1}{2}\cdot\frac{2}{\sigma^2}y^2+\frac{x}{\sigma^2}y}dy
    \end{equation*}

            a=\frac{2}{\sigma^2}\\
            b=\frac{x}{\sigma^2}
        \end{cases}
        \Longrightarrow \ \ \ p(x)&=\frac{1}{2\pi\sigma^2}e^{-\frac{x^2-2x\mu+2\mu^2}{2\sigma^2}}\cdot\sqrt{\frac{2\pi\sigma^2}{2}}e^{\frac{\frac{x^2}{\sigma^4}}{2\frac{2}{\sigma^2}}}=\\

    \end{align*}

    \begin{align*}
        -\frac{x^2-2x\mu+2\mu^2}{2\sigma^2}+\frac{x^2}{4\sigma^2}&=-\frac{2x^2-4x\mu+4\mu^2-x^2}{4\sigma^2}=\\
        &=-\frac{x^2-4x\mu+4\mu^2}{4\sigma^2}
    \end{align*}

    Substituting:

    \begin{equation*}
        p(x)=\frac{1}{\sqrt{2}\sigma\sqrt{2\pi}}e^{-\frac{x^2-4x\mu+4\mu^2}{4\sigma^2}}
    \end{equation*}

    In order to obtain a Gaussian function, let's make the following substitutions:

    \begin{equation*}
        \begin{cases}
            \mu'=2\mu
            \sigma'=\sqrt{2}\sigma
        \end{cases}
    \end{equation*}

    In this way we obtain the final distribution, which is precisely a Gaussian:

    \begin{equation*}
        p(x)=\frac{1}{\sigma'\sqrt{2\pi}}e^{-\frac{x^2-2x\mu'+\mu'^2}{2\sigma'^2}}=\frac{1}{\sigma'\sqrt{2\pi}}e^{-\frac{(x-\mu')^2}{2\sigma'^2}}
    \end{equation*}

    \item Exponentially distributed variables can only take non-negative values, and to ensure this, the integration must take place in the interval $[0,x]$, similarly to what was done for uniformly distributed variables.

    \begin{equation*}
        p(x)=\int_0^x\lambda e^{-\lambda(x-y)}\cdot\lambda e^{-\lambda y}dy=\lambda^2\int_0^xe^{-\lambda(x\cancel{-y}\cancel{+y})}dy=
    \end{equation*}

    From which it follows:

    \begin{equation*}
        p(x)=\lambda^2e^{-\lambda x}\int_0^xdy=\lambda^2xe^{-\lambda x}
    \end{equation*}
\end{enumerate}

    &=\int q\left(x_1\right)dx_1\int\delta\left(x-x_1x_2\right)q\left(x_2\right)dx_2

Let's make the following substitution:

\begin{equation*}
    z=x_1x_2 \ \iff \ x_2=\frac{z}{x_1} \ \Longrightarrow \ dx_2=\frac{1}{x_1}dz
\end{equation*}

\begin{equation*}
    p(x)=\int q\left(x_1\right)dx_1\int\delta(x-z)q\left(\frac{z}{x_1}\right)\frac{1}{x_1}dz=\int q\left(\frac{x}{x_1}\right)q\left(x_1\right)\frac{1}{x_1}dx_1
\end{equation*}

Substituting $x_1$ with the generic variable $y$ we find:

\begin{equation}
    p(x)=\int q\left(\frac{x}{y}\right)q\left(y\right)\frac{1}{y}dy
\end{equation}

4.2.1 Characteristic function of the sum of variables

Calculating a convolution, in general, is not simple. For this reason, it is convenient to calculate the characteristic function associated with it:

\[\begin{align*} \varphi(k)&=\int p(x)e^{ikx}dx=\int\delta\left[x-\left(x_1+x_2\right)\right]q\left(x_1\right)q\left(x_2\right)e^{ikx}dx_1dx_2dx=\\ &=\int q\left(x_1\right)q\left(x_2\right)e^{ik\left(x_1+x_2\right)}dx_1dx_2 \end{align*}\]

Since the variables are independent, we can factor the double integral into a product of integrals:

\[\begin{equation*} \varphi(k)=\int q\left(x_1\right)e^{ikx_1}dx_1\int q\left(x_2\right)e^{ikx_2}dx_2 \end{equation*}\]

Since the distribution functions are the same, the result of the integral is the same for both. Again, choosing \(x_1\) or \(x_2\) is arbitrary, so we can generally denote the \(i\)-th variable as \(x_i\).

\[\begin{equation*} \varphi(k)=\left(\int q\left(x_i\right)e^{ikx_i}dx_i\right)^2=\varphi_i^2(k) \end{equation*}\]

Obviously, the reasoning can be extended to \(n\) independent variables, provided they all have the same distribution function. We thus discover that, under these conditions, we have

\[\begin{equation} \boxed{\varphi(k)=\varphi_i^n(k)} \end{equation}\]

which in words translates to:

The characteristic function associated with the sum of $n$ independent and identically distributed variables is given by the **$\boldsymbol{n-**$th power of the characteristic function} associated with the distribution function

The same reasoning can be repeated with independent but not identically distributed variables. In that case too, we have the simple product of the individual characteristic functions, which this time, however, will differ from each other:

The characteristic function associated with the sum of $n$ independent *but not identically distributed* variables is given by the **product of the characteristic functions** associated with each distribution function

First of all, we must revisit equation (eq:integrali-gaussiani-funzione-caratteristica-calcolo), considering that this time we will have to consider the distribution centered not at $0$ but at $\mu$:

\begin{equation*}
    \varphi(k)=\frac{1}{\sqrt{2\pi\sigma^2}}\int_{-\infty}^{+\infty}e^{-\frac{(x-\mu)^2}{2\sigma^2}}e^{ikx}dx
\end{equation*}

It is sufficient to set $x'=x-\mu$ and we find the form known to us:

\begin{equation*}
    \varphi(k)=\frac{1}{\sqrt{2\pi\sigma^2}}\int_{-\infty}^{+\infty}e^{-\frac{x'^2}{2\sigma^2}}e^{ik(x'+\mu)}dx'=e^{-\frac{\sigma^2}{2}k^2}\cdot e^{ik\mu}=e^{-\frac{\sigma^2}{2}k^2+ik\mu}
\end{equation*}

At this point, we just need to use (eq:leggi-dei-grandi-numeri-somma-variabili-gaussiane-funzione-caratteristica):

\begin{equation*}
    \varphi(k)=e^{-\sigma^2k^2+2ik\mu}
\end{equation*}

By setting $\mu'=2\mu$ and $\sigma'=\sqrt{2}\sigma$ we obtain the characteristic function of a Gaussian, similarly to what was seen in the last exercise.

4.3 The Weak Law of Large Numbers

We now have all the tools we need to deal with the law of large numbers; actually, we will initially give a proof of a weaker formulation of it, and then state the stronger one.

The law is useful for establishing the behavior of the empirical mean of \(n\) independent and identically distributed variables as \(n\) itself increases, that is, of the values extracted from the distribution that is the same for all. In mathematical terms, we are interested in calculating:

\[\begin{equation} \lim_{n\to\infty}\frac{1}{n}\sum_{i=1}^nx_i \end{equation}\]

We will prove that, if \(\mu\) is the mean of the distribution of each random variable, the result of the limit coincides precisely with \(\mu\). In reality, we will not prove the direct convergence of the limit to \(\mu\) but the so-called convergence in distribution, which is obtained through the characteristic function of the pdf: in this sense the law is interpreted as weak.

Let’s therefore write the characteristic function associated with the empirical mean \(x\):

\[\begin{align} \begin{split} \varphi(k)&=\left\langle e^{ikx}\right\rangle=\left\langle e^{i\frac{k}{n}\sum\limits_{i=1}^nx_i}\right\rangle=\int q\left(x_1\right)\dots q\left(x_n\right)e^{i\frac{k}{n}\sum\limits_{i=1}^nx_i}dx_1\dots dx_n=\\ &=\int q\left(x_1\right)e^{i\frac{k}{n}x_1}dx_1\dotsc \int q\left(x_n\right)e^{i\frac{k}{n}x_n}dx_n=\varphi_i^n\left(\frac{k}{n}\right) \end{split} \end{align}\]

where the index \(i\) indicates a generic value from \(1\) to \(n\), as all characteristic functions are equal since the variables are identically distributed.

Let’s make \(\varphi_i\left(\frac{k}{n}\right)\) explicit and expand the exponential in a Taylor series:

\[\begin{align*} \varphi_i\left(\frac{k}{n}\right)&=\int q\left(x_i\right)e^{i\frac{k}{n}x_i}dx_i=\int q\left(x_i\right)\left[1+i\frac{k}{n}x_i+\mathcal{O}\left(\frac{1}{n^2}\right)\right]dx_i=\\ &=\int q\left(x_i\right)dx_i+i\frac{k}{n}\int x_iq\left(x_i\right)dx_i+\mathcal{O}\left(\frac{1}{n^2}\right)=1+i\frac{k}{n}\left\langle x_i\right\rangle+\mathcal{O}\left(\frac{1}{n^2}\right) \end{align*}\]

But having assumed \(\left\langle x_i\right\rangle=\mu\) then

\[\begin{equation*} \varphi_i\left(\frac{k}{n}\right)=1+i\frac{k}{n}\mu+\mathcal{O}\left(\frac{1}{n^2}\right) \end{equation*}\]

Returning to (eq:legge-dei-grandi-numeri-debole-funzione-caratteristica):

\[\begin{equation*} \varphi(k)=\varphi_i^n\left(\frac{k}{n}\right)=\left[1+i\frac{k}{n}\mu+\mathcal{O}\left(\frac{1}{n^2}\right)\right]^n \end{equation*}\]

Finally, passing to the limit \(n\to\infty\) we are able to state that

\[\begin{equation} \boxed{\varphi(k)=\left\langle e^{i\frac{k}{n}\sum\limits_{i=1}^nx_i}\right\rangle\to e^{ik\mu}} \end{equation}\]

This means that, at the limit, the empirical mean is not in turn a random variable, but is a fixed number equal to \(\mu\). Mathematically we can therefore state that the distribution function of \(x\) at the limit is nothing more than a delta function, so much so that we can write:

\[\begin{equation*} e^{ik\mu}=\int\delta(x-\mu)e^{ikx}dx \end{equation*}\]

Intuitively, we can imagine a decrease in statistical fluctuations around the mean \(\mu\) as the number of extractions of variables from the distribution increases.

4.4 The Strong Law of Large Numbers

In reality, it is possible to prove that convergence does not only occur through the characteristic function, but it is the empirical mean itself that converges to \(\mu\); in this formulation, the law of large numbers is called strong.

Before we can write this law, it is necessary to introduce two concepts: that of convergence in probability and that of almost sure convergence.

A sequence of random variables $x_n$ **converges in probability** to a random variable $x$ if

\begin{equation}
    \lim_{n\to\infty}P\left(\abs{x_n-x}>\epsilon\right)=0   \forall\epsilon>0
\end{equation}

In this way, the probability that \(x_n\) deviates from \(x\) by an amount greater than \(\epsilon\) goes to zero as the number of variables considered increases. However, this does not guarantee that \(x_n\) converges to \(x\) for \(n\to\infty\); to understand why, let’s consider the following example of a sequence of random variables:

\[\begin{equation*} x_n= \begin{cases} 0 \ \ \text{with probability} \ p(n)=\frac{1}{n}\\ 1 \ \ \text{with probability} \ 1-p(n)=1-\frac{1}{n} \end{cases} \end{equation*}\]

A realization of such a sequence will consist of \(0\)s and \(1\)s in sequence, with the various \(1\)s becoming increasingly rare:

\[\begin{equation*} 010010000010000000000100000000000000\dots \end{equation*}\]

The sequence converges in probability according to the definition, and it is equally evident that, regardless of the number \(n\), it will always be possible to draw a \(1\), however small that probability may be. If there existed a number \(N\) such that for every \(n>N\) we had \(x_n=0\) with probability equal to \(1\) (or similarly \(x_n=1\) with probability \(0\)) the convergence would be called almost certain.

A sequence of random variables $x_n$ **converges almost surely** to a random variable $x$ if

\begin{equation}
    P\left(\lim_{n\to\infty}x_n=x\right)=1
\end{equation}

We can easily guess that

\[\begin{equation*} \text{**Almost sure convergence**}\ \ \ \genfrac{}{}{0pt}{}{\Longrightarrow}{\cancel{\Longleftarrow}} \ \ \ \text{**Convergence in probability**} \end{equation*}\]

The strong law of large numbers makes use of almost sure convergence, stating that

\[\begin{equation} \boxed{\lim_{n\to\infty}\frac{1}{n}\sum_{i=1}^nx_i=\mu \ \ \text{**almost surely**}} \end{equation}\]

The proof in this case is very elaborate and is therefore not reported.

4.5 Central Limit Theorem

With the law of large numbers we know that as \(n\) grows we have

\[\begin{equation*} \sum_{i=1}^nx_i\sim n\mu \end{equation*}\]

This means that for a finite \(n\), that is, when not at the limit, the difference \(\sum x_i-n\mu\) will return non-zero values that coincide with the statistical fluctuations around \(n\mu\).

In this regard, it is natural to ask what trend these fluctuations have, that is, how that difference behaves. A first important result is that, as \(n\) increases, it is shown that

\[\begin{equation*} \sum_{i=1}^nx_i-n\mu\sim\sqrt{n} \end{equation*}\]

Knowing this trend, it is therefore possible to “normalize” the difference at the limit by dividing by \(\sqrt{n}\). The study of this normalized difference, moreover, allows us to demonstrate that its distribution converges to a normalized Gaussian with \(\mu=0\) and \(\sigma=1\) regardless of the distribution followed by the \(\boldsymbol{x_i}\).

It is important to underline that the two results just written are valid only if the variance \(\sigma^2\) associated with the variables \(x_i\) is finite.

What is written is the heart of one of the most important theorems in statistics and probability theory: the central limit theorem.

Given $n$ *independent* and *identically distributed* random variables with a finite mean $\mu$ and a finite and non-zero variance $\sigma^2$, we have that

\begin{equation}
   \boxed{Y_n=\frac{\sum\limits_{i=1}^nx_i-n\mu}{\sqrt{n}\sigma}\xrightarrow{\text{$n\to\infty$}} \ \mathcal{N}(0,1)}
\end{equation}

where $\mathcal{N}(0,1)$ is a Gaussian distribution with $\mu=0$ and $\sigma=1$

Before proceeding with the proof, it is useful to calculate the mean and variance of \(Y_n\) at the limit for \(n\to\infty\), using the result obtained from the central limit theorem:

\[\begin{align*} &\left\langle Y_n\right\rangle=\frac{\sum\limits_{i=1}^n\left\langle x_i\right\rangle-n\mu}{\sqrt{n}\sigma}=\frac{n\mu-n\mu}{\sqrt{n}\sigma}\to0\\ &Var\left(Y_n\right)=\frac{Var\left(\sum\limits_{i=1}^nx_i-n\mu\right)}{n\sigma^2}=\frac{Var\left(\sum\limits_{i=1}^nx_i\right)}{n\sigma^2}=\frac{\sum\limits_{i=1}^nVar\left(x_i\right)}{n\sigma^2}=\frac{n\sigma^2}{n\sigma^2}\to1 \end{align*}\]

In particular for the variance, the following properties have been exploited:

\[\begin{align*} &Var(ax)=a^2Var(x)\\ &Var(x+b)=Var(x) \end{align*}\]

where \(a\) and \(b\) are constants.

Let’s now proceed with the proof.

First, let's calculate the characteristic function $\varphi(k)$ associated with $Y_n$:

\begin{equation*}
    \varphi(k)=\left\langle e^{ikY_n}\right\rangle=\int q\left(x_1\right)\dots q\left(x_n\right)e^{ik\frac{\sum\limits_{i=1}^nx_i-n\mu}{\sqrt{n}\sigma}}dx_1\dots dx_n
\end{equation*}

We can now factor out $e^{-i\frac{kn}{\sqrt{n}\sigma}\mu}=e^{-i\frac{k\sqrt{n}}{\sigma}\mu}$ and factorize the multiple integral exactly as we did in (eq:legge-dei-grandi-numeri-debole-funzione-caratteristica):

\begin{align}
    \begin{split}
        \varphi(k)&=e^{-i\frac{k\sqrt{n}}{\sigma}\mu}\int q\left(x_1\right)e^{i\frac{k}{\sqrt{n}\sigma}x_1}dx_1\dotsc\int q\left(x_n\right)e^{i\frac{k}{\sqrt{n}\sigma}x_n}dx_n=\\
        &=e^{-i\frac{k\sqrt{n}}{\sigma}\mu}\left(\int q\left(x_i\right)e^{i\frac{k}{\sqrt{n}\sigma}x_i}dx_i\right)^n=e^{-i\frac{k\sqrt{n}}{\sigma}\mu}\varphi_i^n\left(\frac{k}{\sqrt{n}\sigma}\right)
    \end{split}
\end{align}

Continuing in the direction of the proof of the weak law of large numbers, we expand the $i$-th characteristic function found in series:

\begin{align*}
    \varphi_i\left(\frac{k}{\sqrt{n}\sigma}\right)&=\int q\left(x_i\right)e^{i\frac{k}{\sqrt{n}\sigma}x_i}dx_i=\
    &=\int q\left(x_i\right)\left[1+i\frac{k}{\sqrt{n}\sigma}x_i-\frac{k^2}{2n\sigma^2}x_i^2+\mathcal{O}\left(\frac{1}{\sqrt{n^3}}\right)\right]dx_i=\\
    &=1+i\frac{k}{\sqrt{n}\sigma}\left\langle x_i\right\rangle-\frac{k^2}{2n\sigma^2}\left\langle x_i^2\right\rangle+\mathcal{O}\left(\frac{1}{\sqrt{n^3}}\right)
\end{align*}

Substituting $\mu=\left\langle x_i\right\rangle$ and $\sigma^2=\left\langle x_i^2\right\rangle-\left\langle x_i\right\rangle^2 \ \iff \ \left\langle x_i^2\right\rangle=\sigma^2+\mu^2$ we find:

\begin{align*}
    \varphi_i\left(\frac{k}{\sqrt{n}\sigma}\right)&=1+i\frac{k}{\sqrt{n}\sigma}\mu-\frac{k^2}{2n\sigma^2}\left(\sigma^2+\mu^2\right)+\mathcal{O}\left(\frac{1}{\sqrt{n^3}}\right)=\\
    &=1+i\frac{k}{\sqrt{n}\sigma}\mu-\frac{k^2}{2n}-\frac{k^2}{2n\sigma^2}\mu^2+\mathcal{O}\left(\frac{1}{\sqrt{n^3}}\right)
\end{align*}

What is written is nothing more than the series expansion of the following exponential:

\begin{align*}
    e^{i\frac{k}{\sqrt{n}\sigma}\mu-\frac{k^2}{2n}}&\simeq1+i\frac{k}{\sqrt{n}\sigma}\mu-\frac{k^2}{2n}+\frac{1}{2}\left(i\frac{k}{\sqrt{n}\sigma}\mu-\frac{k^2}{2n}\right)^2=\\
    &=1+i\frac{k}{\sqrt{n}\sigma}\mu-\frac{k^2}{2n}-\frac{k^2}{2n\sigma^2}\mu^2+\mathcal{O}\left(\frac{1}{\sqrt{n^3}}\right)
\end{align*}

Substituting in (eq:leggi-dei-grandi-numeri-funzione-caratteristica-limite-centrale) the characteristic function finally results in:

\begin{equation*}
    \varphi(k)=e^{-i\frac{k\sqrt{n}}{\sigma}\mu}\cdot e^{i\frac{kn}{\sqrt{n}\sigma}\mu-\frac{k^2n}{2n}}=e^{-\cancel{i\frac{k\sqrt{n}}{\sigma}\mu}+\cancel{i\frac{k\sqrt{n}}{\sigma}\mu}-\frac{k^2}{2}}=e^{-\frac{k^2}{2}}
\end{equation*}

that is, the characteristic function associated with the normalized Gaussian with $\mu=0$ and $\sigma=1$, just as predicted by the theorem.

Multiplying both sides by $\sqrt{n}\sigma$:

\begin{equation*}
    \sum_{i=1}^nx_i-n\mu\sim\sqrt{n}\sigma\mathcal{N}(0,1)=\mathcal{N}\left(0,n\sigma^2\right)
\end{equation*}

Adding $n\mu$ to both sides:

\begin{equation*}
    \sum_{i=1}^nx_i\sim\mathcal{N}\left(0,n\sigma^2\right)+n\mu=\mathcal{N}\left(n\mu,n\sigma^2\right)
\end{equation*}

In this way we have proved the first relation; to obtain the second one it is sufficient to divide both sides of the last equation by $n$:

\begin{equation*}
    \frac{1}{n}\sum_{i=1}^nx_i\sim\frac{1}{n}\mathcal{N}\left(n\mu,n\sigma^2\right)=\mathcal{N}\left(\frac{n\mu}{n},\frac{n\sigma^2}{n^2}\right)=\mathcal{N}\left(\mu,\frac{\sigma^2}{n}\right)
\end{equation*}

# The Laws of Large Numbers

## Joint Distribution

Let's imagine we have two random variables $x_1$ and $x_2$ that are *independent* and also *identically distributed*, meaning they follow the same distribution law.

The **joint distribution** of these two variables, being independent, can be factored into the product of the individual distributions:

\begin{equation*}
    p\left(x_1,x_2\right)=p_1\left(x_1\right)p_2\left(x_2\right)
\end{equation*}

Specifically, being identically distributed, we can write:

\begin{equation*}
    p_1(x)=p_2(x)=q(x) \ \Longrightarrow \ p\left(x_1,x_2\right)=q\left(x_1\right)q\left(x_2\right)
\end{equation*}

## Convolution

At this point, we can calculate the distribution of the sum $x_1+x_2$ through the following reasoning: let's imagine extracting $x_1$ and $x_2$ according to their distribution $q(x)$; for the same outcome $x$, different combinations of outcomes are possible, and all these outcomes must therefore be weighted according to the corresponding probability:

\begin{equation*}
    p(x)=\int\delta\left[x-\left(x_1+x_2\right)\right]q\left(x_1\right)q\left(x_2\right)dx_1dx_2\equiv\left\langle\delta\left(x-x_1-x_2\right)\right\rangle
\end{equation*}

Rearranging the terms inside the delta function we have

\begin{equation*}
    p(x)=\int q\left(x_1\right)dx_1\int\delta\left[\left(x-x_1\right)-x_2\right]q\left(x_2\right)dx_2=\int q\left(x-x_1\right)q\left(x_1\right)dx_1
\end{equation*}

Choosing the final integral with the variable $x_1$ or $x_2$ is completely arbitrary, so we rename this variable $y$, thus obtaining

\begin{equation}
    \boxed{p(x)=\int q(x-y)q(y)dy}
\end{equation}

The expression obtained is called a **convolution**.

### Characteristic function of the sum of variables

Calculating a convolution, in general, is not simple. For this reason, it is convenient to calculate the characteristic function associated with it:

\begin{align*}
    \varphi(k)&=\int p(x)e^{ikx}dx=\int\delta\left[x-\left(x_1+x_2\right)\right]q\left(x_1\right)q\left(x_2\right)e^{ikx}dx_1dx_2dx=\\
    &=\int q\left(x_1\right)q\left(x_2\right)e^{ik\left(x_1+x_2\right)}dx_1dx_2
\end{align*}

Since the variables are independent, we can factor the double integral into a product of integrals:

\begin{equation*}
    \varphi(k)=\int q\left(x_1\right)e^{ikx_1}dx_1\int q\left(x_2\right)e^{ikx_2}dx_2
\end{equation*}

Since the distribution functions are the same, the result of the integral is the same for both. Again, choosing $x_1$ or $x_2$ is arbitrary, so we can generally denote the $i$-th variable as $x_i$.

\begin{equation*}
    \varphi(k)=\left(\int q\left(x_i\right)e^{ikx_i}dx_i\right)^2=\varphi_i^2(k)
\end{equation*}

Obviously, the reasoning can be extended to $n$ independent variables, provided they all have the same distribution function. We thus discover that, under these conditions, we have

\begin{equation}
    \boxed{\varphi(k)=\varphi_i^n(k)}
\end{equation}

which in words translates to:

::: {.definition}
    The characteristic function associated with the sum of $n$ independent and identically distributed variables is given by the **$\boldsymbol{n-**$th power of the characteristic function} associated with the distribution function

:::

The same reasoning can be repeated with independent but not identically distributed variables. In that case too, we have the simple product of the individual characteristic functions, which this time, however, will differ from each other:

::: {.definition}
    The characteristic function associated with the sum of $n$ independent *but not identically distributed* variables is given by the **product of the characteristic functions** associated with each distribution function

:::

## The Weak Law of Large Numbers

We now have all the tools we need to deal with the law of large numbers; actually, we will initially give a proof of a weaker formulation of it, and then state the stronger one.

The law is useful for establishing the behavior of the empirical mean of $n$ independent and identically distributed variables as $n$ itself increases, that is, of the values extracted from the distribution that is the same for all. In mathematical terms, we are interested in calculating:

\begin{equation}
    \lim_{n\to\infty}\frac{1}{n}\sum_{i=1}^nx_i
\end{equation}

We will prove that, if $\mu$ is the mean of the distribution of each random variable, the result of the limit coincides precisely with $\mu$. In reality, we will not prove the direct convergence of the limit to $\mu$ but the so-called **convergence in distribution**, which is obtained through the characteristic function of the pdf: in this sense the law is interpreted as *weak*.

Let's therefore write the characteristic function associated with the empirical mean $x$:

\begin{align}
    \begin{split}
        \varphi(k)&=\left\langle e^{ikx}\right\rangle=\left\langle e^{i\frac{k}{n}\sum\limits_{i=1}^nx_i}\right\rangle=\int q\left(x_1\right)\dots q\left(x_n\right)e^{i\frac{k}{n}\sum\limits_{i=1}^nx_i}dx_1\dots dx_n=\\
        &=\int q\left(x_1\right)e^{i\frac{k}{n}x_1}dx_1\dotsc \int q\left(x_n\right)e^{i\frac{k}{n}x_n}dx_n=\varphi_i^n\left(\frac{k}{n}\right)
    \end{split}
\end{align}

where the index $i$ indicates a generic value from $1$ to $n$, as all characteristic functions are equal since the variables are identically distributed.

Let's make $\varphi_i\left(\frac{k}{n}\right)$ explicit and expand the exponential in a Taylor series:

\begin{align*}
    \varphi_i\left(\frac{k}{n}\right)&=\int q\left(x_i\right)e^{i\frac{k}{n}x_i}dx_i=\int q\left(x_i\right)\left[1+i\frac{k}{n}x_i+\mathcal{O}\left(\frac{1}{n^2}\right)\right]dx_i=\\
    &=\int q\left(x_i\right)dx_i+i\frac{k}{n}\int x_iq\left(x_i\right)dx_i+\mathcal{O}\left(\frac{1}{n^2}\right)=1+i\frac{k}{n}\left\langle x_i\right\rangle+\mathcal{O}\left(\frac{1}{n^2}\right)
\end{align*}

But having assumed $\left\langle x_i\right\rangle=\mu$ then

\begin{equation*}
    \varphi_i\left(\frac{k}{n}\right)=1+i\frac{k}{n}\mu+\mathcal{O}\left(\frac{1}{n^2}\right)
\end{equation*}

Returning to (eq:legge-dei-grandi-numeri-debole-funzione-caratteristica):

\begin{equation*}
    \varphi(k)=\varphi_i^n\left(\frac{k}{n}\right)=\left[1+i\frac{k}{n}\mu+\mathcal{O}\left(\frac{1}{n^2}\right)\right]^n
\end{equation*}

Finally, passing to the limit $n\to\infty$ we are able to state that

\begin{equation}
    \boxed{\varphi(k)=\left\langle e^{i\frac{k}{n}\sum\limits_{i=1}^nx_i}\right\rangle\to e^{ik\mu}}
\end{equation}

This means that, at the limit, the empirical mean is not in turn a random variable, but is a fixed number equal to $\mu$. Mathematically we can therefore state that the distribution function of $x$ at the limit is nothing more than a delta function, so much so that we can write:

\begin{equation*}
    e^{ik\mu}=\int\delta(x-\mu)e^{ikx}dx
\end{equation*}

Intuitively, we can imagine a decrease in statistical fluctuations around the mean $\mu$ as the number of extractions of variables from the distribution increases.

## The Strong Law of Large Numbers

In reality, it is possible to prove that convergence does not only occur through the characteristic function, but it is the empirical mean itself that converges to $\mu$; in this formulation, the law of large numbers is called *strong*.

Before we can write this law, it is necessary to introduce two concepts: that of *convergence in probability* and that of *almost sure convergence*.

::: {.definition}
    A sequence of random variables $x_n$ **converges in probability** to a random variable $x$ if

    \begin{equation}
        \lim_{n\to\infty}P\left(\abs{x_n-x}>\epsilon\right)=0   \forall\epsilon>0
    \end{equation}

:::

In this way, the probability that $x_n$ deviates from $x$ by an amount greater than $\epsilon$ goes to zero as the number of variables considered increases. However, this does not guarantee that $x_n$ converges to $x$ for $n\to\infty$; to understand why, let's consider the following example of a sequence of random variables:

\begin{equation*}
    x_n=
    \begin{cases}
        0 \ \   \text{with probability} \ p(n)=\frac{1}{n}\\
        1 \ \   \text{with probability} \ 1-p(n)=1-\frac{1}{n}
    \end{cases}
\end{equation*}

A *realization*\footnote{That is, a specific value assumed by the sequence for a particular experiment or observation.} of such a sequence will consist of $0$s and $1$s in sequence, with the various $1$s becoming increasingly rare:

\begin{equation*}
    010010000010000000000100000000000000\dots
\end{equation*}

The sequence converges in probability according to the definition, and it is equally evident that, regardless of the number $n$, it will *always* be possible to draw a $1$, however small that probability may be. If there existed a number $N$ such that for every $n>N$ we had $x_n=0$ with probability equal to $1$ (or similarly $x_n=1$ with probability $0$) the convergence would be called **almost certain**.\footnote{Obviously the written reasoning is only of a qualitative nature, but through the second Borel-Cantelli lemma it is possible to rigorously prove what has been written.}

::: {.definition}
    A sequence of random variables $x_n$ **converges almost surely** to a random variable $x$ if

    \begin{equation}
        P\left(\lim_{n\to\infty}x_n=x\right)=1
    \end{equation}

:::

We can easily guess that

\begin{equation*}
    \text{**Almost sure convergence**}\ \ \ \genfrac{}{}{0pt}{}{\Longrightarrow}{\cancel{\Longleftarrow}} \ \ \ \text{**Convergence in probability**}
\end{equation*}

The strong law of large numbers makes use of almost sure convergence, stating that

\begin{equation}
    \boxed{\lim_{n\to\infty}\frac{1}{n}\sum_{i=1}^nx_i=\mu \ \              \text{**almost surely**}}
\end{equation}

The proof in this case is very elaborate and is therefore not reported.

## Central Limit Theorem

With the law of large numbers we know that as $n$ grows we have

\begin{equation*}
    \sum_{i=1}^nx_i\sim n\mu
\end{equation*}

This means that for a finite $n$, that is, when not at the limit, the difference $\sum x_i-n\mu$ will return non-zero values that coincide with the statistical fluctuations around $n\mu$.

In this regard, it is natural to ask what trend these fluctuations have, that is, how that difference behaves. A first important result is that, as $n$ increases, it is shown that

\begin{equation*}
    \sum_{i=1}^nx_i-n\mu\sim\sqrt{n}
\end{equation*}

Knowing this trend, it is therefore possible to "normalize" the difference at the limit by dividing by $\sqrt{n}$. The study of this normalized difference, moreover, allows us to demonstrate that its distribution converges to a normalized Gaussian with $\mu=0$ and $\sigma=1$ **regardless of the distribution followed by the** $\boldsymbol{x_i}$.

It is important to underline that the two results just written are valid only if the variance $\sigma^2$ associated with the variables $x_i$ is **finite**.

What is written is the heart of one of the most important theorems in statistics and probability theory: the **central limit theorem**.

::: {.theorem name="Central Limit Theorem"}
    Given $n$ *independent* and *identically distributed* random variables with a finite mean $\mu$ and a finite and non-zero variance $\sigma^2$, we have that

    \begin{equation}
       \boxed{Y_n=\frac{\sum\limits_{i=1}^nx_i-n\mu}{\sqrt{n}\sigma}\xrightarrow{\text{$n\to\infty$}} \ \mathcal{N}(0,1)}
    \end{equation}

    where $\mathcal{N}(0,1)$ is a Gaussian distribution with $\mu=0$ and $\sigma=1$

:::

Before proceeding with the proof, it is useful to calculate the mean and variance of $Y_n$ at the limit for $n\to\infty$, using the result obtained from the central limit theorem:

\begin{align*}
    &\left\langle Y_n\right\rangle=\frac{\sum\limits_{i=1}^n\left\langle x_i\right\rangle-n\mu}{\sqrt{n}\sigma}=\frac{n\mu-n\mu}{\sqrt{n}\sigma}\to0\\
    &Var\left(Y_n\right)=\frac{Var\left(\sum\limits_{i=1}^nx_i-n\mu\right)}{n\sigma^2}=\frac{Var\left(\sum\limits_{i=1}^nx_i\right)}{n\sigma^2}=\frac{\sum\limits_{i=1}^nVar\left(x_i\right)}{n\sigma^2}=\frac{n\sigma^2}{n\sigma^2}\to1
\end{align*}

In particular for the variance, the following properties have been exploited:

\begin{align*}
    &Var(ax)=a^2Var(x)\\   &Var(x+b)=Var(x)
\end{align*}

where $a$ and $b$ are constants.

Let's now proceed with the proof.

::: {.theorem}
    First, let's calculate the characteristic function $\varphi(k)$ associated with $Y_n$:

    \begin{equation*}
        \varphi(k)=\left\langle e^{ikY_n}\right\rangle=\int q\left(x_1\right)\dots q\left(x_n\right)e^{ik\frac{\sum\limits_{i=1}^nx_i-n\mu}{\sqrt{n}\sigma}}dx_1\dots dx_n
    \end{equation*}

    We can now factor out $e^{-i\frac{kn}{\sqrt{n}\sigma}\mu}=e^{-i\frac{k\sqrt{n}}{\sigma}\mu}$ and factorize the multiple integral exactly as we did in (eq:legge-dei-grandi-numeri-debole-funzione-caratteristica):

    \begin{align}
        \begin{split}
            \varphi(k)&=e^{-i\frac{k\sqrt{n}}{\sigma}\mu}\int q\left(x_1\right)e^{i\frac{k}{\sqrt{n}\sigma}x_1}dx_1\dotsc\int q\left(x_n\right)e^{i\frac{k}{\sqrt{n}\sigma}x_n}dx_n=\\
            &=e^{-i\frac{k\sqrt{n}}{\sigma}\mu}\left(\int q\left(x_i\right)e^{i\frac{k}{\sqrt{n}\sigma}x_i}dx_i\right)^n=e^{-i\frac{k\sqrt{n}}{\sigma}\mu}\varphi_i^n\left(\frac{k}{\sqrt{n}\sigma}\right)
        \end{split}
    \end{align}

    Continuing in the direction of the proof of the weak law of large numbers, we expand the $i$-th characteristic function found in series:

    \begin{align*}
        \varphi_i\left(\frac{k}{\sqrt{n}\sigma}\right)&=\int q\left(x_i\right)e^{i\frac{k}{\sqrt{n}\sigma}x_i}dx_i=\
        &=\int q\left(x_i\right)\left[1+i\frac{k}{\sqrt{n}\sigma}x_i-\frac{k^2}{2n\sigma^2}x_i^2+\mathcal{O}\left(\frac{1}{\sqrt{n^3}}\right)\right]dx_i=\\
        &=1+i\frac{k}{\sqrt{n}\sigma}\left\langle x_i\right\rangle-\frac{k^2}{2n\sigma^2}\left\langle x_i^2\right\rangle+\mathcal{O}\left(\frac{1}{\sqrt{n^3}}\right)
    \end{align*}

    Substituting $\mu=\left\langle x_i\right\rangle$ and $\sigma^2=\left\langle x_i^2\right\rangle-\left\langle x_i\right\rangle^2 \ \iff \ \left\langle x_i^2\right\rangle=\sigma^2+\mu^2$ we find:

    \begin{align*}
        \varphi_i\left(\frac{k}{\sqrt{n}\sigma}\right)&=1+i\frac{k}{\sqrt{n}\sigma}\mu-\frac{k^2}{2n\sigma^2}\left(\sigma^2+\mu^2\right)+\mathcal{O}\left(\frac{1}{\sqrt{n^3}}\right)=\\
        &=1+i\frac{k}{\sqrt{n}\sigma}\mu-\frac{k^2}{2n}-\frac{k^2}{2n\sigma^2}\mu^2+\mathcal{O}\left(\frac{1}{\sqrt{n^3}}\right)
    \end{align*}

    What is written is nothing more than the series expansion of the following exponential:

    \begin{align*}
        e^{i\frac{k}{\sqrt{n}\sigma}\mu-\frac{k^2}{2n}}&\simeq1+i\frac{k}{\sqrt{n}\sigma}\mu-\frac{k^2}{2n}+\frac{1}{2}\left(i\frac{k}{\sqrt{n}\sigma}\mu-\frac{k^2}{2n}\right)^2=\\
        &=1+i\frac{k}{\sqrt{n}\sigma}\mu-\frac{k^2}{2n}-\frac{k^2}{2n\sigma^2}\mu^2+\mathcal{O}\left(\frac{1}{\sqrt{n^3}}\right)
    \end{align*}

    Substituting in (eq:leggi-dei-grandi-numeri-funzione-caratteristica-limite-centrale) the characteristic function finally results in:

    \begin{equation*}
        \varphi(k)=e^{-i\frac{k\sqrt{n}}{\sigma}\mu}\cdot e^{i\frac{kn}{\sqrt{n}\sigma}\mu-\frac{k^2n}{2n}}=e^{-\cancel{i\frac{k\sqrt{n}}{\sigma}\mu}+\cancel{i\frac{k\sqrt{n}}{\sigma}\mu}-\frac{k^2}{2}}=e^{-\frac{k^2}{2}}
    \end{equation*}

    that is, the characteristic function associated with the normalized Gaussian with $\mu=0$ and $\sigma=1$, just as predicted by the theorem.

*Show that*

::: {.solution}
\begin{itemize}
        \item $\sum\limits_{i=1}^nx_i\sim\mathcal{N}\left(n\mu,n\sigma^2\right)$
        \item $\frac{1}{n}\sum\limits_{i=1}^nx_i\sim\mathcal{N}\left(\mu,\frac{\sigma^2}{n}\right)$
    \end{itemize}

    From the central limit theorem we know that

    \begin{equation*}
        \frac{\sum\limits_{i=1}^nx_i-n\mu}{\sqrt{n}\sigma}\sim\mathcal{N}(0,1)
    \end{equation*}

    Multiplying both sides by $\sqrt{n}\sigma$:

    \begin{equation*}
        \sum_{i=1}^nx_i-n\mu\sim\sqrt{n}\sigma\mathcal{N}(0,1)=\mathcal{N}\left(0,n\sigma^2\right)
    \end{equation*}

    Adding $n\mu$ to both sides:

    \begin{equation*}
        \sum_{i=1}^nx_i\sim\mathcal{N}\left(0,n\sigma^2\right)+n\mu=\mathcal{N}\left(n\mu,n\sigma^2\right)
    \end{equation*}

    In this way we have proved the first relation; to obtain the second one it is sufficient to divide both sides of the last equation by $n$:

    \begin{equation*}
        \frac{1}{n}\sum_{i=1}^nx_i\sim\frac{1}{n}\mathcal{N}\left(n\mu,n\sigma^2\right)=\mathcal{N}\left(\frac{n\mu}{n},\frac{n\sigma^2}{n^2}\right)=\mathcal{N}\left(\mu,\frac{\sigma^2}{n}\right)
    \end{equation*}
