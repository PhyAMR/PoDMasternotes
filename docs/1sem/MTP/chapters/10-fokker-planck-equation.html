12 Fokker-Planck Equation

Having defined and set up the stochastic differential equations, we are now interested in studying the statistics associated with the processes defined by these equations. To do this, and therefore to derive the various moments, we need to find the relative probability density functions.

12.1 Numerical solution

Intuitively, we must calculate how many trajectories randomly generated from the process \(x(t)\) are located between \(x\) and \(x+\Delta x\) at time \(t\).

Let’s therefore consider intervals of the same width that go from a minimum \(x_{min}\) to a maximum \(x_{max}\):

\[\begin{equation*} \left\{[x_{\min},x_{\min}+\Delta x],\ldots,[x_{\max}-\Delta x,x_{\max}]\right\} \end{equation*}\]

To write them as a function of a single index \(i\) we can define:

\[\begin{equation*} \begin{cases} i_{\min}=\dfrac{x_{\min}}{\Delta x}\\ i_{\max}=\dfrac{x_{\max}}{\Delta x} \end{cases} \Longrightarrow\qquad \left\{[x_i,x_i+\Delta x]\right\}_{i=i_{\min}}^{i_{\max}} \end{equation*}\]

Let’s imagine we have performed \(N\) simulations, and we want to count how many of these fall into the interval defined by \(x_j\). In this regard, we can define the indicator function:

\[\begin{equation} I(a,A)= \begin{cases} 1 \ \ \text{if} \ a\in A\\ 0 \ \ \text{if} \ a\notin A \end{cases} \end{equation}\]

In our case, the element \(a\) will be the \(j\)-th simulation at time \(t\), which we can indicate with \(x^{(j)}(t)\), while the interval is \(\left[x_i,x_i+\Delta x\right]\). Through the function just defined we can therefore count how many points among the total \(N\) fall into the interval, and finally normalize by \(N\).

\[\begin{equation*} \frac{1}{N}\sum_{j=1}^NI\left(x^{(j)}(t),\left[x_i,x_i+\Delta x\right]\right) \end{equation*}\]

The probability relative to the \(i\)-th interval at time \(t\) can be obtained by taking the limit \(N\to\infty\):

\[\begin{equation} P\left(x_i,t\right)=\lim_{N\to\infty}\frac{1}{N}\sum_{j=1}^NI\left(x^{(j)}(t),\left[x_i,x_i+\Delta x\right]\right) \end{equation}\]

The probability density is none other than the probability divided by the width of the considered interval, that is \(\Delta x\):

\[\begin{equation*} p\left(x_i,t\right)=\frac{P\left(x_i,t\right)}{\Delta x} \ \iff \ P\left(x_i,t\right)=p\left(x_i,t\right)\Delta x \end{equation*}\]

We thus find:

\[\begin{equation} \boxed{p\left(x_i,t\right)=\lim_{N\to\infty}\frac{1}{N\Delta x}\sum_{j=1}^NI\left(x^{(j)}(t),\left[x_i,x_i+\Delta x\right]\right)} \end{equation}\]

This formula is easily implementable at a computational level, since the indicator function is very easy to define and is not computationally very expensive. Obviously the result will be better approximated the larger \(N\) is, and also the smaller \(\Delta x\) is, since in this way the histogram tends to converge to the real distribution. At a theoretical level, in fact, we have that

\[\begin{equation*} \lim_{\Delta x\to0}\frac{I\left(x^{(j)}(t),\left[x_i,x_i+\Delta x\right]\right)}{\Delta x}=\delta\left(x_i-x^{(j)}(t)\right) \end{equation*}\]

since the interval shrinks until it becomes infinitesimal, thus becoming to all intents and purposes a Dirac delta. Passing therefore from the discrete domain with \(x_i\) to the continuous one with the generic variable \(x\), we will write:

\[\begin{equation} \boxed{p(x,t)=\lim_{N\to\infty}\frac{1}{N}\sum_{j=1}^N\delta\left(x-x^{(j)}(t)\right)} \end{equation}\]

In the case of Brownian motion, for example, we must expect a histogram that, with a sufficiently high number of trajectories and a high discretization of the intervals, correctly approximates a Gaussian centered at the origin and with variance increasing with \(t\).

12.2 Derivation for the Langevin equation

To obtain a general theoretical formula, we must first find an equation relative to \(p(x,t)\) which, when solved, will provide us with the pdf we need.

To obtain a formula, let’s consider a function \(f\) applied to the generic trajectory \(x(t)\), such that it satisfies two conditions:

In this way we can say that \(f(x(t))\) is twice differentiable, and also that for \(x\to\pm\infty\) both \(f(x(t))\) and its derivatives go to zero, since the support, as we have said, is limited.

To obtain a statistic on this function, let’s calculate the expected value, which by definition is given by

\[\begin{equation*} \langle f(x(t))\rangle=\lim_{N\to\infty}\sum_{i=1}^Nf\left(x^{(i)}(t)\right) \end{equation*}\]

Let’s rewrite \(f\left(x^{(i)}(t)\right)\) as a function of the Dirac delta:

\[\begin{equation*} \langle f(x(t))\rangle=\lim_{N\to\infty}\sum_{i=1}^N\int_{-\infty}^{+\infty}f(x)\delta\left(x-x^{(i)}(t)\right)dx \end{equation*}\]

We note at this point that we can compose the terms of the second member until we obtain (eq:fokker-planck-densita-con-delta):

\[\begin{equation*} \langle f(x(t))\rangle=\int_{-\infty}^{+\infty}f(x)\lim_{N\to\infty}\sum_{i=1}^N\delta\left(x-x^{(i)}(t)\right)dx \ \Longrightarrow \end{equation*}\]

\[\begin{equation*} \Longrightarrow \ \langle f(x(t))\rangle=\int_{-\infty}^{+\infty}f(x)p(x,t)dx \end{equation*}\]

Differentiating on the left and on the right with respect to time:

\[\begin{equation} \frac{d}{dt}\langle f(x(t))\rangle=\int_{-\infty}^{+\infty}f(x)\frac{\partial p}{\partial t}dx \end{equation}\]

The goal now is to find another expression for the derivative of the average value of \(f(x(t))\), this time as a function of \(p(x,t)\), so as to equate the two quantities and find a differential equation associated with the probability density.

We can expand the increment \(f(x+\Delta x)\) in a Taylor series:

\[\begin{equation*} f(x+\Delta x)=f(x)+\frac{df}{dx}\Delta x+\frac{1}{2}\frac{d^2f}{dx^2}\Delta x^2+\mathcal{O}\left(\Delta x^3\right) \end{equation*}\]

In temporal terms, \(x+\Delta x\) can be written as:

\[\begin{equation*} x(t)+\Delta x=x(t+\Delta t) \ \iff \ \Delta x=x(t+\Delta t)-x(t) \end{equation*}\]

At this point we must substitute \(\Delta x\) starting from the stochastic differential equation we are considering. Let’s choose the most general Langevin equation, the one shown in (eq:sde-equazione-di-langevin-generalizzata), in which the multiplicative noise is included; it follows that

\[\begin{equation} \Delta x=\mu(x,t)\Delta t+\sigma(x,t)\Delta B \end{equation}\]

being \(\Delta B=B(t+\Delta t)-B(t)\).

Returning to the series expansion, let’s bring \(f(x(t))\) to the first member and substitute the expression for \(\Delta x\):

\[\begin{equation*} f(x(t+\Delta t))-f(x(t))=\frac{df}{dx}(\mu\Delta t+\sigma\Delta B)+\frac{1}{2}\frac{d^2f}{dx^2}(\mu\Delta t+\sigma\Delta B)^2+\mathcal{O}\left(\Delta x^3\right) \end{equation*}\]

By definition, the first member is none other than \(\Delta f\). On the right, instead, let’s expand the square, remembering to ignore the terms of order higher than \(\Delta t\):

\[\begin{equation*} \Delta f=\frac{df}{dx}\mu\Delta t+\frac{df}{dx}\sigma\Delta B+\frac{1}{2}\frac{d^2f}{dx^2}\sigma^2\Delta B^2+\mathcal{O}\left(\Delta x^3\right) \end{equation*}\]

Remembering that \(\Delta B^2=\Delta t\), and applying the average value to both members, it results:

\[\begin{equation*} \langle\Delta f\rangle=\left\langle\frac{df}{dx}\mu\right\rangle\Delta t+\left\langle\frac{df}{dx}\sigma\Delta B\right\rangle+\frac{1}{2}\left\langle\frac{d^2f}{dx^2}\sigma^2\right\rangle\Delta t+\mathcal{O}\left(\Delta x^3\right) \end{equation*}\]

Thanks to the Itô condition:

\[\begin{equation*} \left\langle\frac{df}{dx}\sigma\Delta B\right\rangle=\left\langle\frac{df}{dx}\sigma\right\rangle\cancel{\langle\Delta B\rangle}=0 \end{equation*}\]

It follows that, by dividing both members by \(\Delta t\) and taking the limit for \(\Delta t\to0\), it results:

\[\begin{equation} \lim_{\Delta t\to0}\left\langle\frac{\Delta f}{\Delta t}\right\rangle=\left\langle\frac{df}{dx}\mu\right\rangle+\frac{1}{2}\left\langle\frac{d^2f}{dx^2}\sigma^2\right\rangle \end{equation}\]

We will have that

\[\begin{equation*} \lim_{\Delta t\to0}\left\langle\frac{\Delta f}{\Delta t}\right\rangle=\frac{d}{dt}\langle f(x(t))\rangle \end{equation*}\]

This means that we can equate the second member of (eq:fokker-planck-derivazione-prima-uguaglianza) with the second member of (eq:fokker-planck-derivazione-seconda-uguaglianza):

\[\begin{equation*} \int_{-\infty}^{+\infty}f(x)\frac{\partial p}{\partial t}dx=\left\langle\frac{df}{dx}\mu\right\rangle+\frac{1}{2}\left\langle\frac{d^2f}{dx^2}\sigma^2\right\rangle \end{equation*}\]

The second member, by definition, can be written as

\[\begin{equation*} \int_{-\infty}^{+\infty}f(x)\frac{\partial p}{\partial t}dx=\left\langle\frac{df}{dx}\mu+\frac{1}{2}\frac{d^2f}{dx^2}\sigma^2\right\rangle=\int_{-\infty}^{+\infty}\left[\frac{df}{dx}\mu+\frac{1}{2}\frac{d^2f}{dx^2}\sigma^2\right]p(x,t)dx \end{equation*}\]

Let’s decompose the integral on the right into a sum of integrals:

\[\begin{equation*} \int_{-\infty}^{+\infty}f(x)\frac{\partial p}{\partial t}dx=\int_{-\infty}^{+\infty}\frac{df}{dx}\mu(x,t)p(x,t)dx+\frac{1}{2}\int_{-\infty}^{+\infty}\frac{d^2f}{dx^2}\sigma(x,t)^2p(x,t)dx \end{equation*}\]

Let’s integrate both integrals on the right-hand side by parts:

\[\begin{equation*} \int_{-\infty}^{+\infty}\frac{df}{dx}\mu(x,t)p(x,t)dx=\cancel{f(x)\mu(x,t)p(x,t)\big|_{-\infty}^{+\infty}}-\int_{-\infty}^{+\infty}f(x)\frac{\partial}{\partial x}(\mu p)dx \end{equation*}\]

\[\begin{align*} \int_{-\infty}^{+\infty}\frac{d^2f}{dx^2}\sigma(x,t)^2p(x,t)dx&=\cancel{\frac{df}{dx}\sigma(x,t)^2p(x,t)\bigg|_{-\infty}^{+\infty}}-\int_{-\infty}^{+\infty}\frac{df}{dx}\frac{\partial}{\partial x}\left(\sigma^2p\right)dx=\\ &=-\cancel{f(x)\frac{\partial}{\partial x}\left(\sigma^2p\right)\bigg|_{-\infty}^{+\infty}}+\int_{-\infty}^{+\infty}f(x)\frac{\partial^2}{\partial x^2}\left(\sigma^2p\right)dx \end{align*}\]

Let’s substitute:

\[\begin{align*} \int_{-\infty}^{+\infty}f(x)\frac{\partial p}{\partial t}dx&=-\int_{-\infty}^{+\infty}f(x)\frac{\partial}{\partial x}(\mu p)dx+\frac{1}{2}\int_{-\infty}^{+\infty}f(x)\frac{\partial^2}{\partial x^2}\left(\sigma^2p\right)dx=\\ &=\int_{-\infty}^{+\infty}f(x)\left[-\frac{\partial}{\partial x}(\mu p)+\frac{1}{2}\frac{\partial^2}{\partial x^2}\left(\sigma^2p\right)\right]dx \end{align*}\]

The equality follows:

\[\begin{equation} \boxed{\frac{\partial p}{\partial t}=-\frac{\partial}{\partial x}(\mu p)+\frac{1}{2}\frac{\partial^2}{\partial x^2}\left(\sigma^2p\right)} \end{equation}\]

This is called the Fokker-Planck equation, also called the forward Fokker-Planck equation because it is opposed to the backward Kolmogorov equation, in which time instead flows backwards. Let’s remember that the equation just found is valid only for the Langevin equation and assuming the Itô condition is valid.

12.2.1 Conservation of probability

By setting

\[\begin{equation} J(x,t)=\mu(x,t)p(x,t)-\frac{1}{2}\frac{\partial}{\partial x}\left[\sigma(x,t)^2p(x,t)\right] \end{equation}\]

we note that the Fokker-Planck equation takes on a well-known form:

\[\begin{equation*} \frac{\partial}{\partial t}p(x,t)=-\frac{\partial}{\partial x}J(x,t) \end{equation*}\]

which corresponds to the continuity equation, characteristic of the phenomena of diffusion of currents in which there is local conservation of fluxes.

Integrating in a domain \(\mathcal{D}\subseteq\mathbb{R}\), in fact:

\[\begin{equation*} \frac{\partial}{\partial t}\int_{\mathcal{D}}p(x,t)dx=-\int_{\mathcal{D}}\frac{\partial J}{\partial x}dx=-J(x,t)\big|_{x\in\partial\mathcal{D}} \end{equation*}\]

where \(\partial\mathcal{D}\) is the boundary of the domain \(\mathcal{D}\). For the probability to be conserved over time, therefore, it is necessary that the flux, expressed by \(J(x,t)\), be zero at the boundary at all instants of time.

12.2.3 Equilibrium solution

For the equation to admit an equilibrium solution, it is necessary to assume that the functions \(\mu(x,t)\) and \(\sigma(x,t)\) do not explicitly depend on time, since the probability density \(p(x,t)\) explicitly depends on them.

By definition, a stationary solution of the Fokker-Planck equation is such that

\[\begin{equation} p^{st}(x)=\lim_{t\to\infty}p(x,t) \end{equation}\]

It follows that the time derivative of \(p^{st}(x)\) is zero, and by substituting in the Fokker-Planck equation we obtain:

\[\begin{equation*} -\frac{d}{dx}(\mu p^{st})+\frac{1}{2}\frac{d^2}{dx^2}\left(\sigma^2p^{st}\right)=0 \end{equation*}\]

Let’s factor out \(-\frac{d}{dx}\):

\[\begin{equation} -\frac{d}{dx}\left[\mu p^{st}-\frac{1}{2}\frac{d}{dx}\left(\sigma^2 p^{st}\right)\right]=0 \end{equation}\]

The term in square brackets is none other than the flux, as it was defined in (eq:fokker-planck-flusso-definizione). The derivative written above tells us that this flux must be constant, but for the solution to be stationary, the flux must not only be constant, but it must be zero:

\[\begin{equation} J^{st}(x)=0 \ \iff \ \mu p^{st}-\frac{1}{2}\frac{d}{dx}\left(\sigma^2p^{st}\right)=0 \end{equation}\]

To solve the differential equation, let’s isolate the derivative:

\[\begin{equation*} \frac{d}{dx}\left(\sigma^2p^{st}\right)=2\mu p^{st} \end{equation*}\]

Let’s multiply and divide on the right by \(\sigma^2\) so that we can separate the variables:

\[\begin{equation*} \frac{d}{dx}\left(\sigma^2p^{st}\right)=\frac{2\mu}{\sigma^2}\left(\sigma^2p^{st}\right) \ \Longrightarrow \ \int\frac{d\left(\sigma^2p^{st}\right)}{\sigma^2p^{st}}=2\int_{x_{min}}^x\frac{\mu(x')}{\sigma(x')^2}dx' \end{equation*}\]

\[\begin{equation*} \ln\left(\sigma^2p^{st}\right)=2\int_{x_{min}}^x\frac{\mu(x')}{\sigma(x')^2}dx'+c \end{equation*}\]

From which it follows:

\[\begin{equation*} \sigma(x)^2p^{st}(x)=ke^{2\int_{x_{min}}^x\frac{\mu(x')}{\sigma(x')^2}dx'} \ \Longrightarrow \ p^{st}(x)=\frac{k}{\sigma(x)^2}e^{2\int_{x_{min}}^x\frac{\mu(x')}{\sigma(x')^2}dx'} \end{equation*}\]

where \(k=e^{c}\).

To find \(k\) we must impose the normalization of the probability density:

\[\begin{equation*} \int_{x_{min}}^{x_{max}}p^{st}(x)dx=k\int_{x_{min}}^{x_{max}}\frac{1}{\sigma(x)^2}e^{2\int_{x_{min}}^{x_{max}}\frac{\mu(x')}{\sigma(x')^2}dx'}=1 \end{equation*}\]

Let’s set

\[\begin{equation*} Z=\frac{1}{k}=\int_{x_{min}}^{x_{max}}\frac{1}{\sigma(x)^2}e^{2\int_{x_{min}}^{x_{max}}\frac{\mu(x')}{\sigma(x')^2}dx'} \end{equation*}\]

so as to obtain the complete solution in the following form:

\[\begin{equation} \boxed{p^{st}(x)=\frac{1}{Z\sigma(x)^2}e^{2\int_{x_{min}}^x\frac{\mu(x')}{\sigma(x')^2}dx'}} \end{equation}\]

Let’s pay attention to the fact that an equilibrium solution does not necessarily exist: what we have shown is that, if it exists, it has the form shown in (eq:fokker-planck-soluzione-all-equilibrio-forma-finale). This solution, moreover, does not depend in any way on the initial conditions of the system; it is therefore an intrinsic property of the system, depending only on the functions \(\mu(x)\) and \(\sigma(x)\).

# Fokker-Planck Equation

Having defined and set up the stochastic differential equations, we are now interested in studying the *statistics* associated with the processes defined by these equations. To do this, and therefore to derive the various *moments*, we need to find the relative *probability density functions*.

## Numerical solution

Intuitively, we must calculate how many trajectories randomly generated from the process $x(t)$ are located between $x$ and $x+\Delta x$ at time $t$.

Let's therefore consider intervals of the same width that go from a minimum $x_{min}$ to a maximum $x_{max}$:

\begin{equation*}
\left\{[x_{\min},x_{\min}+\Delta x],\ldots,[x_{\max}-\Delta x,x_{\max}]\right\}
\end{equation*}

To write them as a function of a single index $i$ we can define:

\begin{equation*}
    \begin{cases}
        i_{\min}=\dfrac{x_{\min}}{\Delta x}\\
        i_{\max}=\dfrac{x_{\max}}{\Delta x}
    \end{cases}
    \Longrightarrow\qquad \left\{[x_i,x_i+\Delta x]\right\}_{i=i_{\min}}^{i_{\max}}
\end{equation*}

Let's imagine we have performed $N$ simulations, and we want to count how many of these fall into the interval defined by $x_j$. In this regard, we can define the **indicator function**:

\begin{equation}
    I(a,A)=
    \begin{cases}
        1 \ \ \text{if} \ a\in A\\
        0 \ \ \text{if} \ a\notin A
    \end{cases}
\end{equation}

In our case, the element $a$ will be the $j$-th simulation at time $t$, which we can indicate with $x^{(j)}(t)$, while the interval is $\left[x_i,x_i+\Delta x\right]$. Through the function just defined we can therefore count how many points among the total $N$ fall into the interval, and finally normalize by $N$.

\begin{equation*}
    \frac{1}{N}\sum_{j=1}^NI\left(x^{(j)}(t),\left[x_i,x_i+\Delta x\right]\right)
\end{equation*}

The probability relative to the $i$-th interval at time $t$ can be obtained by taking the limit $N\to\infty$:

\begin{equation}
    P\left(x_i,t\right)=\lim_{N\to\infty}\frac{1}{N}\sum_{j=1}^NI\left(x^{(j)}(t),\left[x_i,x_i+\Delta x\right]\right)
\end{equation}

The probability density is none other than the probability divided by the width of the considered interval, that is $\Delta x$:

\begin{equation*}
    p\left(x_i,t\right)=\frac{P\left(x_i,t\right)}{\Delta x} \ \iff \ P\left(x_i,t\right)=p\left(x_i,t\right)\Delta x
\end{equation*}

We thus find:

\begin{equation}
    \boxed{p\left(x_i,t\right)=\lim_{N\to\infty}\frac{1}{N\Delta x}\sum_{j=1}^NI\left(x^{(j)}(t),\left[x_i,x_i+\Delta x\right]\right)}
\end{equation}

This formula is easily implementable at a computational level, since the indicator function is very easy to define and is not computationally very expensive. Obviously the result will be better approximated the larger $N$ is, and also the smaller $\Delta x$ is, since in this way the histogram tends to converge to the real distribution. At a theoretical level, in fact, we have that

\begin{equation*}
    \lim_{\Delta x\to0}\frac{I\left(x^{(j)}(t),\left[x_i,x_i+\Delta x\right]\right)}{\Delta x}=\delta\left(x_i-x^{(j)}(t)\right)
\end{equation*}

since the interval shrinks until it becomes infinitesimal, thus becoming to all intents and purposes a Dirac delta. Passing therefore from the discrete domain with $x_i$ to the continuous one with the generic variable $x$, we will write:

\begin{equation}
    \boxed{p(x,t)=\lim_{N\to\infty}\frac{1}{N}\sum_{j=1}^N\delta\left(x-x^{(j)}(t)\right)}
\end{equation}

In the case of Brownian motion, for example, we must expect a histogram that, with a sufficiently high number of trajectories and a high discretization of the intervals, correctly approximates a Gaussian centered at the origin and with variance increasing with $t$.

## Derivation for the Langevin equation

To obtain a general theoretical formula, we must first find an equation relative to $p(x,t)$ which, when solved, will provide us with the pdf we need.

To obtain a formula, let's consider a function $f$ applied to the generic trajectory $x(t)$, such that it satisfies two conditions:

\begin{enumerate}
    \item $f$ must be *twice differentiable*:

    \begin{equation*}
        f\in C^2(\mathbb{R})
    \end{equation*}

    \item $f$ must have *limited support*; by *support* we mean the set of points where $f$ is non-zero, while by *compact* we mean that:

    \begin{itemize}
        \item the set is *closed*, that is, it includes all its accumulation points
        \item the set is *bounded*, that is, it is contained within a finite interval
    \end{itemize}
\end{enumerate}

In this way we can say that $f(x(t))$ is twice differentiable, and also that for $x\to\pm\infty$ both $f(x(t))$ and its derivatives go to zero, since the support, as we have said, is limited.

To obtain a statistic on this function, let's calculate the expected value, which by definition is given by

\begin{equation*}
    \langle f(x(t))\rangle=\lim_{N\to\infty}\sum_{i=1}^Nf\left(x^{(i)}(t)\right)
\end{equation*}

Let's rewrite $f\left(x^{(i)}(t)\right)$ as a function of the Dirac delta:

\begin{equation*}
    \langle f(x(t))\rangle=\lim_{N\to\infty}\sum_{i=1}^N\int_{-\infty}^{+\infty}f(x)\delta\left(x-x^{(i)}(t)\right)dx
\end{equation*}

We note at this point that we can compose the terms of the second member until we obtain (eq:fokker-planck-densita-con-delta):

\begin{equation*}
    \langle f(x(t))\rangle=\int_{-\infty}^{+\infty}f(x)\lim_{N\to\infty}\sum_{i=1}^N\delta\left(x-x^{(i)}(t)\right)dx \ \Longrightarrow
\end{equation*}

\begin{equation*}
    \Longrightarrow \ \langle f(x(t))\rangle=\int_{-\infty}^{+\infty}f(x)p(x,t)dx
\end{equation*}

Differentiating on the left and on the right with respect to time:

\begin{equation}
    \frac{d}{dt}\langle f(x(t))\rangle=\int_{-\infty}^{+\infty}f(x)\frac{\partial p}{\partial t}dx
\end{equation}

The goal now is to find another expression for the derivative of the average value of $f(x(t))$, this time as a function of $p(x,t)$, so as to equate the two quantities and find a differential equation associated with the probability density.

We can expand the increment $f(x+\Delta x)$ in a Taylor series:

\begin{equation*}
    f(x+\Delta x)=f(x)+\frac{df}{dx}\Delta x+\frac{1}{2}\frac{d^2f}{dx^2}\Delta x^2+\mathcal{O}\left(\Delta x^3\right)
\end{equation*}

In temporal terms, $x+\Delta x$ can be written as:

\begin{equation*}
    x(t)+\Delta x=x(t+\Delta t) \ \iff \ \Delta x=x(t+\Delta t)-x(t)
\end{equation*}

At this point we must substitute $\Delta x$ starting from the stochastic differential equation we are considering. Let's choose the most general Langevin equation, the one shown in (eq:sde-equazione-di-langevin-generalizzata), in which the multiplicative noise is included; it follows that

\begin{equation}
    \Delta x=\mu(x,t)\Delta t+\sigma(x,t)\Delta B
\end{equation}

being $\Delta B=B(t+\Delta t)-B(t)$.

Returning to the series expansion, let's bring $f(x(t))$ to the first member and substitute the expression for $\Delta x$:

\begin{equation*}
    f(x(t+\Delta t))-f(x(t))=\frac{df}{dx}(\mu\Delta t+\sigma\Delta B)+\frac{1}{2}\frac{d^2f}{dx^2}(\mu\Delta t+\sigma\Delta B)^2+\mathcal{O}\left(\Delta x^3\right)
\end{equation*}

By definition, the first member is none other than $\Delta f$. On the right, instead, let's expand the square, remembering to ignore the terms of order higher than $\Delta t$:

\begin{equation*}
    \Delta f=\frac{df}{dx}\mu\Delta t+\frac{df}{dx}\sigma\Delta B+\frac{1}{2}\frac{d^2f}{dx^2}\sigma^2\Delta B^2+\mathcal{O}\left(\Delta x^3\right)
\end{equation*}

Remembering that $\Delta B^2=\Delta t$, and applying the average value to both members, it results:

\begin{equation*}
    \langle\Delta f\rangle=\left\langle\frac{df}{dx}\mu\right\rangle\Delta t+\left\langle\frac{df}{dx}\sigma\Delta B\right\rangle+\frac{1}{2}\left\langle\frac{d^2f}{dx^2}\sigma^2\right\rangle\Delta t+\mathcal{O}\left(\Delta x^3\right)
\end{equation*}

Thanks to the Itô condition:

\begin{equation*}
    \left\langle\frac{df}{dx}\sigma\Delta B\right\rangle=\left\langle\frac{df}{dx}\sigma\right\rangle\cancel{\langle\Delta B\rangle}=0
\end{equation*}

It follows that, by dividing both members by $\Delta t$ and taking the limit for $\Delta t\to0$, it results:

\begin{equation}
    \lim_{\Delta t\to0}\left\langle\frac{\Delta f}{\Delta t}\right\rangle=\left\langle\frac{df}{dx}\mu\right\rangle+\frac{1}{2}\left\langle\frac{d^2f}{dx^2}\sigma^2\right\rangle
\end{equation}

We will have that

\begin{equation*}
    \lim_{\Delta t\to0}\left\langle\frac{\Delta f}{\Delta t}\right\rangle=\frac{d}{dt}\langle f(x(t))\rangle
\end{equation*}

This means that we can equate the second member of (eq:fokker-planck-derivazione-prima-uguaglianza) with the second member of (eq:fokker-planck-derivazione-seconda-uguaglianza):

\begin{equation*}
    \int_{-\infty}^{+\infty}f(x)\frac{\partial p}{\partial t}dx=\left\langle\frac{df}{dx}\mu\right\rangle+\frac{1}{2}\left\langle\frac{d^2f}{dx^2}\sigma^2\right\rangle
\end{equation*}

The second member, by definition, can be written as

\begin{equation*}
    \int_{-\infty}^{+\infty}f(x)\frac{\partial p}{\partial t}dx=\left\langle\frac{df}{dx}\mu+\frac{1}{2}\frac{d^2f}{dx^2}\sigma^2\right\rangle=\int_{-\infty}^{+\infty}\left[\frac{df}{dx}\mu+\frac{1}{2}\frac{d^2f}{dx^2}\sigma^2\right]p(x,t)dx
\end{equation*}

Let's decompose the integral on the right into a sum of integrals:

\begin{equation*}
    \int_{-\infty}^{+\infty}f(x)\frac{\partial p}{\partial t}dx=\int_{-\infty}^{+\infty}\frac{df}{dx}\mu(x,t)p(x,t)dx+\frac{1}{2}\int_{-\infty}^{+\infty}\frac{d^2f}{dx^2}\sigma(x,t)^2p(x,t)dx
\end{equation*}

Let's integrate both integrals on the right-hand side by parts:

\begin{equation*}
    \int_{-\infty}^{+\infty}\frac{df}{dx}\mu(x,t)p(x,t)dx=\cancel{f(x)\mu(x,t)p(x,t)\big|_{-\infty}^{+\infty}}-\int_{-\infty}^{+\infty}f(x)\frac{\partial}{\partial x}(\mu p)dx
\end{equation*}

\begin{align*}
    \int_{-\infty}^{+\infty}\frac{d^2f}{dx^2}\sigma(x,t)^2p(x,t)dx&=\cancel{\frac{df}{dx}\sigma(x,t)^2p(x,t)\bigg|_{-\infty}^{+\infty}}-\int_{-\infty}^{+\infty}\frac{df}{dx}\frac{\partial}{\partial x}\left(\sigma^2p\right)dx=\\
    &=-\cancel{f(x)\frac{\partial}{\partial x}\left(\sigma^2p\right)\bigg|_{-\infty}^{+\infty}}+\int_{-\infty}^{+\infty}f(x)\frac{\partial^2}{\partial x^2}\left(\sigma^2p\right)dx
\end{align*}

Let's substitute:

\begin{align*}
    \int_{-\infty}^{+\infty}f(x)\frac{\partial p}{\partial t}dx&=-\int_{-\infty}^{+\infty}f(x)\frac{\partial}{\partial x}(\mu p)dx+\frac{1}{2}\int_{-\infty}^{+\infty}f(x)\frac{\partial^2}{\partial x^2}\left(\sigma^2p\right)dx=\\
    &=\int_{-\infty}^{+\infty}f(x)\left[-\frac{\partial}{\partial x}(\mu p)+\frac{1}{2}\frac{\partial^2}{\partial x^2}\left(\sigma^2p\right)\right]dx
\end{align*}

The equality follows:

\begin{equation}
    \boxed{\frac{\partial p}{\partial t}=-\frac{\partial}{\partial x}(\mu p)+\frac{1}{2}\frac{\partial^2}{\partial x^2}\left(\sigma^2p\right)}
\end{equation}

This is called the **Fokker-Planck equation**, also called the *forward Fokker-Planck equation* because it is opposed to the *backward Kolmogorov equation*, in which time instead flows backwards. Let's remember that the equation just found is valid only for the **Langevin equation** and assuming the **Itô condition** is valid.

### Conservation of probability

By setting

\begin{equation}
    J(x,t)=\mu(x,t)p(x,t)-\frac{1}{2}\frac{\partial}{\partial x}\left[\sigma(x,t)^2p(x,t)\right]
\end{equation}

we note that the Fokker-Planck equation takes on a well-known form:

\begin{equation*}
    \frac{\partial}{\partial t}p(x,t)=-\frac{\partial}{\partial x}J(x,t)
\end{equation*}

which corresponds to the *continuity equation*, characteristic of the phenomena of diffusion of currents in which there is local conservation of fluxes.

Integrating in a domain $\mathcal{D}\subseteq\mathbb{R}$, in fact:

\begin{equation*}
    \frac{\partial}{\partial t}\int_{\mathcal{D}}p(x,t)dx=-\int_{\mathcal{D}}\frac{\partial J}{\partial x}dx=-J(x,t)\big|_{x\in\partial\mathcal{D}}
\end{equation*}

where $\partial\mathcal{D}$ is the boundary of the domain $\mathcal{D}$. For the probability to be conserved over time, therefore, it is necessary that the flux, expressed by $J(x,t)$, be zero at the boundary at all instants of time.

### Boundary conditions

If the process is defined in a domain $\mathcal{D}\subset\mathbb{R}$ then it is necessary to introduce **boundary conditions** that specify the behavior at the edges of the definition set.

A typical example is the one in which $\mathcal{D}=\mathbb{R}^+$, which we have already studied in section (sec:diffusione-in-un-dominio-semi-infinito) in the diffusion process. The possibilities, let's remember, are two:

\begin{itemize}
    \item **Absorbing boundary conditions**, in which the probability of finding the system at the boundary is zero:

    \begin{equation}
        p(x,t)\big|_{x\in\partial\mathcal{D}}=0   \forall t
    \end{equation}

    \item **Reflecting boundary conditions**, in which the flux at the boundary is zero:

    \begin{equation}
        J(x,t)\big|_{x\in\partial\mathcal{D}}=0   \forall t
    \end{equation}
\end{itemize}

### Equilibrium solution

For the equation to admit an equilibrium solution, it is necessary to assume that the functions $\mu(x,t)$ and $\sigma(x,t)$ do not explicitly depend on time, since the probability density $p(x,t)$ explicitly depends on them.

By definition, a stationary solution of the Fokker-Planck equation is such that

\begin{equation}
    p^{st}(x)=\lim_{t\to\infty}p(x,t)
\end{equation}

It follows that the time derivative of $p^{st}(x)$ is zero, and by substituting in the Fokker-Planck equation we obtain:

\begin{equation*}
    -\frac{d}{dx}(\mu p^{st})+\frac{1}{2}\frac{d^2}{dx^2}\left(\sigma^2p^{st}\right)=0
\end{equation*}

Let's factor out $-\frac{d}{dx}$:

\begin{equation}
    -\frac{d}{dx}\left[\mu p^{st}-\frac{1}{2}\frac{d}{dx}\left(\sigma^2 p^{st}\right)\right]=0
\end{equation}

The term in square brackets is none other than the flux, as it was defined in (eq:fokker-planck-flusso-definizione). The derivative written above tells us that this flux must be constant, but for the solution to be stationary, the flux must not only be constant, but it must be *zero*:

\begin{equation}
    J^{st}(x)=0 \ \iff \ \mu p^{st}-\frac{1}{2}\frac{d}{dx}\left(\sigma^2p^{st}\right)=0
\end{equation}

To solve the differential equation, let's isolate the derivative:

\begin{equation*}
    \frac{d}{dx}\left(\sigma^2p^{st}\right)=2\mu p^{st}
\end{equation*}

Let's multiply and divide on the right by $\sigma^2$ so that we can separate the variables:

\begin{equation*}
    \frac{d}{dx}\left(\sigma^2p^{st}\right)=\frac{2\mu}{\sigma^2}\left(\sigma^2p^{st}\right) \ \Longrightarrow \ \int\frac{d\left(\sigma^2p^{st}\right)}{\sigma^2p^{st}}=2\int_{x_{min}}^x\frac{\mu(x')}{\sigma(x')^2}dx'
\end{equation*}

\begin{equation*}
    \ln\left(\sigma^2p^{st}\right)=2\int_{x_{min}}^x\frac{\mu(x')}{\sigma(x')^2}dx'+c
\end{equation*}

From which it follows:

\begin{equation*}
    \sigma(x)^2p^{st}(x)=ke^{2\int_{x_{min}}^x\frac{\mu(x')}{\sigma(x')^2}dx'} \ \Longrightarrow \ p^{st}(x)=\frac{k}{\sigma(x)^2}e^{2\int_{x_{min}}^x\frac{\mu(x')}{\sigma(x')^2}dx'}
\end{equation*}

where $k=e^{c}$.

To find $k$ we must impose the normalization of the probability density:

\begin{equation*}
    \int_{x_{min}}^{x_{max}}p^{st}(x)dx=k\int_{x_{min}}^{x_{max}}\frac{1}{\sigma(x)^2}e^{2\int_{x_{min}}^{x_{max}}\frac{\mu(x')}{\sigma(x')^2}dx'}=1
\end{equation*}

Let's set

\begin{equation*}
    Z=\frac{1}{k}=\int_{x_{min}}^{x_{max}}\frac{1}{\sigma(x)^2}e^{2\int_{x_{min}}^{x_{max}}\frac{\mu(x')}{\sigma(x')^2}dx'}
\end{equation*}

so as to obtain the complete solution in the following form:

\begin{equation}
    \boxed{p^{st}(x)=\frac{1}{Z\sigma(x)^2}e^{2\int_{x_{min}}^x\frac{\mu(x')}{\sigma(x')^2}dx'}}
\end{equation}

Let's pay attention to the fact that an equilibrium solution does not necessarily exist: what we have shown is that, if it exists, it has the form shown in (eq:fokker-planck-soluzione-all-equilibrio-forma-finale). This solution, moreover, does not depend in any way on the initial conditions of the system;\footnote{Not by chance, we have not used any initial condition in the derivation of the final solution.} it is therefore an intrinsic property of the system, depending only on the functions $\mu(x)$ and $\sigma(x)$.
