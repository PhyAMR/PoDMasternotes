24.1 Classification of Stayed/Churned Customers

The Customer Churn table contains information on all 3,758 customers from a Telecommunications company in California in Q2 2022. Companies are naturally interested in churn, i.e., in which users are likely to switch to another company soon to get a better deal, and which are more loyal customers.

The dataset contains three features:

  • Tenure in Months: Number of months the customer has stayed with the company
  • Monthly Charge: The amount charged to the customer monthly
  • Age: Customer’s age

The aim of the task is to predict if a customer will churn or not based on the three features.

24.4 Decision tree

Now complete the class Tree and all auxiliary functions.

The input parameters to pass to the id3_training function are:

  • \(X\): the matrix of input features, one row for each sample
  • \(Y\): the vector of labels for the input features matrix X
  • \(max\_depth\): the maximum depth of the tree

Complete the class Tree: implement classify (point classification by traversing the tree), id3_training (iterate over features and thresholds to find the best entropy-minimising split, then recurse), and extra_training (same but with a random threshold per feature).

class Tree:

    def __init__(self):
        self.idx = -1    # The index of the feature over which you split (no split: -1)
        self.thresh = 0  # The threshold value over which you split (<=: left, >: right)
        self.leaf = 0    # 1 if it is a leaf of class 1, -1 if it is a leaf of class -1, 0 if it is an internal node
        self.left = []   # Left subtree (empty if it is a leaf)
        self.right = []  # Right subtree (empty if it is a leaf)


    def entropy(left, right):
        H = 0
        tot_length = len(left) + len(right)
        left_prob = len(np.where(left > 0)[0]) / len(left)
        if (left_prob > 0):
            H -= len(left) * left_prob * np.log2(left_prob) / tot_length
        if (left_prob < 1):
            H -= len(left) * (1 - left_prob) * np.log2(1 - left_prob) / tot_length
        right_prob = len(np.where(right > 0)[0]) / len(right)
        if (right_prob > 0):
            H -= len(right) * right_prob * np.log2(right_prob) / tot_length
        if (right_prob < 1):
            H -= len(right) * (1 - right_prob) * np.log2(1 - right_prob) / tot_length
        return H

    def classify(self, x):
        if (self.leaf == 0):
            if (x[self.idx] > self.thresh):
                return self.right.classify(x)
            else:
                return self.left.classify(x)
        else:
            return self.leaf

    def id3_training(self, X, Y, max_depth, printing):
        # Check if the node is a leaf (all nodes have the same label)
        if (np.max(Y) - np.min(Y) < 1e-3):
            self.leaf = np.max(Y)
            if (printing):
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all labels are the same over ' + str(len(Y)) + ' points)')
            return
        # If the maximum depth is 0, the node must be a leaf!
        if (max_depth < 1):
            if (printing):
                print('Remaining depth: ' + str(max_depth) + ', leaf node (maximum depth reached, ' + str(len(Y)) + ' points)')
            if (len(np.where(Y > 0)) > len(Y) / 2):
                self.leaf = 1
            else:
                self.leaf = -1
            return
        # Find the best split: iterate over features
        best_idx = -1
        best_thresh = -1
        best_entropy = 1e9
        for idx in range(X.shape[1]):
            values = X[:, idx]
            sorted_ind = np.argsort(values)
            values = np.unique(values[sorted_ind])
            # Iterate over all possible splits
            for j in range(len(values) - 1):
                thresh = (values[j] + values[j + 1]) / 2
                left = np.where(X[:, idx] <= thresh)[0]
                right = np.where(X[:, idx] > thresh)[0]
                if (len(left) == 0 or len(right) == 0):
                    print('error!',best_idx,thresh,values)
                H = Tree.entropy(Y[left], Y[right])
                if (H < best_entropy):
                    best_idx = idx
                    best_thresh = thresh
                    best_entropy = H
        if (best_idx == -1):
            # No valid features! The points are all identical
            self.leaf = np.sign(np.sum(Y))
            if (self.leaf == 0):
                self.leaf = 1
            if (printing):
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all inputs are the same over ' + str(len(Y)) + ' points)')
            return
        left_samples = np.where(X[:, best_idx] <= best_thresh)[0]
        right_samples = np.where(X[:, best_idx] > best_thresh)[0]
        if (printing):
            print('Remaining depth: ' + str(max_depth) + ', splitting ' + str(len(Y)) + ' elements into ' + str(len(left_samples)) + ' and ' + str(len(right_samples)) + ' over feature ' + str(best_idx))
        self.idx = best_idx
        self.thresh = best_thresh
        self.left = Tree()
        self.right = Tree()
        self.left.id3_training(X[left_samples, :], Y[left_samples], max_depth - 1, printing)
        self.right.id3_training(X[right_samples, :], Y[right_samples], max_depth - 1, printing)


    def extra_training(self, X, Y, max_depth, printing):
        # Check if the node is a leaf (all nodes have the same label)
        if (np.max(Y) - np.min(Y) < 1e-3):
            self.leaf = np.max(Y)
            if (printing):
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all labels are the same over ' + str(len(Y)) + ' points)')
            return
        # If the maximum depth is 0, the node must be a leaf!
        if (max_depth < 1):
            if (printing):
                print('Remaining depth: ' + str(max_depth) + ', leaf node (maximum depth reached, ' + str(len(Y)) + ' points)')
            if (len(np.where(Y > 0)) > len(Y) / 2):
                self.leaf = 1
            else:
                self.leaf = -1
            return
        # Find the best split: iterate over features
        best_idx = -1
        best_thresh = -1
        best_entropy = 1e9
        for idx in range(X.shape[1]):
            values = X[:, idx]
            sorted_ind = np.argsort(values)
            values = np.unique(values[sorted_ind])
            # Randomize the split!
            if (len(values) > 1):
                thresh = rnd.uniform(np.min(values), np.max(values))
                left = np.where(X[:, idx] <= thresh)[0]
                right = np.where(X[:, idx] > thresh)[0]
                H = Tree.entropy(left, right)
                if (H < best_entropy):
                    best_idx = idx
                    best_thresh = thresh
                    best_entropy = H
        if (best_idx == -1):
            # No valid features! The points are all identical
            self.leaf = np.sign(np.sum(Y))
            if (self.leaf == 0):
                self.leaf = 1
            if (printing):
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all inputs are the same over ' + str(len(Y)) + ' points)')
            return
        left_samples = np.where(X[:, best_idx] <= best_thresh)[0]
        right_samples = np.where(X[:, best_idx] > best_thresh)[0]
        if (printing):
            print('Remaining depth: ' + str(max_depth) + ', splitting ' + str(len(Y)) + ' elements into ' + str(len(left_samples)) + ' and ' + str(len(right_samples)) + ' over feature ' + str(best_idx))
        self.idx = best_idx
        self.thresh = best_thresh
        self.left = Tree()
        self.right = Tree()
        self.left.extra_training(X[left_samples, :], Y[left_samples], max_depth - 1, printing)
        self.right.extra_training(X[right_samples, :], Y[right_samples], max_depth - 1, printing)

Solution (vectorized solution). The per-point classify traversal stays recursive (each node decision depends on the previous one), but the inner threshold search of id3_training is the real hotspot and it vectorises cleanly: for each candidate feature we can compute the entropy of every candidate split simultaneously via cumulative counts of positives/negatives along the sorted-by-feature order. extra_training only tests one random threshold per feature, so we vectorise the per-feature entropy evaluation across all features at once with boolean masks.

Note: the recursion over nodes is inherently sequential, so we only vectorise the per-node split search and the impurity computations.

Show code 
class Tree:

    def __init__(self):
        self.idx = -1
        self.thresh = 0
        self.leaf = 0
        self.left = []
        self.right = []

    @staticmethod
    def entropy(left, right):
        # Vectorised weighted entropy of two children (labels in {-1,+1}).
        nL, nR = len(left), len(right)
        tot = nL + nR
        H = 0.0
        for child, n in ((left, nL), (right, nR)):
            if n == 0:
                continue
            p = np.mean(child > 0)
            probs = np.array([p, 1.0 - p])
            probs = probs[probs > 0]
            H -= (n / tot) * np.sum(probs * np.log2(probs))
        return H

    def classify(self, x):
        # Tree traversal: still recursive per point, but `x` is a vector op.
        if self.leaf == 0:
            child = self.right if x[self.idx] > self.thresh else self.left
            return child.classify(x)
        return self.leaf

    def _best_split_vectorised(self, X, Y):
        # For every (feature, candidate threshold) compute the weighted entropy
        # using cumulative counts of positives along the sorted feature order.
        n, d = X.shape
        pos = (Y > 0).astype(np.int64)
        total_pos = pos.sum()
        best_idx, best_thresh, best_H = -1, -1, np.inf

        for idx in range(d):
            col = X[:, idx]
            order = np.argsort(col, kind="mergesort")
            sorted_vals = col[order]
            sorted_pos = pos[order]
            # boundary mask: only consider splits between distinct values
            boundary = sorted_vals[:-1] != sorted_vals[1:]
            if not boundary.any():
                continue
            cum_pos = np.cumsum(sorted_pos)[:-1]            # positives on the left
            left_n = np.arange(1, n)                        # samples on the left
            right_n = n - left_n
            right_pos = total_pos - cum_pos

            # vectorised entropy of each potential split
            with np.errstate(divide="ignore", invalid="ignore"):
                pL = cum_pos / left_n
                pR = right_pos / right_n
                def h(p):
                    e = np.zeros_like(p, dtype=float)
                    m1 = (p > 0) & (p < 1)
                    e[m1] = -(p[m1] * np.log2(p[m1]) + (1 - p[m1]) * np.log2(1 - p[m1]))
                    return e
                H = (left_n * h(pL) + right_n * h(pR)) / n

            H = np.where(boundary, H, np.inf)
            j = int(np.argmin(H))
            if H[j] < best_H:
                best_H = float(H[j])
                best_idx = idx
                best_thresh = 0.5 * (sorted_vals[j] + sorted_vals[j + 1])
        return best_idx, best_thresh

    def id3_training(self, X, Y, max_depth, printing):
        if np.max(Y) - np.min(Y) < 1e-3:
            self.leaf = np.max(Y)
            if printing:
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all labels are the same over ' + str(len(Y)) + ' points)')
            return
        if max_depth < 1:
            if printing:
                print('Remaining depth: ' + str(max_depth) + ', leaf node (maximum depth reached, ' + str(len(Y)) + ' points)')
            self.leaf = 1 if np.sum(Y > 0) > len(Y) / 2 else -1
            return

        best_idx, best_thresh = self._best_split_vectorised(X, Y)
        if best_idx == -1:
            self.leaf = np.sign(np.sum(Y)) or 1
            if printing:
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all inputs are the same over ' + str(len(Y)) + ' points)')
            return

        mask_left = X[:, best_idx] <= best_thresh
        left_samples = np.where(mask_left)[0]
        right_samples = np.where(~mask_left)[0]
        if printing:
            print('Remaining depth: ' + str(max_depth) + ', splitting ' + str(len(Y)) + ' elements into ' + str(len(left_samples)) + ' and ' + str(len(right_samples)) + ' over feature ' + str(best_idx))
        self.idx = best_idx
        self.thresh = best_thresh
        self.left = Tree()
        self.right = Tree()
        self.left.id3_training(X[left_samples, :], Y[left_samples], max_depth - 1, printing)
        self.right.id3_training(X[right_samples, :], Y[right_samples], max_depth - 1, printing)

    def extra_training(self, X, Y, max_depth, printing):
        if np.max(Y) - np.min(Y) < 1e-3:
            self.leaf = np.max(Y)
            if printing:
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all labels are the same over ' + str(len(Y)) + ' points)')
            return
        if max_depth < 1:
            if printing:
                print('Remaining depth: ' + str(max_depth) + ', leaf node (maximum depth reached, ' + str(len(Y)) + ' points)')
            self.leaf = 1 if np.sum(Y > 0) > len(Y) / 2 else -1
            return

        # One random threshold per feature, evaluated in parallel.
        mins = X.min(axis=0)
        maxs = X.max(axis=0)
        valid = maxs > mins
        thresholds = np.where(valid, np.random.uniform(mins, maxs), np.nan)

        best_idx, best_thresh, best_H = -1, -1, np.inf
        for idx in range(X.shape[1]):
            if not valid[idx]:
                continue
            mask = X[:, idx] <= thresholds[idx]
            H = Tree.entropy(np.where(mask)[0], np.where(~mask)[0])
            if H < best_H:
                best_H = H
                best_idx = idx
                best_thresh = float(thresholds[idx])

        if best_idx == -1:
            self.leaf = np.sign(np.sum(Y)) or 1
            if printing:
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all inputs are the same over ' + str(len(Y)) + ' points)')
            return

        mask_left = X[:, best_idx] <= best_thresh
        left_samples = np.where(mask_left)[0]
        right_samples = np.where(~mask_left)[0]
        if printing:
            print('Remaining depth: ' + str(max_depth) + ', splitting ' + str(len(Y)) + ' elements into ' + str(len(left_samples)) + ' and ' + str(len(right_samples)) + ' over feature ' + str(best_idx))
        self.idx = best_idx
        self.thresh = best_thresh
        self.left = Tree()
        self.right = Tree()
        self.left.extra_training(X[left_samples, :], Y[left_samples], max_depth - 1, printing)
        self.right.extra_training(X[right_samples, :], Y[right_samples], max_depth - 1, printing)

Solution (solution).

Show code 
class Forest:

    def __init__(self, trees, n_features, n_samples, max_depth):
        self.forest = []
        self.features = n_features
        self.max_depth = max_depth
        self.samples = n_samples
        for i in range(trees):
            self.forest.append(Tree())

    def classify(self, x):
        vote = 0
        for tree in self.forest:
            vote += tree.classify(x)
        return np.sign(vote)

    def train(self, X, Y):
        for tree in self.forest:
            X_train, Y_train = self.bag(X, Y)
            tree.id3_training(X_train, Y_train, self.max_depth, False)

    def bag(self, X, Y):
        features = X.shape[1]
        points = X.shape[0]
        # Bagging: sample with replacement
        bagged = rnd.choices(range(points), k = self.samples)
        X_bagged = X[bagged, :]
        # Remove features that are not part of the tree
        selected = rnd.sample(range(features), k = self.features)
        for i in range(features):
            if i not in selected:
                X_bagged[:, i] = 0
        return X_bagged, Y[bagged]

Solution (vectorized solution). Ensemble aggregation and bagging vectorise nicely with np.random.randint, fancy indexing and np.sign(sum(...)). Classifying a whole batch at once is also a clean reduction across trees.

Show code 
class Forest:

    def __init__(self, trees, n_features, n_samples, max_depth):
        self.forest = [Tree() for _ in range(trees)]
        self.features = n_features
        self.max_depth = max_depth
        self.samples = n_samples

    def classify(self, x):
        # Vectorised aggregation over trees (same signature: classify a single point).
        votes = np.fromiter((tree.classify(x) for tree in self.forest),
                            dtype=np.int64, count=len(self.forest))
        return np.sign(votes.sum())

    def train(self, X, Y):
        for tree in self.forest:
            X_train, Y_train = self.bag(X, Y)
            tree.id3_training(X_train, Y_train, self.max_depth, False)

    def bag(self, X, Y):
        n_features = X.shape[1]
        n_points = X.shape[0]
        # Sample-with-replacement indices via NumPy
        bagged = np.random.randint(0, n_points, size=self.samples)
        X_bagged = X[bagged, :].copy()
        # Random feature subset, zero out the rest with a boolean mask
        selected = np.random.choice(n_features, size=self.features, replace=False)
        keep_mask = np.zeros(n_features, dtype=bool)
        keep_mask[selected] = True
        X_bagged[:, ~keep_mask] = 0
        return X_bagged, Y[bagged]

Complete the class `Tree`: implement `classify` (point classification by traversing the tree), `id3_training` (iterate over features and thresholds to find the best entropy-minimising split, then recurse), and `extra_training` (same but with a random threshold per feature).

```{python}
class Tree:

    def __init__(self):
        self.idx = -1    # The index of the feature over which you split (no split: -1)
        self.thresh = 0  # The threshold value over which you split (<=: left, >: right)
        self.leaf = 0    # 1 if it is a leaf of class 1, -1 if it is a leaf of class -1, 0 if it is an internal node
        self.left = []   # Left subtree (empty if it is a leaf)
        self.right = []  # Right subtree (empty if it is a leaf)


    def entropy(left, right):
        H = 0
        tot_length = len(left) + len(right)
        left_prob = len(np.where(left > 0)[0]) / len(left)
        if (left_prob > 0):
            H -= len(left) * left_prob * np.log2(left_prob) / tot_length
        if (left_prob < 1):
            H -= len(left) * (1 - left_prob) * np.log2(1 - left_prob) / tot_length
        right_prob = len(np.where(right > 0)[0]) / len(right)
        if (right_prob > 0):
            H -= len(right) * right_prob * np.log2(right_prob) / tot_length
        if (right_prob < 1):
            H -= len(right) * (1 - right_prob) * np.log2(1 - right_prob) / tot_length
        return H

    def classify(self, x):
        if (self.leaf == 0):
            if (x[self.idx] > self.thresh):
                return self.right.classify(x)
            else:
                return self.left.classify(x)
        else:
            return self.leaf

    def id3_training(self, X, Y, max_depth, printing):
        # Check if the node is a leaf (all nodes have the same label)
        if (np.max(Y) - np.min(Y) < 1e-3):
            self.leaf = np.max(Y)
            if (printing):
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all labels are the same over ' + str(len(Y)) + ' points)')
            return
        # If the maximum depth is 0, the node must be a leaf!
        if (max_depth < 1):
            if (printing):
                print('Remaining depth: ' + str(max_depth) + ', leaf node (maximum depth reached, ' + str(len(Y)) + ' points)')
            if (len(np.where(Y > 0)) > len(Y) / 2):
                self.leaf = 1
            else:
                self.leaf = -1
            return
        # Find the best split: iterate over features
        best_idx = -1
        best_thresh = -1
        best_entropy = 1e9
        for idx in range(X.shape[1]):
            values = X[:, idx]
            sorted_ind = np.argsort(values)
            values = np.unique(values[sorted_ind])
            # Iterate over all possible splits
            for j in range(len(values) - 1):
                thresh = (values[j] + values[j + 1]) / 2
                left = np.where(X[:, idx] <= thresh)[0]
                right = np.where(X[:, idx] > thresh)[0]
                if (len(left) == 0 or len(right) == 0):
                    print('error!',best_idx,thresh,values)
                H = Tree.entropy(Y[left], Y[right])
                if (H < best_entropy):
                    best_idx = idx
                    best_thresh = thresh
                    best_entropy = H
        if (best_idx == -1):
            # No valid features! The points are all identical
            self.leaf = np.sign(np.sum(Y))
            if (self.leaf == 0):
                self.leaf = 1
            if (printing):
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all inputs are the same over ' + str(len(Y)) + ' points)')
            return
        left_samples = np.where(X[:, best_idx] <= best_thresh)[0]
        right_samples = np.where(X[:, best_idx] > best_thresh)[0]
        if (printing):
            print('Remaining depth: ' + str(max_depth) + ', splitting ' + str(len(Y)) + ' elements into ' + str(len(left_samples)) + ' and ' + str(len(right_samples)) + ' over feature ' + str(best_idx))
        self.idx = best_idx
        self.thresh = best_thresh
        self.left = Tree()
        self.right = Tree()
        self.left.id3_training(X[left_samples, :], Y[left_samples], max_depth - 1, printing)
        self.right.id3_training(X[right_samples, :], Y[right_samples], max_depth - 1, printing)


    def extra_training(self, X, Y, max_depth, printing):
        # Check if the node is a leaf (all nodes have the same label)
        if (np.max(Y) - np.min(Y) < 1e-3):
            self.leaf = np.max(Y)
            if (printing):
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all labels are the same over ' + str(len(Y)) + ' points)')
            return
        # If the maximum depth is 0, the node must be a leaf!
        if (max_depth < 1):
            if (printing):
                print('Remaining depth: ' + str(max_depth) + ', leaf node (maximum depth reached, ' + str(len(Y)) + ' points)')
            if (len(np.where(Y > 0)) > len(Y) / 2):
                self.leaf = 1
            else:
                self.leaf = -1
            return
        # Find the best split: iterate over features
        best_idx = -1
        best_thresh = -1
        best_entropy = 1e9
        for idx in range(X.shape[1]):
            values = X[:, idx]
            sorted_ind = np.argsort(values)
            values = np.unique(values[sorted_ind])
            # Randomize the split!
            if (len(values) > 1):
                thresh = rnd.uniform(np.min(values), np.max(values))
                left = np.where(X[:, idx] <= thresh)[0]
                right = np.where(X[:, idx] > thresh)[0]
                H = Tree.entropy(left, right)
                if (H < best_entropy):
                    best_idx = idx
                    best_thresh = thresh
                    best_entropy = H
        if (best_idx == -1):
            # No valid features! The points are all identical
            self.leaf = np.sign(np.sum(Y))
            if (self.leaf == 0):
                self.leaf = 1
            if (printing):
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all inputs are the same over ' + str(len(Y)) + ' points)')
            return
        left_samples = np.where(X[:, best_idx] <= best_thresh)[0]
        right_samples = np.where(X[:, best_idx] > best_thresh)[0]
        if (printing):
            print('Remaining depth: ' + str(max_depth) + ', splitting ' + str(len(Y)) + ' elements into ' + str(len(left_samples)) + ' and ' + str(len(right_samples)) + ' over feature ' + str(best_idx))
        self.idx = best_idx
        self.thresh = best_thresh
        self.left = Tree()
        self.right = Tree()
        self.left.extra_training(X[left_samples, :], Y[left_samples], max_depth - 1, printing)
        self.right.extra_training(X[right_samples, :], Y[right_samples], max_depth - 1, printing)
```

::: {.solution name="vectorized solution"}

The per-point `classify` traversal stays recursive (each node decision depends on the previous one), but the inner *threshold search* of `id3_training` is the real hotspot and it vectorises cleanly: for each candidate feature we can compute the entropy of *every* candidate split simultaneously via cumulative counts of positives/negatives along the sorted-by-feature order. `extra_training` only tests one random threshold per feature, so we vectorise the per-feature entropy evaluation across all features at once with boolean masks.

*Note: the recursion over nodes is inherently sequential, so we only vectorise the per-node split search and the impurity computations.*

```{python}
class Tree:

    def __init__(self):
        self.idx = -1
        self.thresh = 0
        self.leaf = 0
        self.left = []
        self.right = []

    @staticmethod
    def entropy(left, right):
        # Vectorised weighted entropy of two children (labels in {-1,+1}).
        nL, nR = len(left), len(right)
        tot = nL + nR
        H = 0.0
        for child, n in ((left, nL), (right, nR)):
            if n == 0:
                continue
            p = np.mean(child > 0)
            probs = np.array([p, 1.0 - p])
            probs = probs[probs > 0]
            H -= (n / tot) * np.sum(probs * np.log2(probs))
        return H

    def classify(self, x):
        # Tree traversal: still recursive per point, but `x` is a vector op.
        if self.leaf == 0:
            child = self.right if x[self.idx] > self.thresh else self.left
            return child.classify(x)
        return self.leaf

    def _best_split_vectorised(self, X, Y):
        # For every (feature, candidate threshold) compute the weighted entropy
        # using cumulative counts of positives along the sorted feature order.
        n, d = X.shape
        pos = (Y > 0).astype(np.int64)
        total_pos = pos.sum()
        best_idx, best_thresh, best_H = -1, -1, np.inf

        for idx in range(d):
            col = X[:, idx]
            order = np.argsort(col, kind="mergesort")
            sorted_vals = col[order]
            sorted_pos = pos[order]
            # boundary mask: only consider splits between distinct values
            boundary = sorted_vals[:-1] != sorted_vals[1:]
            if not boundary.any():
                continue
            cum_pos = np.cumsum(sorted_pos)[:-1]            # positives on the left
            left_n = np.arange(1, n)                        # samples on the left
            right_n = n - left_n
            right_pos = total_pos - cum_pos

            # vectorised entropy of each potential split
            with np.errstate(divide="ignore", invalid="ignore"):
                pL = cum_pos / left_n
                pR = right_pos / right_n
                def h(p):
                    e = np.zeros_like(p, dtype=float)
                    m1 = (p > 0) & (p < 1)
                    e[m1] = -(p[m1] * np.log2(p[m1]) + (1 - p[m1]) * np.log2(1 - p[m1]))
                    return e
                H = (left_n * h(pL) + right_n * h(pR)) / n

            H = np.where(boundary, H, np.inf)
            j = int(np.argmin(H))
            if H[j] < best_H:
                best_H = float(H[j])
                best_idx = idx
                best_thresh = 0.5 * (sorted_vals[j] + sorted_vals[j + 1])
        return best_idx, best_thresh

    def id3_training(self, X, Y, max_depth, printing):
        if np.max(Y) - np.min(Y) < 1e-3:
            self.leaf = np.max(Y)
            if printing:
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all labels are the same over ' + str(len(Y)) + ' points)')
            return
        if max_depth < 1:
            if printing:
                print('Remaining depth: ' + str(max_depth) + ', leaf node (maximum depth reached, ' + str(len(Y)) + ' points)')
            self.leaf = 1 if np.sum(Y > 0) > len(Y) / 2 else -1
            return

        best_idx, best_thresh = self._best_split_vectorised(X, Y)
        if best_idx == -1:
            self.leaf = np.sign(np.sum(Y)) or 1
            if printing:
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all inputs are the same over ' + str(len(Y)) + ' points)')
            return

        mask_left = X[:, best_idx] <= best_thresh
        left_samples = np.where(mask_left)[0]
        right_samples = np.where(~mask_left)[0]
        if printing:
            print('Remaining depth: ' + str(max_depth) + ', splitting ' + str(len(Y)) + ' elements into ' + str(len(left_samples)) + ' and ' + str(len(right_samples)) + ' over feature ' + str(best_idx))
        self.idx = best_idx
        self.thresh = best_thresh
        self.left = Tree()
        self.right = Tree()
        self.left.id3_training(X[left_samples, :], Y[left_samples], max_depth - 1, printing)
        self.right.id3_training(X[right_samples, :], Y[right_samples], max_depth - 1, printing)

    def extra_training(self, X, Y, max_depth, printing):
        if np.max(Y) - np.min(Y) < 1e-3:
            self.leaf = np.max(Y)
            if printing:
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all labels are the same over ' + str(len(Y)) + ' points)')
            return
        if max_depth < 1:
            if printing:
                print('Remaining depth: ' + str(max_depth) + ', leaf node (maximum depth reached, ' + str(len(Y)) + ' points)')
            self.leaf = 1 if np.sum(Y > 0) > len(Y) / 2 else -1
            return

        # One random threshold per feature, evaluated in parallel.
        mins = X.min(axis=0)
        maxs = X.max(axis=0)
        valid = maxs > mins
        thresholds = np.where(valid, np.random.uniform(mins, maxs), np.nan)

        best_idx, best_thresh, best_H = -1, -1, np.inf
        for idx in range(X.shape[1]):
            if not valid[idx]:
                continue
            mask = X[:, idx] <= thresholds[idx]
            H = Tree.entropy(np.where(mask)[0], np.where(~mask)[0])
            if H < best_H:
                best_H = H
                best_idx = idx
                best_thresh = float(thresholds[idx])

        if best_idx == -1:
            self.leaf = np.sign(np.sum(Y)) or 1
            if printing:
                print('Remaining depth: ' + str(max_depth) + ', leaf node (all inputs are the same over ' + str(len(Y)) + ' points)')
            return

        mask_left = X[:, best_idx] <= best_thresh
        left_samples = np.where(mask_left)[0]
        right_samples = np.where(~mask_left)[0]
        if printing:
            print('Remaining depth: ' + str(max_depth) + ', splitting ' + str(len(Y)) + ' elements into ' + str(len(left_samples)) + ' and ' + str(len(right_samples)) + ' over feature ' + str(best_idx))
        self.idx = best_idx
        self.thresh = best_thresh
        self.left = Tree()
        self.right = Tree()
        self.left.extra_training(X[left_samples, :], Y[left_samples], max_depth - 1, printing)
        self.right.extra_training(X[right_samples, :], Y[right_samples], max_depth - 1, printing)
```

```{python}
class Forest:

    def __init__(self, trees, n_features, n_samples, max_depth):
        self.forest = []
        self.features = n_features
        self.max_depth = max_depth
        self.samples = n_samples
        for i in range(trees):
            self.forest.append(Tree())

    def classify(self, x):
        vote = 0
        for tree in self.forest:
            vote += tree.classify(x)
        return np.sign(vote)

    def train(self, X, Y):
        for tree in self.forest:
            X_train, Y_train = self.bag(X, Y)
            tree.id3_training(X_train, Y_train, self.max_depth, False)

    def bag(self, X, Y):
        features = X.shape[1]
        points = X.shape[0]
        # Bagging: sample with replacement
        bagged = rnd.choices(range(points), k = self.samples)
        X_bagged = X[bagged, :]
        # Remove features that are not part of the tree
        selected = rnd.sample(range(features), k = self.features)
        for i in range(features):
            if i not in selected:
                X_bagged[:, i] = 0
        return X_bagged, Y[bagged]
```

::: {.solution name="vectorized solution"}

Ensemble aggregation and bagging vectorise nicely with `np.random.randint`, fancy indexing and `np.sign(sum(...))`. Classifying a whole batch at once is also a clean reduction across trees.

```{python}
class Forest:

    def __init__(self, trees, n_features, n_samples, max_depth):
        self.forest = [Tree() for _ in range(trees)]
        self.features = n_features
        self.max_depth = max_depth
        self.samples = n_samples

    def classify(self, x):
        # Vectorised aggregation over trees (same signature: classify a single point).
        votes = np.fromiter((tree.classify(x) for tree in self.forest),
                            dtype=np.int64, count=len(self.forest))
        return np.sign(votes.sum())

    def train(self, X, Y):
        for tree in self.forest:
            X_train, Y_train = self.bag(X, Y)
            tree.id3_training(X_train, Y_train, self.max_depth, False)

    def bag(self, X, Y):
        n_features = X.shape[1]
        n_points = X.shape[0]
        # Sample-with-replacement indices via NumPy
        bagged = np.random.randint(0, n_points, size=self.samples)
        X_bagged = X[bagged, :].copy()
        # Random feature subset, zero out the rest with a boolean mask
        selected = np.random.choice(n_features, size=self.features, replace=False)
        keep_mask = np.zeros(n_features, dtype=bool)
        keep_mask[selected] = True
        X_bagged[:, ~keep_mask] = 0
        return X_bagged, Y[bagged]
```
