Complete count_outliers so it returns a tuple containing two numbers: first, the number of total outliers (distance below the margin); second, the number of misclassified outliers. Complete find_margin so it returns the margin for the selected model. Finally, complete sgd_soft_svm: initialise the weights, compute the current weights, take a stochastic gradient step on a random sample, and average the iterates over the final averaging_iterations steps.

Solution (solution).

Show code 
def count_outliers(current_w, X, Y):
    # Find all outliers (distance below the margin)
    Y_pred = np.dot(X, current_w)
    margin = Y_pred * Y
    outliers = np.where(margin < 1)[0]
    misclassified = np.where(margin < 0)[0]
    no = len(outliers)
    nm = len(misclassified)
    return no, nm


def find_margin(current_w):
    norm = np.sqrt(np.sum(np.asarray(current_w) ** 2))
    return 1 / np.max([1e-9, norm])

def sgd_soft_svm(X, Y, lambda_par, max_num_iterations, averaging_iterations):
    # Initialize the weights of the algorith with w=0
    theta = np.zeros(X.shape[1])
    best_w = np.zeros(X.shape[1])
    num_samples = X.shape[0]

    # Loop the SGD algorithm
    for num_iter in range(max_num_iterations):
        # Compute the current weights
        curr_w = theta / lambda_par / (num_iter + 1)
        # Compute the gradient over a random point
        idx = rnd.sample(range(num_samples), 1)[0]
        if (np.dot(X[idx, :], curr_w) * Y[idx] < 1):
            theta += Y[idx] * X[idx, :]
        if (num_iter >= max_num_iterations - averaging_iterations):
            best_w += curr_w / averaging_iterations

    margin = find_margin(best_w)
    outliers, misclassified = count_outliers(best_w, X, Y)
    return best_w, margin, outliers, misclassified

Solution (vectorized solution). count_outliers and find_margin are pure broadcasts and collapse to one-liners. sgd_soft_svm itself stays loop-based: each Pegasos step reads the iterate produced by the previous step, so the recurrence cannot be unrolled into a single NumPy expression without changing the algorithm. We keep the loop and only vectorise the helpers.

A vectorised solution for sgd_soft_svm is not possible — the SGD update is sequential by design (each step uses the previous weights).

Show code 
def count_outliers(current_w, X, Y):
    # One-line broadcast: score every sample, compare against margin / sign in one go.
    margins = (X @ current_w) * Y
    return int(np.sum(margins < 1)), int(np.sum(margins < 0))


def find_margin(current_w):
    # One-line: reciprocal of the (clipped) L2 norm.
    return 1.0 / max(1e-9, float(np.linalg.norm(current_w)))


def sgd_soft_svm(X, Y, lambda_par, max_num_iterations, averaging_iterations):
    # Pegasos-style SGD: the update is inherently sequential, but we can pre-draw
    # all random indices in one numpy call and reuse the vectorised helpers above.
    theta = np.zeros(X.shape[1])
    best_w = np.zeros(X.shape[1])
    num_samples = X.shape[0]
    idxs = np.random.randint(0, num_samples, size=max_num_iterations)

    for num_iter in range(max_num_iterations):
        curr_w = theta / (lambda_par * (num_iter + 1))
        idx = int(idxs[num_iter])
        if X[idx] @ curr_w * Y[idx] < 1:
            theta += Y[idx] * X[idx]
        if num_iter >= max_num_iterations - averaging_iterations:
            best_w += curr_w / averaging_iterations

    margin = find_margin(best_w)
    outliers, misclassified = count_outliers(best_w, X, Y)
    return best_w, margin, outliers, misclassified

Write the perceptron randomized update function perceptron_randomized_update, and write the perceptron main loop perceptron_with_randomization. The perceptron should run for up to max_num_iterations, or stop if it finds a solution with ERM = 0.

def perceptron_randomized_update(current_w, X, Y):
    new_w=current_w
    n, idx = count_errors(current_w, X, Y)
    # Choose a random misclassified sample
    if (n > 0):
        chosen = rnd.sample(sorted(idx), 1)[0]
        new_w = current_w + Y[chosen] * X[chosen,:]
    return new_w

def count_errors(current_w, X, Y):
    # Find all indices which have a different sign from the corresponding labels
    index = np.nonzero(np.sign(np.dot(X, current_w)) - Y)[0]
    n = np.array(index).shape[0]
    if (n == 0):
        # There are no misclassified samples
        return 0, -1
    return n, index

def perceptron_with_randomization(X, Y, max_num_iterations):

    # Initialize some support variables
    num_samples = X.shape[0]
    num_iter = 0

    # Initialize the weights of the algorith with w=0
    curr_w = np.zeros(X.shape[1])

    # Compute the number of misclassified samples and the indexes
    num_misclassified, index_misclassified = count_errors(curr_w, X, Y)
    # Initialize the best model
    best_error = num_misclassified / num_samples
    best_w = curr_w.copy()

    # Main loop: continue until all samples correctly classified or max # iterations reached
    # Remember: if no errors were found, index_misclassified = -1
    while num_misclassified != 0 and num_iter < max_num_iterations:
        # Update the perceptron
        curr_w = perceptron_randomized_update(curr_w, X, Y)
        # Find the new error
        num_misclassified, index_misclassified = count_errors(curr_w, X, Y)
        if num_misclassified / num_samples < best_error:
            best_error = num_misclassified / num_samples
            best_w = curr_w.copy()
        num_iter += 1

    return best_w, best_error

Solution (vectorized solution). count_errors is a textbook broadcast. The randomized update is a single sample mutation, so we only collapse the bookkeeping. The outer loop is sequential by construction (each update reads the previous weight vector), but we can still avoid Python overhead inside count_errors and perceptron_randomized_update.

A vectorised solution for perceptron_with_randomization is not possible — each update depends on the misclassified set induced by the previous weights, which forces sequential execution.

Show code 
def count_errors(current_w, X, Y):
    # One-line broadcast: indices where sign(score) disagrees with the label.
    index = np.flatnonzero(np.sign(X @ current_w) != Y)
    if index.size == 0:
        return 0, -1
    return int(index.size), index


def perceptron_randomized_update(current_w, X, Y):
    n, idx = count_errors(current_w, X, Y)
    if n == 0:
        return current_w
    chosen = int(np.random.choice(idx))
    return current_w + Y[chosen] * X[chosen]


def perceptron_with_randomization(X, Y, max_num_iterations):
    num_samples = X.shape[0]
    curr_w = np.zeros(X.shape[1])

    num_misclassified, _ = count_errors(curr_w, X, Y)
    best_error = num_misclassified / num_samples
    best_w = curr_w.copy()

    num_iter = 0
    while num_misclassified != 0 and num_iter < max_num_iterations:
        curr_w = perceptron_randomized_update(curr_w, X, Y)
        num_misclassified, _ = count_errors(curr_w, X, Y)
        err = num_misclassified / num_samples
        if err < best_error:
            best_error = err
            best_w = curr_w.copy()
        num_iter += 1

    return best_w, best_error

def K_fold(X_training: np.ndarray, Y_training: np.ndarray, lambda_vec: np.ndarray, K: np.ndarray) -> None:
    perm = np.arange(0, np.shape(X_training)[0], 1)
    rnd.shuffle(perm)
    X_shuffled = np.asarray(X_training[perm, :])
    Y_shuffled = np.asarray(Y_training[perm])
    best = -1
    best_perf = 1e20
    results = []
    models = []
    max_idx = np.shape(Y_training)[0]
    # Divide training set in K folds
    fold_points = int(np.floor(max_idx / K))
    for lambda_idx in range(len(lambda_vec)):
        lambda_perf = 0
        for test in range(K):
            x_train_fold = np.concatenate([X_shuffled[: fold_points * test, :], X_shuffled[fold_points * (test + 1) + 1 :, :]])
            y_train_fold = np.concatenate([Y_shuffled[: fold_points * test], Y_shuffled[fold_points * (test + 1) + 1 :]])
            x_validate_fold = X_shuffled[fold_points * test : fold_points * (test + 1) + 1, :]
            y_validate_fold = Y_shuffled[fold_points * test : fold_points * (test + 1) + 1]
            k_model, k_margin, k_out, k_misc = sgd_soft_svm(x_train_fold, y_train_fold, lambda_par[lambda_idx], int(1e5), int(1e4))
            lambda_perf += count_outliers(k_model, x_validate_fold, y_validate_fold)[1] / len(y_validate_fold)
        lambda_perf /= K
        results.append(lambda_perf)
        models.append(sgd_soft_svm(X_shuffled, Y_shuffled, lambda_par[lambda_idx], int(1e5), int(1e4)))
        if (best_perf > lambda_perf):
            best = sgd_soft_svm(X_shuffled, Y_shuffled, lambda_par[lambda_idx], int(1e5), int(1e4))
            best_perf = lambda_perf
    return best, best_perf, models, results

Solution (vectorized solution). The K-fold loop ultimately calls sgd_soft_svm, which is sequential. We still rewrite the slicing as a single boolean-mask broadcast (no per-fold concatenate) and reuse the vectorised helpers, but the outer structure remains.

A vectorised solution for K_fold is not possible — it dispatches the sequential Pegasos SGD on every fold, so the inner training itself cannot be unrolled.

Show code 
def K_fold(X_training: np.ndarray, Y_training: np.ndarray, lambda_vec, K: int):
    n = X_training.shape[0]
    perm = np.random.permutation(n)
    X_shuffled = X_training[perm]
    Y_shuffled = Y_training[perm]

    fold_points = n // K
    # Pre-compute fold assignment as a single broadcast over indices.
    fold_id = np.minimum(np.arange(n) // fold_points, K - 1)

    best = -1
    best_perf = 1e20
    results = []
    models = []

    for lam in lambda_vec:
        lambda_perf = 0.0
        for k in range(K):
            val_mask = fold_id == k
            x_train_fold = X_shuffled[~val_mask]
            y_train_fold = Y_shuffled[~val_mask]
            x_validate_fold = X_shuffled[val_mask]
            y_validate_fold = Y_shuffled[val_mask]
            k_model, _, _, _ = sgd_soft_svm(x_train_fold, y_train_fold, lam, int(1e5), int(1e4))
            lambda_perf += count_outliers(k_model, x_validate_fold, y_validate_fold)[1] / len(y_validate_fold)
        lambda_perf /= K
        results.append(lambda_perf)
        full_model = sgd_soft_svm(X_shuffled, Y_shuffled, lam, int(1e5), int(1e4))
        models.append(full_model)
        if lambda_perf < best_perf:
            best = full_model
            best_perf = lambda_perf
    return best, best_perf, models, results

Complete `find_margin` so it returns the margin for the selected model. Finally, complete `sgd_soft_svm`: initialise the weights, compute the current weights, take a stochastic gradient step on a random sample, and average the iterates over the final `averaging_iterations` steps.

::: {.solution name="solution"}
```{python}
def count_outliers(current_w, X, Y):
    # Find all outliers (distance below the margin)
    Y_pred = np.dot(X, current_w)
    margin = Y_pred * Y
    outliers = np.where(margin < 1)[0]
    misclassified = np.where(margin < 0)[0]
    no = len(outliers)
    nm = len(misclassified)
    return no, nm


def find_margin(current_w):
    norm = np.sqrt(np.sum(np.asarray(current_w) ** 2))
    return 1 / np.max([1e-9, norm])

def sgd_soft_svm(X, Y, lambda_par, max_num_iterations, averaging_iterations):
    # Initialize the weights of the algorith with w=0
    theta = np.zeros(X.shape[1])
    best_w = np.zeros(X.shape[1])
    num_samples = X.shape[0]

    # Loop the SGD algorithm
    for num_iter in range(max_num_iterations):
        # Compute the current weights
        curr_w = theta / lambda_par / (num_iter + 1)
        # Compute the gradient over a random point
        idx = rnd.sample(range(num_samples), 1)[0]
        if (np.dot(X[idx, :], curr_w) * Y[idx] < 1):
            theta += Y[idx] * X[idx, :]
        if (num_iter >= max_num_iterations - averaging_iterations):
            best_w += curr_w / averaging_iterations

    margin = find_margin(best_w)
    outliers, misclassified = count_outliers(best_w, X, Y)
    return best_w, margin, outliers, misclassified
```

::: {.solution name="vectorized solution"}
`count_outliers` and `find_margin` are pure broadcasts and collapse to one-liners. `sgd_soft_svm` itself stays loop-based: each Pegasos step reads the iterate produced by the previous step, so the recurrence cannot be unrolled into a single NumPy expression without changing the algorithm. We keep the loop and only vectorise the helpers.

*A vectorised solution for `sgd_soft_svm` is not possible — the SGD update is sequential by design (each step uses the previous weights).*

```{python}
def count_outliers(current_w, X, Y):
    # One-line broadcast: score every sample, compare against margin / sign in one go.
    margins = (X @ current_w) * Y
    return int(np.sum(margins < 1)), int(np.sum(margins < 0))


def find_margin(current_w):
    # One-line: reciprocal of the (clipped) L2 norm.
    return 1.0 / max(1e-9, float(np.linalg.norm(current_w)))


def sgd_soft_svm(X, Y, lambda_par, max_num_iterations, averaging_iterations):
    # Pegasos-style SGD: the update is inherently sequential, but we can pre-draw
    # all random indices in one numpy call and reuse the vectorised helpers above.
    theta = np.zeros(X.shape[1])
    best_w = np.zeros(X.shape[1])
    num_samples = X.shape[0]
    idxs = np.random.randint(0, num_samples, size=max_num_iterations)

    for num_iter in range(max_num_iterations):
        curr_w = theta / (lambda_par * (num_iter + 1))
        idx = int(idxs[num_iter])
        if X[idx] @ curr_w * Y[idx] < 1:
            theta += Y[idx] * X[idx]
        if num_iter >= max_num_iterations - averaging_iterations:
            best_w += curr_w / averaging_iterations

    margin = find_margin(best_w)
    outliers, misclassified = count_outliers(best_w, X, Y)
    return best_w, margin, outliers, misclassified
```

```{python}
    index = np.nonzero(np.sign(np.dot(X, current_w)) - Y)[0]

    num_samples = X.shape[0]
    num_iter = 0

    # Initialize the weights of the algorith with w=0

    # Compute the number of misclassified samples and the indexes
    num_misclassified, index_misclassified = count_errors(curr_w, X, Y)
    # Initialize the best model
    best_error = num_misclassified / num_samples
    best_w = curr_w.copy()

    # Main loop: continue until all samples correctly classified or max # iterations reached
    # Remember: if no errors were found, index_misclassified = -1

        # Update the perceptron
```

`count_errors` is a textbook broadcast. The randomized update is a single sample mutation, so we only collapse the bookkeeping. The outer loop is sequential by construction (each update reads the previous weight vector), but we can still avoid Python overhead inside `count_errors` and `perceptron_randomized_update`.

```{python}
def count_errors(current_w, X, Y):
    # One-line broadcast: indices where sign(score) disagrees with the label.
    index = np.flatnonzero(np.sign(X @ current_w) != Y)
    if index.size == 0:
        return 0, -1
    return int(index.size), index


def perceptron_randomized_update(current_w, X, Y):
    n, idx = count_errors(current_w, X, Y)
    if n == 0:
        return current_w
    chosen = int(np.random.choice(idx))
    return current_w + Y[chosen] * X[chosen]


def perceptron_with_randomization(X, Y, max_num_iterations):
    num_samples = X.shape[0]
    curr_w = np.zeros(X.shape[1])

    num_misclassified, _ = count_errors(curr_w, X, Y)
    best_error = num_misclassified / num_samples
    best_w = curr_w.copy()

    num_iter = 0
    while num_misclassified != 0 and num_iter < max_num_iterations:
        curr_w = perceptron_randomized_update(curr_w, X, Y)
        num_misclassified, _ = count_errors(curr_w, X, Y)
        err = num_misclassified / num_samples
        if err < best_error:
            best_error = err
            best_w = curr_w.copy()
        num_iter += 1

    return best_w, best_error
```

```{python}
    perm = np.arange(0, np.shape(X_training)[0], 1)
    rnd.shuffle(perm)
    X_shuffled = np.asarray(X_training[perm, :])
    Y_shuffled = np.asarray(Y_training[perm])

    best_perf = 1e20
    results = []
    models = []
    max_idx = np.shape(Y_training)[0]

    fold_points = int(np.floor(max_idx / K))

        lambda_perf = 0
        for test in range(K):

            y_train_fold = np.concatenate([Y_shuffled[: fold_points * test], Y_shuffled[fold_points * (test + 1) + 1 :]])
            x_validate_fold = X_shuffled[fold_points * test : fold_points * (test + 1) + 1, :]
```

::: {.solution name="vectorized solution"}
The K-fold loop ultimately calls `sgd_soft_svm`, which is sequential. We still rewrite the slicing as a single boolean-mask broadcast (no per-fold `concatenate`) and reuse the vectorised helpers, but the outer structure remains.

*A vectorised solution for `K_fold` is not possible — it dispatches the sequential Pegasos SGD on every fold, so the inner training itself cannot be unrolled.*

```{python}
def K_fold(X_training: np.ndarray, Y_training: np.ndarray, lambda_vec, K: int):
    n = X_training.shape[0]
    perm = np.random.permutation(n)
    X_shuffled = X_training[perm]
    Y_shuffled = Y_training[perm]

    fold_points = n // K
    # Pre-compute fold assignment as a single broadcast over indices.
    fold_id = np.minimum(np.arange(n) // fold_points, K - 1)

    best = -1
    best_perf = 1e20
    results = []
    models = []

    for lam in lambda_vec:
        lambda_perf = 0.0
        for k in range(K):
            val_mask = fold_id == k
            x_train_fold = X_shuffled[~val_mask]
            y_train_fold = Y_shuffled[~val_mask]
            x_validate_fold = X_shuffled[val_mask]
            y_validate_fold = Y_shuffled[val_mask]
            k_model, _, _, _ = sgd_soft_svm(x_train_fold, y_train_fold, lam, int(1e5), int(1e4))
            lambda_perf += count_outliers(k_model, x_validate_fold, y_validate_fold)[1] / len(y_validate_fold)
        lambda_perf /= K
        results.append(lambda_perf)
        full_model = sgd_soft_svm(X_shuffled, Y_shuffled, lam, int(1e5), int(1e4))
        models.append(full_model)
        if lambda_perf < best_perf:
            best = full_model
            best_perf = lambda_perf
    return best, best_perf, models, results
```
