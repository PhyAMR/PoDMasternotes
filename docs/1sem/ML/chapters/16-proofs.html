17.1 1. PAC Learning Theorem for Finite Hypothesis Classes

Let \(H\) be a finite hypothesis class, and let \(\delta, \epsilon \in (0,1)\) and \(m \in \mathbb{N}\) such that \(m \geq \frac{1}{\epsilon^2} \log\left(\frac{|H|}{\delta}\right)\). Then, for any learning rule and for any distribution \(D\) on \(X \times Y\) for which the realizability assumption is valid, if \(h_S = \arg\min_{h \in H} L_S(h)\) is returned by ERM on a training set \(S\) of \(m\) i.i.d. samples from \(D\), with probability at least \(1-\delta\) it holds that \(L_D(h_S) \leq \epsilon\).

Solution. Let us define the set of bad hypotheses as \(H_B = \{h \in H : L_D(h) > \epsilon\}\). Under the realizability assumption, every bad hypothesis must have \(L_S(h) > 0\) (i.e., at least one training error). We bound the probability that ERM returns a bad hypothesis:

\[\mathbb{P}^m(M_B) = \mathbb{P}^m\left(\exists h \in H_B : L_S(h) = 0\right) \leq \sum_{h \in H_B} \mathbb{P}^m(L_S(h) = 0)\]

Each misclassified point on \(S\) occurs with probability at most \(1-\epsilon\) (since \(L_D(h) > \epsilon\)). Therefore, the probability that a bad hypothesis is correct on all \(m\) samples is at most \((1-\epsilon)^m \leq e^{-\epsilon m}\).

Summing over all bad hypotheses: \[\mathbb{P}^m(M_B) \leq |H_B| e^{-\epsilon m} \leq |H| e^{-\epsilon m}\]

Setting this probability equal to \(\delta\) and solving for \(m\): \[|H| e^{-\epsilon m} \leq \delta \quad \Rightarrow \quad -\epsilon m \leq \log(\delta/|H|) \quad \Rightarrow \quad m \geq \frac{1}{\epsilon}\log(|H|/\delta)\]

Thus, with \(m \geq \frac{1}{\epsilon^2}\log(|H|/\delta)\) samples, with probability at least \(1-\delta\), the ERM hypothesis \(h_S\) satisfies \(L_D(h_S) \leq \epsilon\). \(\square\)

17.2 2. Lemma: Representativeness of the Sample

Let \(S\) be an \(\frac{\epsilon}{2}\)-representative sample with respect to domain \(Z\), hypothesis class \(H\), loss function \(\ell\), and distribution \(D\). Then any output of \(\text{ERM}_H(S)\), i.e., \(h_S \in \arg\min_{h \in H} L_S(h)\), satisfies: \[L_D(h_S) \leq \min_{h \in H} L_D(h) + \epsilon\]

Solution. The proof proceeds in three steps.

Step 1. Using the \(\frac{\epsilon}{2}\)-representativeness of \(S\), for \(h_S\): \[|L_S(h_S) - L_D(h_S)| \leq \frac{\epsilon}{2} \quad \Rightarrow \quad L_D(h_S) \leq L_S(h_S) + \frac{\epsilon}{2}\]

Step 2. Since \(h_S\) is obtained from ERM, it minimizes the empirical risk: \[L_S(h_S) \leq L_S(h) \quad \text{for all } h \in H\]

Therefore: \(L_D(h_S) \leq L_S(h) + \frac{\epsilon}{2}\).

Step 3. Again using \(\frac{\epsilon}{2}\)-representativeness, for any \(h \in H\): \[|L_S(h) - L_D(h)| \leq \frac{\epsilon}{2} \quad \Rightarrow \quad L_S(h) \leq L_D(h) + \frac{\epsilon}{2}\]

Substituting into the bound from Step 2: \[L_D(h_S) \leq L_D(h) + \frac{\epsilon}{2} + \frac{\epsilon}{2} = L_D(h) + \epsilon\]

Since this holds for all \(h \in H\), we can choose the one minimizing \(L_D(h)\): \[L_D(h_S) \leq \min_{h \in H} L_D(h) + \epsilon \quad \square\]

17.3 3. Agnostic PAC Learning Theorem for Finite \(H\)

A hypothesis class \(H\) is agnostic PAC learnable if there exists a function \(m_H : (0,1)^2 \to \mathbb{N}\) and a learning algorithm such that for every \(\epsilon, \delta \in (0,1)\), for every distribution \(D\) on \(X \times Y\), when the algorithm is run on \(m \geq m_H(\epsilon, \delta)\) i.i.d. samples from \(D\), it returns a hypothesis \(h\) such that with probability at least \(1-\delta\): \[L_D(h) \leq \min_{h' \in H} L_D(h') + \epsilon\]

For finite \(H\), this requires \(m_H(\epsilon, \delta) = O\left(\frac{\log(|H|/\delta)}{\epsilon^2}\right)\).

Solution. We prove this using uniform convergence and Hoeffding’s inequality.

Uniform Convergence Suffices: By the Representativeness Lemma, if \(S\) is \(\frac{\epsilon}{2}\)-representative (i.e., \(|L_S(h) - L_D(h)| \leq \frac{\epsilon}{2}\) for all \(h \in H\)), then any ERM output \(h_S\) satisfies \(L_D(h_S) \leq \min_{h \in H} L_D(h) + \epsilon\).

Apply Union Bound: The failure probability is: \[\mathbb{P}\left(\exists h \in H : |L_S(h) - L_D(h)| > \frac{\epsilon}{2}\right) \leq \sum_{h \in H} \mathbb{P}\left(|L_S(h) - L_D(h)| > \frac{\epsilon}{2}\right)\]

Hoeffding’s Inequality: For a fixed \(h\), the loss values \(\ell(h, z_i)\) are i.i.d. and bounded in \([0,1]\). By Hoeffding’s inequality: \[\mathbb{P}\left(|L_S(h) - L_D(h)| > \frac{\epsilon}{2}\right) \leq 2\exp\left(-2m\left(\frac{\epsilon}{2}\right)^2\right) = 2\exp\left(-\frac{m\epsilon^2}{2}\right)\]

Combine the Bounds: \[\mathbb{P}\left(\exists h \in H : |L_S(h) - L_D(h)| > \frac{\epsilon}{2}\right) \leq 2|H|\exp\left(-\frac{m\epsilon^2}{2}\right)\]

Derive Sample Complexity: Requiring this to be at most \(\delta\): \[2|H|\exp\left(-\frac{m\epsilon^2}{2}\right) \leq \delta \quad \Rightarrow \quad m \geq \frac{2\log(2|H|/\delta)}{\epsilon^2}\]

With this many samples, with probability \(\geq 1-\delta\), every hypothesis is \(\frac{\epsilon}{2}\)-representative, so ERM returns a hypothesis satisfying the agnostic PAC guarantee. \(\square\)

17.4 4. Hoeffding’s Inequality

Let \(Z_1, \ldots, Z_m\) be i.i.d. random variables taking values in \([a, b]\), and let \(\mu = \mathbb{E}[Z_i]\). Then for any \(\delta > 0\): \[\mathbb{P}\left(|\frac{1}{m}\sum_{i=1}^m Z_i - \mu| > \delta\right) \leq 2\exp\left(-\frac{2m\delta^2}{(b-a)^2}\right)\]

Solution. This is a fundamental concentration inequality. The proof uses Markov’s inequality applied to the exponential of the empirical mean. For the 0-1 loss (where \(a=0, b=1\)), this yields: \[\mathbb{P}(|L_S(h) - L_D(h)| > \epsilon) \leq 2\exp(-2m\epsilon^2)\]

We cite this result as a building block (see Understanding Machine Learning, Lemma B.6). \(\square\)

17.5 5. Sauer-Shelah-Perles Lemma

Let \(H\) be a hypothesis class with VC dimension \(d = \text{VCdim}(H)\). Then the growth function satisfies: \[\tau_H(m) \leq \sum_{i=0}^{d} \binom{m}{i}\]

For \(m \geq d\), this implies \(\tau_H(m) \leq (em/d)^d\).

Solution. Proof Sketch (Shifting Argument): We prove by induction on \(m\) that if \(\text{VCdim}(H) = d < \infty\), then \(\tau_H(m) \leq \sum_{i=0}^{d} \binom{m}{i}\).

Base Case: For \(m \leq d\), all sets of size \(m\) can be shattered, so \(\tau_H(m) = 2^m = \sum_{i=0}^{m} \binom{m}{i} \leq \sum_{i=0}^{d} \binom{m}{i}\) (since the RHS is larger when \(m \leq d\)).

Inductive Step: Fix a set \(C\) of size \(m > d\). Partition the dichotomies (binary labelings) consistent with \(H\): - Let \(D_0\) be dichotomies where all points in \(C\) get one assignment. - Consider a point \(x \in C\) and partition based on behavior on \(x\).

By the definition of VC dimension, the set \(C \setminus \{x\}\) has size \(m-1\), so by induction, there are at most \(\sum_{i=0}^{d} \binom{m-1}{i}\) dichotomies on \(C \setminus \{x\}\).

The total number of dichotomies on \(C\) is bounded by dichotomies agreeing on \(x\) plus those disagreeing, yielding: \[\tau_H(m) \leq \sum_{i=0}^{d} \binom{m-1}{i} + \sum_{i=0}^{d} \binom{m-1}{i} = \sum_{i=0}^{d} \binom{m}{i}\]

Corollary: For \(m \geq d\), by Stirling’s approximation: \[\sum_{i=0}^{d} \binom{m}{i} \leq (em/d)^d\]

\(\square\)

17.6 6. Fundamental Theorem of Statistical Learning (Theorem 6.8)

Let \(H\) be a hypothesis class of functions from \(X\) to \(\{0, 1\}\) with 0-1 loss. The following six conditions are equivalent:

  1. \(H\) has the uniform convergence property.
  2. Every ERM algorithm is an agnostic PAC learner for \(H\).
  3. \(H\) is agnostic PAC learnable.
  4. \(H\) is PAC learnable.
  5. Every ERM algorithm is a PAC learner for \(H\).
  6. \(H\) has finite VC dimension.

Solution. Direction 1 \(\Rightarrow\) 2 (Uniform Convergence to Agnostic PAC): Uniform convergence means that with probability \(\geq 1-\delta\), all hypotheses satisfy \(|L_S(h) - L_D(h)| \leq \epsilon\) simultaneously. By the Representativeness Lemma, this implies any ERM hypothesis satisfies \(L_D(h_S) \leq \min_h L_D(h) + \epsilon\), so ERM is an agnostic PAC learner.

Direction 2 \(\Rightarrow\) 3 (ERM Agnostic PAC \(\Rightarrow\) H Agnostic PAC): Trivial: if the ERM algorithm is agnostic PAC, then \(H\) is agnostic PAC learnable (by definition).

Direction 3 \(\Rightarrow\) 4 (Agnostic PAC \(\Rightarrow\) PAC): PAC learning is the special case of agnostic PAC learning when the realizability assumption holds. If \(H\) is agnostic PAC learnable with sample complexity \(m_H(\epsilon, \delta)\), then under realizability, running the same algorithm on \(m_H(\epsilon, \delta)\) samples produces a hypothesis with \(L_D(h) \leq 0 + \epsilon = \epsilon\), so \(H\) is PAC learnable.

Direction 1 \(\Rightarrow\) 5 (Uniform Convergence to PAC via ERM): Under the realizability assumption, if uniform convergence holds, then with probability \(\geq 1-\delta\), every bad hypothesis (with \(L_D(h) > \epsilon\)) must have \(L_S(h) > 0\). ERM will never choose such a hypothesis, so ERM returns a PAC hypothesis.

Direction 4 \(\Rightarrow\) 6 (PAC Learnability \(\Rightarrow\) Finite VC Dimension): Suppose \(H\) has infinite VC dimension. Then for arbitrarily large \(m\), there exists a set \(C\) of size \(2m\) that is shattered by \(H\). By the No-Free-Lunch theorem, there exists a distribution \(D\) on \(C\) such that any learning algorithm fails to PAC-learn \(H\) with sample size \(m\). This contradicts the assumption that \(H\) is PAC learnable. Therefore, \(H\) must have finite VC dimension.

Direction 6 \(\Rightarrow\) 1 (Finite VC Dimension \(\Rightarrow\) Uniform Convergence): Let \(d = \text{VCdim}(H)\). By the Sauer-Shelah-Perles lemma and concentration inequalities (see Theorem 6.11), uniform convergence holds with sample complexity: \[m_{UC}(\epsilon, \delta) = O\left(\frac{d + \log(1/\delta)}{\epsilon^2}\right)\]

Therefore, finite VC dimension implies uniform convergence, completing the cycle. \(\square\)

17.7 7. Structural Risk Minimisation Bound

Let \(w : \mathbb{N} \to [0,1]\) be a function such that \(\sum_{i=1}^{\infty} w(i) \leq 1\), and let \(H_1, H_2, \ldots\) be a sequence of hypothesis classes such that \(H = \bigcup_{n=1}^{\infty} H_n\). Then for every \(\delta \in (0,1)\), \(\epsilon \in (0,1)\), and for every distribution \(D\) on \(X \times Y\), with probability at least \(1-\delta\), for every training set \(S\) of size \(m\) i.i.d. from \(D\), and for every hypothesis \(h \in H_n\) for some \(n\): \[L_D(h) \leq L_S(h) + 2\min_n \epsilon(m, w(n)\delta)\]

Solution. Setup: Write \(H = \bigcup_{n \in \mathbb{N}} H_n\). For each \(n\), set \(\delta_n := w(n)\delta\), so \(\sum_{n=1}^{\infty} \delta_n = \delta \sum_{n=1}^{\infty} w(n) \leq \delta\).

Uniform Convergence on Each Subclass: By uniform convergence (Corollary 4.6 applied to \(H_n\)), for each fixed \(n\): \[\mathbb{P}\left(\exists h \in H_n : |L_S(h) - L_D(h)| > \epsilon(m, \delta_n)\right) \leq \delta_n\]

Union Bound Over Subclasses: Applying the union bound over all \(n \in \mathbb{N}\): \[\mathbb{P}\left(\exists n, \exists h \in H_n : |L_S(h) - L_D(h)| > \epsilon(m, \delta_n)\right) \leq \sum_{n=1}^{\infty} \delta_n \leq \delta\]

Conclusion: With probability \(\geq 1-\delta\), for every \(h \in H_n\): \[|L_D(h) - L_S(h)| \leq \epsilon(m, \delta_n) = \epsilon(m, w(n)\delta)\]

Rearranging yields the claimed bound. \(\square\)

17.8 8. Novikoff Perceptron Convergence Bound

Let \((x_1, y_1), \ldots, (x_m, y_m)\) be a linearly separable dataset, and let \(w^* \in \mathbb{R}^d\) be a separating hyperplane (i.e., \(y_i \langle w^*, x_i \rangle \geq 1\) for all \(i\)). Define \(B = \|w^*\|\) and \(R = \max_i \|x_i\|\). Then the perceptron algorithm converges in at most \((RB)^2\) iterations.

Solution. Let \(w^{(t)}\) be the weight vector after \(t\) iterations, and suppose the algorithm stops after \(T\) iterations. We prove two key bounds.

Part 1: Lower Bound on Numerator. Consider the telescoping sum: \[\langle w^*, w^{(T+1)} \rangle = \sum_{t=1}^T \left(\langle w^*, w^{(t+1)} \rangle - \langle w^*, w^{(t)} \rangle\right)\]

At each update step, we update \(w^{(t+1)} = w^{(t)} + y_i x_i\) for some misclassified example \((x_i, y_i)\). Thus: \[\langle w^*, w^{(t+1)} \rangle - \langle w^*, w^{(t)} \rangle = y_i \langle w^*, x_i \rangle \geq 1\]

Summing over all \(T\) updates: \[\langle w^*, w^{(T+1)} \rangle \geq T\]

Part 2: Upper Bound on Denominator. We bound \(\|w^{(T+1)}\|^2\) by decomposing: \[\|w^{(T+1)}\|^2 = \sum_{t=1}^T \left(\|w^{(t+1)}\|^2 - \|w^{(t)}\|^2\right)\]

At each update, \(w^{(t+1)} = w^{(t)} + y_i x_i\): \[\|w^{(t+1)}\|^2 = \|w^{(t)}\|^2 + 2y_i\langle w^{(t)}, x_i \rangle + \|x_i\|^2\]

Since the example is misclassified, \(y_i \langle w^{(t)}, x_i \rangle < 0\), so: \[\|w^{(t+1)}\|^2 - \|w^{(t)}\|^2 = 2y_i\langle w^{(t)}, x_i \rangle + \|x_i\|^2 \leq \|x_i\|^2 \leq R^2\]

Summing over all \(T\) updates: \[\|w^{(T+1)}\|^2 \leq TR^2\]

Conclusion: By Cauchy-Schwarz: \(\langle w^*, w^{(T+1)} \rangle \leq \|w^*\| \|w^{(T+1)}\|\), so: \[T \leq \langle w^*, w^{(T+1)} \rangle \leq \|w^*\| \|w^{(T+1)}\| \leq B\sqrt{TR^2} = BR\sqrt{T}\]

Dividing by \(\sqrt{T}\): \(\sqrt{T} \leq BR\), hence \(T \leq (BR)^2\). \(\square\)

17.9 9. Optimality of Least Squares (and Ridge Regression)

For a linear regression problem with design matrix \(X \in \mathbb{R}^{m \times d}\) and labels \(y \in \mathbb{R}^m\), the least-squares solution \(w_{LS} = (X^T X)^{-1}X^T y\) minimizes the sum of squared errors \(\|Xw - y\|^2\). Ridge regression with regularization parameter \(\lambda \geq 0\) gives \(w_{\lambda} = (X^T X + \lambda I)^{-1}X^T y\), which stabilises the solution when \(X^T X\) is singular or ill-conditioned.

Solution. Least Squares: The objective is: \[L(w) = \|Xw - y\|^2 = (Xw - y)^T(Xw - y)\]

Taking the gradient and setting to zero: \[\nabla L(w) = 2X^T(Xw - y) = 0 \quad \Rightarrow \quad X^T Xw = X^T y\]

If \(X^T X\) is invertible, the unique minimizer is: \[w_{LS} = (X^T X)^{-1}X^T y\]

The Hessian is \(2X^T X\), which is positive semidefinite, confirming this is a global minimum.

Ridge Regression: Adding an \(\ell_2\) regularizer: \[L_\lambda(w) = \|Xw - y\|^2 + \lambda \|w\|^2\]

The gradient condition becomes: \[X^T(Xw - y) + \lambda w = 0 \quad \Rightarrow \quad (X^T X + \lambda I)w = X^T y\]

The regularized solution is: \[w_\lambda = (X^T X + \lambda I)^{-1}X^T y\]

For \(\lambda > 0\), the matrix \(X^T X + \lambda I\) is always invertible (even if \(X\) is rank-deficient). Ridge regression reduces variance at the cost of introducing bias. \(\square\)

17.10 10. Distance from a Point to a Hyperplane

The Euclidean distance from a point \(x \in \mathbb{R}^d\) to the hyperplane defined by \(\langle w, \cdot \rangle = 0\) is: \[\text{dist}(x, H_w) = \frac{|\langle w, x \rangle|}{\|w\|}\]

Solution. The hyperplane \(H_w = \{x : \langle w, x \rangle = 0\}\) is the null space of the linear functional \(\langle w, \cdot \rangle\).

The distance from \(x\) to \(H_w\) is: \[\text{dist}(x, H_w) = \min_{\tilde{x} \in H_w} \|x - \tilde{x}\|\]

The closest point on \(H_w\) to \(x\) lies along the normal direction \(w\). Writing \(\tilde{x} = x - tw\) for some \(t \in \mathbb{R}\), we require \(\langle w, \tilde{x} \rangle = 0\): \[\langle w, x - tw \rangle = 0 \quad \Rightarrow \quad \langle w, x \rangle = t\|w\|^2 \quad \Rightarrow \quad t = \frac{\langle w, x \rangle}{\|w\|^2}\]

Thus: \[\tilde{x} = x - \frac{\langle w, x \rangle}{\|w\|^2} w\]

The distance is: \[\|x - \tilde{x}\| = \left\|\frac{\langle w, x \rangle}{\|w\|^2} w\right\| = \frac{|\langle w, x \rangle|}{\|w\|^2} \|w\| = \frac{|\langle w, x \rangle|}{\|w\|} \quad \square\]

17.11 11. SVM with Margin: Sample Complexity

For Support Vector Machines with margin \(\gamma > 0\), if the data is separable with margin \(\gamma\) and lies in a ball of radius \(R\), the VC dimension of the linear classifier is at most \(\min(d, \lceil R^2/\gamma^2 \rceil)\). The PAC sample complexity is: \[m_H(\epsilon, \delta) = O\left(\frac{R^2}{\gamma^2 \epsilon^2} \log\frac{1}{\delta}\right)\]

Solution. The key insight is that margin requirements constrain the hypothesis class. If all examples lie in a ball of radius \(R\) and must be separated with margin \(\gamma\), the separating hyperplanes form a restricted set.

By a geometric argument (Theorem 15.7 of the textbook), the VC dimension of linear classifiers with margin \(\gamma\) and radius \(R\) is bounded by: \[\text{VCdim}(H_{\gamma, R}) \leq \min\left(d, \left\lceil \frac{R^2}{\gamma^2} \right\rceil\right)\]

Intuitively, the margin requirement makes the function class smaller: the number of essentially different hyperplanes that separate points in a ball of radius \(R\) with margin \(\gamma\) is limited.

Using the fundamental theorem, since the VC dimension is finite, the sample complexity follows from the uniform convergence bound: \[m_H(\epsilon, \delta) = O\left(\frac{\text{VCdim}(H) + \log(1/\delta)}{\epsilon^2}\right) = O\left(\frac{R^2/\gamma^2}{\epsilon^2} \log\frac{1}{\delta}\right)\]

Thus, larger margins lead to faster learning. \(\square\)

17.12 12. Polynomial Kernel as a Mercer Kernel

The polynomial kernel \(K(x, y) = (1 + \langle x, y \rangle)^k\) is a valid Mercer kernel for any positive integer \(k\). Its feature map is explicitly given by the monomials of degree at most \(k\) in the input coordinates.

Solution. A kernel \(K\) is a valid Mercer kernel if and only if it can be expressed as \(K(x,y) = \langle \phi(x), \phi(y) \rangle\) for some feature map \(\phi\).

Explicit Feature Map Construction: For the polynomial kernel with \(k=2\) (and for simplicity, \(x, y \in \mathbb{R}^2\)), consider the feature map: \[\phi(x) = (1, \sqrt{2}x_1, \sqrt{2}x_2, x_1^2, \sqrt{2}x_1 x_2, x_2^2)^T\]

Then: \[\langle \phi(x), \phi(y) \rangle = 1 + 2x_1 y_1 + 2x_2 y_2 + x_1^2 y_1^2 + 2x_1 y_1 x_2 y_2 + x_2^2 y_2^2\] \[= (1 + x_1 y_1 + x_2 y_2)^2 = (1 + \langle x, y \rangle)^2\]

General Case: For degree-\(k\) polynomial kernel, the feature map consists of all monomials \(x^{\alpha}\) where \(\alpha\) is a multi-index with \(|\alpha| \leq k\). The dimension of this feature space is \(\binom{d+k}{k}\), where \(d\) is the input dimension. The kernel computes: \[K(x,y) = \langle \phi(x), \phi(y) \rangle = (1 + \langle x, y \rangle)^k\]

by the multinomial theorem. Since \(K\) is expressed as an inner product, it is a valid Mercer kernel. \(\square\)

17.13 13. Representer Theorem

For kernel-based regularized risk minimization: \[\min_{w} \left[\frac{1}{m}\sum_{i=1}^m \ell(y_i, \langle w, \phi(x_i) \rangle) + \lambda \|w\|^2\right]\] there exists an optimal solution of the form: \[w^* = \sum_{i=1}^m \alpha_i \phi(x_i)\] for some coefficients \(\alpha_1, \ldots, \alpha_m \in \mathbb{R}\).

Solution. Proof Idea: Consider any optimal solution \(w^*\) in the feature space. Decompose it as: \[w^* = w_\parallel + w_\perp\] where \(w_\parallel\) lies in the span of \(\{\phi(x_1), \ldots, \phi(x_m)\}\) and \(w_\perp\) is orthogonal to this span.

The empirical loss term \(\frac{1}{m}\sum_{i=1}^m \ell(y_i, \langle w, \phi(x_i) \rangle)\) depends only on \(\langle w, \phi(x_i) \rangle\). Since \(\langle w_\perp, \phi(x_i) \rangle = 0\) for all \(i\), replacing \(w\) with \(w_\parallel\) does not change the loss.

However, the regularization term \(\lambda\|w\|^2 = \lambda(\|w_\parallel\|^2 + \|w_\perp\|^2)\) increases if \(w_\perp \neq 0\). Thus, the optimal solution must have \(w_\perp = 0\), meaning: \[w^* = \sum_{i=1}^m \alpha_i \phi(x_i)\]

for some \(\alpha_1, \ldots, \alpha_m\). In SVM, these coefficients are the Lagrange multipliers of the dual problem. \(\square\)

17.14 14. AdaBoost Training-Error Bound

AdaBoost combines weak learners to achieve low training error. If \(\gamma_t\) is the edge of the \(t\)-th weak classifier (i.e., error \(= \frac{1}{2} - \gamma_t\)), then the training error of the final ensemble satisfies: \[L_S(H_T) \leq \exp\left(-2\sum_{t=1}^T \gamma_t^2\right)\]

Solution. AdaBoost maintains a distribution \(D_t\) over the training examples, initially uniform. At round \(t\): 1. Train a weak learner on \(S\) weighted by \(D_t\), obtaining classifier \(h_t\). 2. Compute the weighted error: \(\epsilon_t = \sum_{i : h_t(x_i) \neq y_i} D_t(i)\). 3. Set the weight: \(\alpha_t = \frac{1}{2}\log\frac{1-\epsilon_t}{\epsilon_t}\). 4. Update: \(D_{t+1}(i) = \frac{D_t(i) \exp(-\alpha_t y_i h_t(x_i))}{Z_t}\), where \(Z_t\) normalizes.

Proof: The final classifier is \(H_T(x) = \text{sign}\left(\sum_{t=1}^T \alpha_t h_t(x)\right)\).

An example \((x_i, y_i)\) is misclassified if \(y_i \sum_{t=1}^T \alpha_t h_t(x_i) < 0\).

Tracking the distribution evolution: \[D_{T+1}(i) = \frac{\exp\left(-y_i\sum_{t=1}^T \alpha_t h_t(x_i)\right)}{\prod_t Z_t}\]

Summing over all examples: \[L_S(H_T) = \frac{1}{m}|\{i : y_i H_T(x_i) < 0\}| \leq \sum_{i=1}^m D_{T+1}(i) = \frac{1}{\prod_t Z_t}\]

At each round, with \(\epsilon_t = \frac{1}{2} - \gamma_t\): \[Z_t = 2\sqrt{\epsilon_t(1-\epsilon_t)} = 2\sqrt{(\tfrac{1}{2}-\gamma_t)(\tfrac{1}{2}+\gamma_t)} = \sqrt{1-4\gamma_t^2} \leq e^{-2\gamma_t^2}\]

Therefore: \[L_S(H_T) \leq \prod_{t=1}^T e^{2\gamma_t^2} = \exp\left(-2\sum_{t=1}^T \gamma_t^2\right) \quad \square\]

17.15 15. SGD Convergence for Convex-Lipschitz Functions

For stochastic gradient descent on a convex-Lipschitz loss function \(L\) with Lipschitz constant \(\rho\), using step size \(\eta = \frac{B}{\rho\sqrt{T}}\) where \(\|w\|_2 \leq B\), the average iterate satisfies: \[\mathbb{E}[L(\bar{w}) - L(w^*)] \leq \frac{B\rho}{\sqrt{T}}\] where \(\bar{w} = \frac{1}{T}\sum_{t=1}^T w^{(t)}\) and the expectation is over the stochasticity in the algorithm.

Solution. Setup: Let \(g_t = \nabla \ell(w^{(t)}, z_t)\) be a stochastic gradient estimate. The update is \(w^{(t+1)} = w^{(t)} - \eta g_t\).

Key Inequality (Convexity): For a convex function \(L\): \[L(w) - L(w^*) \leq \langle \nabla L(w^*), w - w^* \rangle\]

At the optimum, \(\nabla L(w^*) = 0\), so: \[L(w^{(t)}) - L(w^*) \leq \langle g_t, w^{(t)} - w^* \rangle\]

where we use an unbiased stochastic gradient.

Regret Bound: Taking expectations and summing over \(t=1, \ldots, T\): \[\sum_{t=1}^T \mathbb{E}[L(w^{(t)}) - L(w^*)] \leq \sum_{t=1}^T \mathbb{E}[\langle g_t, w^{(t)} - w^* \rangle]\]

By Lipschitz continuity, \(\|g_t\| \leq \rho\). Standard regret analysis (via the projection inequality) gives: \[\sum_{t=1}^T \mathbb{E}[\langle g_t, w^{(t)} - w^* \rangle] \leq T\eta \rho^2 + \frac{B^2}{2\eta}\]

Setting \(\eta = \frac{B}{\rho\sqrt{T}}\) balances these terms: \[\text{Regret} \leq 2B\rho\sqrt{T}\]

Dividing by \(T\) and using convexity of averaging: \[\mathbb{E}[L(\bar{w}) - L(w^*)] \leq \frac{B\rho}{\sqrt{T}} \quad \square\]

17.16 16. SGD Bound for Strongly-Convex Regularised Loss

For strongly-convex loss functions (e.g., regularised empirical risk with \(\lambda\)-strong convexity), SGD with appropriately chosen step size achieves: \[\mathbb{E}[L(w^{(T)}) - L(w^*)] = O\left(\frac{\log T}{\lambda T}\right)\]

Solution. Strong Convexity Assumption: A function \(L\) is \(\lambda\)-strongly convex if: \[L(w) \geq L(w^*) + \langle \nabla L(w^*), w - w^* \rangle + \frac{\lambda}{2}\|w - w^*\|^2\]

At the optimum, \(\nabla L(w^*) = 0\): \[L(w) - L(w^*) \geq \frac{\lambda}{2}\|w - w^*\|^2\]

SGD with Decreasing Step Size: Using step size \(\eta_t = \frac{1}{\lambda(t+1)}\), the convergence analysis for strongly-convex functions yields: \[\mathbb{E}[\|w^{(t)} - w^*\|^2] = O\left(\frac{\log t}{t}\right)\]

Converting to Loss Difference: By strong convexity: \[\mathbb{E}[L(w^{(t)}) - L(w^*)] \geq \frac{\lambda}{2}\mathbb{E}[\|w^{(t)} - w^*\|^2] = O\left(\frac{\log t}{\lambda t}\right)\]

Averaging the final iterate (or using iterate averaging) with the appropriate analysis yields: \[\mathbb{E}[L(w^{(T)}) - L(w^*)] = O\left(\frac{\log T}{\lambda T}\right)\]

This shows exponential improvement over the convex (non-strongly-convex) case. \(\square\)

17.17 17. k-means: Lloyd’s Algorithm Monotonicity

Lloyd’s algorithm (the standard k-means algorithm) alternates between assigning points to the nearest centroid and recomputing centroids. At each iteration, the overall cost (sum of squared distances to cluster centroids) is non-increasing.

Solution. Notation: Let \(c_1, \ldots, c_k\) be the centroids, and let \(\Phi = \sum_{i=1}^m \|x_i - c_{j(i)}\|^2\) be the clustering cost, where \(j(i)\) is the cluster assignment for example \(x_i\).

Lloyd’s Algorithm: 1. Assignment Step: Assign each point to the nearest centroid: \(j(i) = \arg\min_j \|x_i - c_j\|^2\). 2. Update Step: Recompute each centroid as the mean of its assigned points: \(c_j = \frac{1}{|C_j|}\sum_{i \in C_j} x_i\).

Monotonicity of Assignment Step: When we reassign points to nearest centroids (keeping centroids fixed), the cost \(\Phi\) decreases or stays the same. Each reassigned point contributes less distance, and points remaining in the same cluster incur the same distance.

Monotonicity of Update Step: For a fixed cluster \(C_j\) with assigned points \(\{x_i : i \in C_j\}\), the centroid minimizing the sum of squared distances is the mean: \[c_j^* = \arg\min_c \sum_{i \in C_j} \|x_i - c\|^2 = \frac{1}{|C_j|}\sum_{i \in C_j} x_i\]

Thus, updating to the mean reduces the cluster cost, and total cost \(\Phi\) does not increase.

Convergence: Since the cost is bounded below by 0 and monotonically non-increasing, the sequence converges. The algorithm terminates when assignments no longer change, which occurs in finite time. \(\square\)

17.18 18. Kleinberg’s Impossibility Theorem for Clustering

No clustering algorithm can simultaneously satisfy the following three axioms:

  1. Scale-invariance: If \(D\) is a distance metric and the output clustering is \(\mathcal{C}\), then applying the algorithm to \(\alpha D\) (for \(\alpha > 0\)) yields \(\mathcal{C}\).
  2. Richness: For any partition \(\mathcal{C}\) of the data, there exists a distance metric \(D\) such that the algorithm outputs \(\mathcal{C}\).
  3. Consistency: If \(D\) is transformed to \(D'\) such that intra-cluster distances decrease and inter-cluster distances increase, the algorithm still outputs \(\mathcal{C}\).

Solution. Proof (by contradiction):

Suppose an algorithm \(A\) satisfies all three axioms. We construct two distance metrics on a small set and derive a contradiction.

Construction: Consider a set \(X = \{1, 2, 3, 4\}\) with: - Metric \(D_1\): Two clusters \(\{1,2\}\) and \(\{3,4\}\) (intra-cluster close, inter-cluster far). - Metric \(D_2\): Two different clusters \(\{1,3\}\) and \(\{2,4\}\) (different grouping, similar structure).

Applying Richness: By richness, there exist metrics \(D\) and \(D'\) such that: - Algorithm \(A\) on \(D\) outputs \(\mathcal{C}_1 = \{\{1,2\}, \{3,4\}\}\). - Algorithm \(A\) on \(D'\) outputs \(\mathcal{C}_2 = \{\{1,3\}, \{2,4\}\}\).

Applying Scale-Invariance: For any \(\alpha > 0\), \(A\) on \(\alpha D\) outputs \(\mathcal{C}_1\), and \(A\) on \(\alpha D'\) outputs \(\mathcal{C}_2\). The algorithm cannot distinguish between scaled versions.

Deriving the Contradiction: Via careful construction using consistency: we can modify \(D\) by shrinking intra-cluster distances and expanding inter-cluster distances such that the resulting metric approaches (in the limit or after finitely many steps) a metric structurally identical to \(D'\) or vice versa. By consistency, the output must remain \(\mathcal{C}_1\). But by scale-invariance applied to sufficiently scaled or modified versions, the algorithm must eventually output \(\mathcal{C}_2\). This contradiction shows no algorithm can satisfy all three axioms.

Practical Implication: In practice, one must relax at least one axiom: skip scale-invariance (use absolute thresholds), accept that richness fails (not all partitions are achievable), or allow divergence under metric transformation. \(\square\)

## 1. PAC Learning Theorem for Finite Hypothesis Classes

:::{.exercise name="Statement"}
*Let $H$ be a finite hypothesis class, and let $\delta, \epsilon \in (0,1)$ and $m \in \mathbb{N}$ such that $m \geq \frac{1}{\epsilon^2} \log\left(\frac{|H|}{\delta}\right)$. Then, for any learning rule and for any distribution $D$ on $X \times Y$ for which the realizability assumption is valid, if $h_S = \arg\min_{h \in H} L_S(h)$ is returned by ERM on a training set $S$ of $m$ i.i.d. samples from $D$, with probability at least $1-\delta$ it holds that $L_D(h_S) \leq \epsilon$.*
:::

:::{.solution}
Let us define the set of bad hypotheses as $H_B = \{h \in H : L_D(h) > \epsilon\}$. Under the realizability assumption, every bad hypothesis must have $L_S(h) > 0$ (i.e., at least one training error). We bound the probability that ERM returns a bad hypothesis:

$$\mathbb{P}^m(M_B) = \mathbb{P}^m\left(\exists h \in H_B : L_S(h) = 0\right) \leq \sum_{h \in H_B} \mathbb{P}^m(L_S(h) = 0)$$

Each misclassified point on $S$ occurs with probability at most $1-\epsilon$ (since $L_D(h) > \epsilon$). Therefore, the probability that a bad hypothesis is correct on all $m$ samples is at most $(1-\epsilon)^m \leq e^{-\epsilon m}$.

Summing over all bad hypotheses:
$$\mathbb{P}^m(M_B) \leq |H_B| e^{-\epsilon m} \leq |H| e^{-\epsilon m}$$

Setting this probability equal to $\delta$ and solving for $m$:
$$|H| e^{-\epsilon m} \leq \delta \quad \Rightarrow \quad -\epsilon m \leq \log(\delta/|H|) \quad \Rightarrow \quad m \geq \frac{1}{\epsilon}\log(|H|/\delta)$$

Thus, with $m \geq \frac{1}{\epsilon^2}\log(|H|/\delta)$ samples, with probability at least $1-\delta$, the ERM hypothesis $h_S$ satisfies $L_D(h_S) \leq \epsilon$. $\square$
:::

## 2. Lemma: Representativeness of the Sample

:::{.exercise name="Statement"}
*Let $S$ be an $\frac{\epsilon}{2}$-representative sample with respect to domain $Z$, hypothesis class $H$, loss function $\ell$, and distribution $D$. Then any output of $\text{ERM}_H(S)$, i.e., $h_S \in \arg\min_{h \in H} L_S(h)$, satisfies:*
$$L_D(h_S) \leq \min_{h \in H} L_D(h) + \epsilon$$
:::

:::{.solution}
The proof proceeds in three steps.

**Step 1.** Using the $\frac{\epsilon}{2}$-representativeness of $S$, for $h_S$:
$$|L_S(h_S) - L_D(h_S)| \leq \frac{\epsilon}{2} \quad \Rightarrow \quad L_D(h_S) \leq L_S(h_S) + \frac{\epsilon}{2}$$

**Step 2.** Since $h_S$ is obtained from ERM, it minimizes the empirical risk:
$$L_S(h_S) \leq L_S(h) \quad \text{for all } h \in H$$

Therefore: $L_D(h_S) \leq L_S(h) + \frac{\epsilon}{2}$.

**Step 3.** Again using $\frac{\epsilon}{2}$-representativeness, for any $h \in H$:
$$|L_S(h) - L_D(h)| \leq \frac{\epsilon}{2} \quad \Rightarrow \quad L_S(h) \leq L_D(h) + \frac{\epsilon}{2}$$

Substituting into the bound from Step 2:
$$L_D(h_S) \leq L_D(h) + \frac{\epsilon}{2} + \frac{\epsilon}{2} = L_D(h) + \epsilon$$

Since this holds for all $h \in H$, we can choose the one minimizing $L_D(h)$:
$$L_D(h_S) \leq \min_{h \in H} L_D(h) + \epsilon \quad \square$$
:::

## 3. Agnostic PAC Learning Theorem for Finite $H$

:::{.exercise name="Statement"}
*A hypothesis class $H$ is agnostic PAC learnable if there exists a function $m_H : (0,1)^2 \to \mathbb{N}$ and a learning algorithm such that for every $\epsilon, \delta \in (0,1)$, for every distribution $D$ on $X \times Y$, when the algorithm is run on $m \geq m_H(\epsilon, \delta)$ i.i.d. samples from $D$, it returns a hypothesis $h$ such that with probability at least $1-\delta$:*
$$L_D(h) \leq \min_{h' \in H} L_D(h') + \epsilon$$

*For finite $H$, this requires $m_H(\epsilon, \delta) = O\left(\frac{\log(|H|/\delta)}{\epsilon^2}\right)$.*
:::

:::{.solution}
We prove this using uniform convergence and Hoeffding's inequality.

**Uniform Convergence Suffices:** By the Representativeness Lemma, if $S$ is $\frac{\epsilon}{2}$-representative (i.e., $|L_S(h) - L_D(h)| \leq \frac{\epsilon}{2}$ for all $h \in H$), then any ERM output $h_S$ satisfies $L_D(h_S) \leq \min_{h \in H} L_D(h) + \epsilon$.

**Apply Union Bound:** The failure probability is:
$$\mathbb{P}\left(\exists h \in H : |L_S(h) - L_D(h)| > \frac{\epsilon}{2}\right) \leq \sum_{h \in H} \mathbb{P}\left(|L_S(h) - L_D(h)| > \frac{\epsilon}{2}\right)$$

**Hoeffding's Inequality:** For a fixed $h$, the loss values $\ell(h, z_i)$ are i.i.d. and bounded in $[0,1]$. By Hoeffding's inequality:
$$\mathbb{P}\left(|L_S(h) - L_D(h)| > \frac{\epsilon}{2}\right) \leq 2\exp\left(-2m\left(\frac{\epsilon}{2}\right)^2\right) = 2\exp\left(-\frac{m\epsilon^2}{2}\right)$$

**Combine the Bounds:**
$$\mathbb{P}\left(\exists h \in H : |L_S(h) - L_D(h)| > \frac{\epsilon}{2}\right) \leq 2|H|\exp\left(-\frac{m\epsilon^2}{2}\right)$$

**Derive Sample Complexity:** Requiring this to be at most $\delta$:
$$2|H|\exp\left(-\frac{m\epsilon^2}{2}\right) \leq \delta \quad \Rightarrow \quad m \geq \frac{2\log(2|H|/\delta)}{\epsilon^2}$$

With this many samples, with probability $\geq 1-\delta$, every hypothesis is $\frac{\epsilon}{2}$-representative, so ERM returns a hypothesis satisfying the agnostic PAC guarantee. $\square$
:::

## 4. Hoeffding's Inequality

:::{.exercise name="Statement"}
*Let $Z_1, \ldots, Z_m$ be i.i.d. random variables taking values in $[a, b]$, and let $\mu = \mathbb{E}[Z_i]$. Then for any $\delta > 0$:*
$$\mathbb{P}\left(|\frac{1}{m}\sum_{i=1}^m Z_i - \mu| > \delta\right) \leq 2\exp\left(-\frac{2m\delta^2}{(b-a)^2}\right)$$
:::

:::{.solution}
This is a fundamental concentration inequality. The proof uses Markov's inequality applied to the exponential of the empirical mean. For the 0-1 loss (where $a=0, b=1$), this yields:
$$\mathbb{P}(|L_S(h) - L_D(h)| > \epsilon) \leq 2\exp(-2m\epsilon^2)$$

We cite this result as a building block (see *Understanding Machine Learning*, Lemma B.6). $\square$
:::

## 5. Sauer-Shelah-Perles Lemma

:::{.exercise name="Statement"}
*Let $H$ be a hypothesis class with VC dimension $d = \text{VCdim}(H)$. Then the growth function satisfies:*
$$\tau_H(m) \leq \sum_{i=0}^{d} \binom{m}{i}$$

*For $m \geq d$, this implies $\tau_H(m) \leq (em/d)^d$.*
:::

:::{.solution}
**Proof Sketch (Shifting Argument):** We prove by induction on $m$ that if $\text{VCdim}(H) = d < \infty$, then $\tau_H(m) \leq \sum_{i=0}^{d} \binom{m}{i}$.

**Base Case:** For $m \leq d$, all sets of size $m$ can be shattered, so $\tau_H(m) = 2^m = \sum_{i=0}^{m} \binom{m}{i} \leq \sum_{i=0}^{d} \binom{m}{i}$ (since the RHS is larger when $m \leq d$).

**Inductive Step:** Fix a set $C$ of size $m > d$. Partition the dichotomies (binary labelings) consistent with $H$:
- Let $D_0$ be dichotomies where all points in $C$ get one assignment.
- Consider a point $x \in C$ and partition based on behavior on $x$.

By the definition of VC dimension, the set $C \setminus \{x\}$ has size $m-1$, so by induction, there are at most $\sum_{i=0}^{d} \binom{m-1}{i}$ dichotomies on $C \setminus \{x\}$.

The total number of dichotomies on $C$ is bounded by dichotomies agreeing on $x$ plus those disagreeing, yielding:
$$\tau_H(m) \leq \sum_{i=0}^{d} \binom{m-1}{i} + \sum_{i=0}^{d} \binom{m-1}{i} = \sum_{i=0}^{d} \binom{m}{i}$$

**Corollary:** For $m \geq d$, by Stirling's approximation:
$$\sum_{i=0}^{d} \binom{m}{i} \leq (em/d)^d$$

$\square$
:::

## 6. Fundamental Theorem of Statistical Learning (Theorem 6.8)

:::{.exercise name="Statement"}
*Let $H$ be a hypothesis class of functions from $X$ to $\{0, 1\}$ with 0-1 loss. The following six conditions are equivalent:*

1. *$H$ has the uniform convergence property.*
2. *Every ERM algorithm is an agnostic PAC learner for $H$.*
3. *$H$ is agnostic PAC learnable.*
4. *$H$ is PAC learnable.*
5. *Every ERM algorithm is a PAC learner for $H$.*
6. *$H$ has finite VC dimension.*
:::

:::{.solution}
**Direction 1 $\Rightarrow$ 2 (Uniform Convergence to Agnostic PAC):**
Uniform convergence means that with probability $\geq 1-\delta$, all hypotheses satisfy $|L_S(h) - L_D(h)| \leq \epsilon$ simultaneously. By the Representativeness Lemma, this implies any ERM hypothesis satisfies $L_D(h_S) \leq \min_h L_D(h) + \epsilon$, so ERM is an agnostic PAC learner.

**Direction 2 $\Rightarrow$ 3 (ERM Agnostic PAC $\Rightarrow$ H Agnostic PAC):**
Trivial: if the ERM algorithm is agnostic PAC, then $H$ is agnostic PAC learnable (by definition).

**Direction 3 $\Rightarrow$ 4 (Agnostic PAC $\Rightarrow$ PAC):**
PAC learning is the special case of agnostic PAC learning when the realizability assumption holds. If $H$ is agnostic PAC learnable with sample complexity $m_H(\epsilon, \delta)$, then under realizability, running the same algorithm on $m_H(\epsilon, \delta)$ samples produces a hypothesis with $L_D(h) \leq 0 + \epsilon = \epsilon$, so $H$ is PAC learnable.

**Direction 1 $\Rightarrow$ 5 (Uniform Convergence to PAC via ERM):**
Under the realizability assumption, if uniform convergence holds, then with probability $\geq 1-\delta$, every bad hypothesis (with $L_D(h) > \epsilon$) must have $L_S(h) > 0$. ERM will never choose such a hypothesis, so ERM returns a PAC hypothesis.

**Direction 4 $\Rightarrow$ 6 (PAC Learnability $\Rightarrow$ Finite VC Dimension):**
Suppose $H$ has infinite VC dimension. Then for arbitrarily large $m$, there exists a set $C$ of size $2m$ that is shattered by $H$. By the No-Free-Lunch theorem, there exists a distribution $D$ on $C$ such that any learning algorithm fails to PAC-learn $H$ with sample size $m$. This contradicts the assumption that $H$ is PAC learnable. Therefore, $H$ must have finite VC dimension.

**Direction 6 $\Rightarrow$ 1 (Finite VC Dimension $\Rightarrow$ Uniform Convergence):**
Let $d = \text{VCdim}(H)$. By the Sauer-Shelah-Perles lemma and concentration inequalities (see Theorem 6.11), uniform convergence holds with sample complexity:
$$m_{UC}(\epsilon, \delta) = O\left(\frac{d + \log(1/\delta)}{\epsilon^2}\right)$$

Therefore, finite VC dimension implies uniform convergence, completing the cycle. $\square$
:::

## 7. Structural Risk Minimisation Bound

:::{.exercise name="Statement"}
*Let $w : \mathbb{N} \to [0,1]$ be a function such that $\sum_{i=1}^{\infty} w(i) \leq 1$, and let $H_1, H_2, \ldots$ be a sequence of hypothesis classes such that $H = \bigcup_{n=1}^{\infty} H_n$. Then for every $\delta \in (0,1)$, $\epsilon \in (0,1)$, and for every distribution $D$ on $X \times Y$, with probability at least $1-\delta$, for every training set $S$ of size $m$ i.i.d. from $D$, and for every hypothesis $h \in H_n$ for some $n$:*
$$L_D(h) \leq L_S(h) + 2\min_n \epsilon(m, w(n)\delta)$$
:::

:::{.solution}
**Setup:** Write $H = \bigcup_{n \in \mathbb{N}} H_n$. For each $n$, set $\delta_n := w(n)\delta$, so $\sum_{n=1}^{\infty} \delta_n = \delta \sum_{n=1}^{\infty} w(n) \leq \delta$.

**Uniform Convergence on Each Subclass:** By uniform convergence (Corollary 4.6 applied to $H_n$), for each fixed $n$:
$$\mathbb{P}\left(\exists h \in H_n : |L_S(h) - L_D(h)| > \epsilon(m, \delta_n)\right) \leq \delta_n$$

**Union Bound Over Subclasses:** Applying the union bound over all $n \in \mathbb{N}$:
$$\mathbb{P}\left(\exists n, \exists h \in H_n : |L_S(h) - L_D(h)| > \epsilon(m, \delta_n)\right) \leq \sum_{n=1}^{\infty} \delta_n \leq \delta$$

**Conclusion:** With probability $\geq 1-\delta$, for every $h \in H_n$:
$$|L_D(h) - L_S(h)| \leq \epsilon(m, \delta_n) = \epsilon(m, w(n)\delta)$$

Rearranging yields the claimed bound. $\square$
:::

## 8. Novikoff Perceptron Convergence Bound

:::{.exercise name="Statement"}
*Let $(x_1, y_1), \ldots, (x_m, y_m)$ be a linearly separable dataset, and let $w^* \in \mathbb{R}^d$ be a separating hyperplane (i.e., $y_i \langle w^*, x_i \rangle \geq 1$ for all $i$). Define $B = \|w^*\|$ and $R = \max_i \|x_i\|$. Then the perceptron algorithm converges in at most $(RB)^2$ iterations.*
:::

:::{.solution}
Let $w^{(t)}$ be the weight vector after $t$ iterations, and suppose the algorithm stops after $T$ iterations. We prove two key bounds.

**Part 1: Lower Bound on Numerator.**
Consider the telescoping sum:
$$\langle w^*, w^{(T+1)} \rangle = \sum_{t=1}^T \left(\langle w^*, w^{(t+1)} \rangle - \langle w^*, w^{(t)} \rangle\right)$$

At each update step, we update $w^{(t+1)} = w^{(t)} + y_i x_i$ for some misclassified example $(x_i, y_i)$. Thus:
$$\langle w^*, w^{(t+1)} \rangle - \langle w^*, w^{(t)} \rangle = y_i \langle w^*, x_i \rangle \geq 1$$

Summing over all $T$ updates:
$$\langle w^*, w^{(T+1)} \rangle \geq T$$

**Part 2: Upper Bound on Denominator.**
We bound $\|w^{(T+1)}\|^2$ by decomposing:
$$\|w^{(T+1)}\|^2 = \sum_{t=1}^T \left(\|w^{(t+1)}\|^2 - \|w^{(t)}\|^2\right)$$

At each update, $w^{(t+1)} = w^{(t)} + y_i x_i$:
$$\|w^{(t+1)}\|^2 = \|w^{(t)}\|^2 + 2y_i\langle w^{(t)}, x_i \rangle + \|x_i\|^2$$

Since the example is misclassified, $y_i \langle w^{(t)}, x_i \rangle < 0$, so:
$$\|w^{(t+1)}\|^2 - \|w^{(t)}\|^2 = 2y_i\langle w^{(t)}, x_i \rangle + \|x_i\|^2 \leq \|x_i\|^2 \leq R^2$$

Summing over all $T$ updates:
$$\|w^{(T+1)}\|^2 \leq TR^2$$

**Conclusion:**
By Cauchy-Schwarz: $\langle w^*, w^{(T+1)} \rangle \leq \|w^*\| \|w^{(T+1)}\|$, so:
$$T \leq \langle w^*, w^{(T+1)} \rangle \leq \|w^*\| \|w^{(T+1)}\| \leq B\sqrt{TR^2} = BR\sqrt{T}$$

Dividing by $\sqrt{T}$: $\sqrt{T} \leq BR$, hence $T \leq (BR)^2$. $\square$
:::

## 9. Optimality of Least Squares (and Ridge Regression)

:::{.exercise name="Statement"}
*For a linear regression problem with design matrix $X \in \mathbb{R}^{m \times d}$ and labels $y \in \mathbb{R}^m$, the least-squares solution $w_{LS} = (X^T X)^{-1}X^T y$ minimizes the sum of squared errors $\|Xw - y\|^2$. Ridge regression with regularization parameter $\lambda \geq 0$ gives $w_{\lambda} = (X^T X + \lambda I)^{-1}X^T y$, which stabilises the solution when $X^T X$ is singular or ill-conditioned.*
:::

:::{.solution}
**Least Squares:** The objective is:
$$L(w) = \|Xw - y\|^2 = (Xw - y)^T(Xw - y)$$

Taking the gradient and setting to zero:
$$\nabla L(w) = 2X^T(Xw - y) = 0 \quad \Rightarrow \quad X^T Xw = X^T y$$

If $X^T X$ is invertible, the unique minimizer is:
$$w_{LS} = (X^T X)^{-1}X^T y$$

The Hessian is $2X^T X$, which is positive semidefinite, confirming this is a global minimum.

**Ridge Regression:** Adding an $\ell_2$ regularizer:
$$L_\lambda(w) = \|Xw - y\|^2 + \lambda \|w\|^2$$

The gradient condition becomes:
$$X^T(Xw - y) + \lambda w = 0 \quad \Rightarrow \quad (X^T X + \lambda I)w = X^T y$$

The regularized solution is:
$$w_\lambda = (X^T X + \lambda I)^{-1}X^T y$$

For $\lambda > 0$, the matrix $X^T X + \lambda I$ is always invertible (even if $X$ is rank-deficient). Ridge regression reduces variance at the cost of introducing bias. $\square$
:::

## 10. Distance from a Point to a Hyperplane

:::{.exercise name="Statement"}
*The Euclidean distance from a point $x \in \mathbb{R}^d$ to the hyperplane defined by $\langle w, \cdot \rangle = 0$ is:*
$$\text{dist}(x, H_w) = \frac{|\langle w, x \rangle|}{\|w\|}$$
:::

:::{.solution}
The hyperplane $H_w = \{x : \langle w, x \rangle = 0\}$ is the null space of the linear functional $\langle w, \cdot \rangle$.

The distance from $x$ to $H_w$ is:
$$\text{dist}(x, H_w) = \min_{\tilde{x} \in H_w} \|x - \tilde{x}\|$$

The closest point on $H_w$ to $x$ lies along the normal direction $w$. Writing $\tilde{x} = x - tw$ for some $t \in \mathbb{R}$, we require $\langle w, \tilde{x} \rangle = 0$:
$$\langle w, x - tw \rangle = 0 \quad \Rightarrow \quad \langle w, x \rangle = t\|w\|^2 \quad \Rightarrow \quad t = \frac{\langle w, x \rangle}{\|w\|^2}$$

Thus:
$$\tilde{x} = x - \frac{\langle w, x \rangle}{\|w\|^2} w$$

The distance is:
$$\|x - \tilde{x}\| = \left\|\frac{\langle w, x \rangle}{\|w\|^2} w\right\| = \frac{|\langle w, x \rangle|}{\|w\|^2} \|w\| = \frac{|\langle w, x \rangle|}{\|w\|} \quad \square$$
:::

## 11. SVM with Margin: Sample Complexity

:::{.exercise name="Statement"}
*For Support Vector Machines with margin $\gamma > 0$, if the data is separable with margin $\gamma$ and lies in a ball of radius $R$, the VC dimension of the linear classifier is at most $\min(d, \lceil R^2/\gamma^2 \rceil)$. The PAC sample complexity is:*
$$m_H(\epsilon, \delta) = O\left(\frac{R^2}{\gamma^2 \epsilon^2} \log\frac{1}{\delta}\right)$$
:::

:::{.solution}
The key insight is that margin requirements constrain the hypothesis class. If all examples lie in a ball of radius $R$ and must be separated with margin $\gamma$, the separating hyperplanes form a restricted set.

By a geometric argument (Theorem 15.7 of the textbook), the VC dimension of linear classifiers with margin $\gamma$ and radius $R$ is bounded by:
$$\text{VCdim}(H_{\gamma, R}) \leq \min\left(d, \left\lceil \frac{R^2}{\gamma^2} \right\rceil\right)$$

Intuitively, the margin requirement makes the function class smaller: the number of essentially different hyperplanes that separate points in a ball of radius $R$ with margin $\gamma$ is limited.

Using the fundamental theorem, since the VC dimension is finite, the sample complexity follows from the uniform convergence bound:
$$m_H(\epsilon, \delta) = O\left(\frac{\text{VCdim}(H) + \log(1/\delta)}{\epsilon^2}\right) = O\left(\frac{R^2/\gamma^2}{\epsilon^2} \log\frac{1}{\delta}\right)$$

Thus, larger margins lead to faster learning. $\square$
:::

## 12. Polynomial Kernel as a Mercer Kernel

:::{.exercise name="Statement"}
*The polynomial kernel $K(x, y) = (1 + \langle x, y \rangle)^k$ is a valid Mercer kernel for any positive integer $k$. Its feature map is explicitly given by the monomials of degree at most $k$ in the input coordinates.*
:::

:::{.solution}
A kernel $K$ is a valid Mercer kernel if and only if it can be expressed as $K(x,y) = \langle \phi(x), \phi(y) \rangle$ for some feature map $\phi$.

**Explicit Feature Map Construction:**
For the polynomial kernel with $k=2$ (and for simplicity, $x, y \in \mathbb{R}^2$), consider the feature map:
$$\phi(x) = (1, \sqrt{2}x_1, \sqrt{2}x_2, x_1^2, \sqrt{2}x_1 x_2, x_2^2)^T$$

Then:
$$\langle \phi(x), \phi(y) \rangle = 1 + 2x_1 y_1 + 2x_2 y_2 + x_1^2 y_1^2 + 2x_1 y_1 x_2 y_2 + x_2^2 y_2^2$$
$$= (1 + x_1 y_1 + x_2 y_2)^2 = (1 + \langle x, y \rangle)^2$$

**General Case:**
For degree-$k$ polynomial kernel, the feature map consists of all monomials $x^{\alpha}$ where $\alpha$ is a multi-index with $|\alpha| \leq k$. The dimension of this feature space is $\binom{d+k}{k}$, where $d$ is the input dimension. The kernel computes:
$$K(x,y) = \langle \phi(x), \phi(y) \rangle = (1 + \langle x, y \rangle)^k$$

by the multinomial theorem. Since $K$ is expressed as an inner product, it is a valid Mercer kernel. $\square$
:::

## 13. Representer Theorem

:::{.exercise name="Statement"}
*For kernel-based regularized risk minimization:*
$$\min_{w} \left[\frac{1}{m}\sum_{i=1}^m \ell(y_i, \langle w, \phi(x_i) \rangle) + \lambda \|w\|^2\right]$$
*there exists an optimal solution of the form:*
$$w^* = \sum_{i=1}^m \alpha_i \phi(x_i)$$
*for some coefficients $\alpha_1, \ldots, \alpha_m \in \mathbb{R}$.*
:::

:::{.solution}
**Proof Idea:**
Consider any optimal solution $w^*$ in the feature space. Decompose it as:
$$w^* = w_\parallel + w_\perp$$
where $w_\parallel$ lies in the span of $\{\phi(x_1), \ldots, \phi(x_m)\}$ and $w_\perp$ is orthogonal to this span.

The empirical loss term $\frac{1}{m}\sum_{i=1}^m \ell(y_i, \langle w, \phi(x_i) \rangle)$ depends only on $\langle w, \phi(x_i) \rangle$. Since $\langle w_\perp, \phi(x_i) \rangle = 0$ for all $i$, replacing $w$ with $w_\parallel$ does not change the loss.

However, the regularization term $\lambda\|w\|^2 = \lambda(\|w_\parallel\|^2 + \|w_\perp\|^2)$ increases if $w_\perp \neq 0$. Thus, the optimal solution must have $w_\perp = 0$, meaning:
$$w^* = \sum_{i=1}^m \alpha_i \phi(x_i)$$

for some $\alpha_1, \ldots, \alpha_m$. In SVM, these coefficients are the Lagrange multipliers of the dual problem. $\square$
:::

## 14. AdaBoost Training-Error Bound

:::{.exercise name="Statement"}
*AdaBoost combines weak learners to achieve low training error. If $\gamma_t$ is the edge of the $t$-th weak classifier (i.e., error $= \frac{1}{2} - \gamma_t$), then the training error of the final ensemble satisfies:*
$$L_S(H_T) \leq \exp\left(-2\sum_{t=1}^T \gamma_t^2\right)$$
:::

:::{.solution}
AdaBoost maintains a distribution $D_t$ over the training examples, initially uniform. At round $t$:
1. Train a weak learner on $S$ weighted by $D_t$, obtaining classifier $h_t$.
2. Compute the weighted error: $\epsilon_t = \sum_{i : h_t(x_i) \neq y_i} D_t(i)$.
3. Set the weight: $\alpha_t = \frac{1}{2}\log\frac{1-\epsilon_t}{\epsilon_t}$.
4. Update: $D_{t+1}(i) = \frac{D_t(i) \exp(-\alpha_t y_i h_t(x_i))}{Z_t}$, where $Z_t$ normalizes.

**Proof:**
The final classifier is $H_T(x) = \text{sign}\left(\sum_{t=1}^T \alpha_t h_t(x)\right)$.

An example $(x_i, y_i)$ is misclassified if $y_i \sum_{t=1}^T \alpha_t h_t(x_i) < 0$.

Tracking the distribution evolution:
$$D_{T+1}(i) = \frac{\exp\left(-y_i\sum_{t=1}^T \alpha_t h_t(x_i)\right)}{\prod_t Z_t}$$

Summing over all examples:
$$L_S(H_T) = \frac{1}{m}|\{i : y_i H_T(x_i) < 0\}| \leq \sum_{i=1}^m D_{T+1}(i) = \frac{1}{\prod_t Z_t}$$

At each round, with $\epsilon_t = \frac{1}{2} - \gamma_t$:
$$Z_t = 2\sqrt{\epsilon_t(1-\epsilon_t)} = 2\sqrt{(\tfrac{1}{2}-\gamma_t)(\tfrac{1}{2}+\gamma_t)} = \sqrt{1-4\gamma_t^2} \leq e^{-2\gamma_t^2}$$

Therefore:
$$L_S(H_T) \leq \prod_{t=1}^T e^{2\gamma_t^2} = \exp\left(-2\sum_{t=1}^T \gamma_t^2\right) \quad \square$$
:::

## 15. SGD Convergence for Convex-Lipschitz Functions

:::{.exercise name="Statement"}
*For stochastic gradient descent on a convex-Lipschitz loss function $L$ with Lipschitz constant $\rho$, using step size $\eta = \frac{B}{\rho\sqrt{T}}$ where $\|w\|_2 \leq B$, the average iterate satisfies:*
$$\mathbb{E}[L(\bar{w}) - L(w^*)] \leq \frac{B\rho}{\sqrt{T}}$$
*where $\bar{w} = \frac{1}{T}\sum_{t=1}^T w^{(t)}$ and the expectation is over the stochasticity in the algorithm.*
:::

:::{.solution}
**Setup:** Let $g_t = \nabla \ell(w^{(t)}, z_t)$ be a stochastic gradient estimate. The update is $w^{(t+1)} = w^{(t)} - \eta g_t$.

**Key Inequality (Convexity):**
For a convex function $L$:
$$L(w) - L(w^*) \leq \langle \nabla L(w^*), w - w^* \rangle$$

At the optimum, $\nabla L(w^*) = 0$, so:
$$L(w^{(t)}) - L(w^*) \leq \langle g_t, w^{(t)} - w^* \rangle$$

where we use an unbiased stochastic gradient.

**Regret Bound:**
Taking expectations and summing over $t=1, \ldots, T$:
$$\sum_{t=1}^T \mathbb{E}[L(w^{(t)}) - L(w^*)] \leq \sum_{t=1}^T \mathbb{E}[\langle g_t, w^{(t)} - w^* \rangle]$$

By Lipschitz continuity, $\|g_t\| \leq \rho$. Standard regret analysis (via the projection inequality) gives:
$$\sum_{t=1}^T \mathbb{E}[\langle g_t, w^{(t)} - w^* \rangle] \leq T\eta \rho^2 + \frac{B^2}{2\eta}$$

Setting $\eta = \frac{B}{\rho\sqrt{T}}$ balances these terms:
$$\text{Regret} \leq 2B\rho\sqrt{T}$$

Dividing by $T$ and using convexity of averaging:
$$\mathbb{E}[L(\bar{w}) - L(w^*)] \leq \frac{B\rho}{\sqrt{T}} \quad \square$$
:::

## 16. SGD Bound for Strongly-Convex Regularised Loss

:::{.exercise name="Statement"}
*For strongly-convex loss functions (e.g., regularised empirical risk with $\lambda$-strong convexity), SGD with appropriately chosen step size achieves:*
$$\mathbb{E}[L(w^{(T)}) - L(w^*)] = O\left(\frac{\log T}{\lambda T}\right)$$
:::

:::{.solution}
**Strong Convexity Assumption:**
A function $L$ is $\lambda$-strongly convex if:
$$L(w) \geq L(w^*) + \langle \nabla L(w^*), w - w^* \rangle + \frac{\lambda}{2}\|w - w^*\|^2$$

At the optimum, $\nabla L(w^*) = 0$:
$$L(w) - L(w^*) \geq \frac{\lambda}{2}\|w - w^*\|^2$$

**SGD with Decreasing Step Size:**
Using step size $\eta_t = \frac{1}{\lambda(t+1)}$, the convergence analysis for strongly-convex functions yields:
$$\mathbb{E}[\|w^{(t)} - w^*\|^2] = O\left(\frac{\log t}{t}\right)$$

**Converting to Loss Difference:**
By strong convexity:
$$\mathbb{E}[L(w^{(t)}) - L(w^*)] \geq \frac{\lambda}{2}\mathbb{E}[\|w^{(t)} - w^*\|^2] = O\left(\frac{\log t}{\lambda t}\right)$$

Averaging the final iterate (or using iterate averaging) with the appropriate analysis yields:
$$\mathbb{E}[L(w^{(T)}) - L(w^*)] = O\left(\frac{\log T}{\lambda T}\right)$$

This shows exponential improvement over the convex (non-strongly-convex) case. $\square$
:::

## 17. k-means: Lloyd's Algorithm Monotonicity

:::{.exercise name="Statement"}
*Lloyd's algorithm (the standard k-means algorithm) alternates between assigning points to the nearest centroid and recomputing centroids. At each iteration, the overall cost (sum of squared distances to cluster centroids) is non-increasing.*
:::

:::{.solution}
**Notation:**
Let $c_1, \ldots, c_k$ be the centroids, and let $\Phi = \sum_{i=1}^m \|x_i - c_{j(i)}\|^2$ be the clustering cost, where $j(i)$ is the cluster assignment for example $x_i$.

**Lloyd's Algorithm:**
1. **Assignment Step:** Assign each point to the nearest centroid: $j(i) = \arg\min_j \|x_i - c_j\|^2$.
2. **Update Step:** Recompute each centroid as the mean of its assigned points: $c_j = \frac{1}{|C_j|}\sum_{i \in C_j} x_i$.

**Monotonicity of Assignment Step:**
When we reassign points to nearest centroids (keeping centroids fixed), the cost $\Phi$ decreases or stays the same. Each reassigned point contributes less distance, and points remaining in the same cluster incur the same distance.

**Monotonicity of Update Step:**
For a fixed cluster $C_j$ with assigned points $\{x_i : i \in C_j\}$, the centroid minimizing the sum of squared distances is the mean:
$$c_j^* = \arg\min_c \sum_{i \in C_j} \|x_i - c\|^2 = \frac{1}{|C_j|}\sum_{i \in C_j} x_i$$

Thus, updating to the mean reduces the cluster cost, and total cost $\Phi$ does not increase.

**Convergence:**
Since the cost is bounded below by 0 and monotonically non-increasing, the sequence converges. The algorithm terminates when assignments no longer change, which occurs in finite time. $\square$
:::

## 18. Kleinberg's Impossibility Theorem for Clustering

:::{.exercise name="Statement"}
*No clustering algorithm can simultaneously satisfy the following three axioms:*

1. *Scale-invariance: If $D$ is a distance metric and the output clustering is $\mathcal{C}$, then applying the algorithm to $\alpha D$ (for $\alpha > 0$) yields $\mathcal{C}$.*
2. *Richness: For any partition $\mathcal{C}$ of the data, there exists a distance metric $D$ such that the algorithm outputs $\mathcal{C}$.*
3. *Consistency: If $D$ is transformed to $D'$ such that intra-cluster distances decrease and inter-cluster distances increase, the algorithm still outputs $\mathcal{C}$.*
:::

:::{.solution}
**Proof (by contradiction):**

Suppose an algorithm $A$ satisfies all three axioms. We construct two distance metrics on a small set and derive a contradiction.

**Construction:**
Consider a set $X = \{1, 2, 3, 4\}$ with:
- **Metric $D_1$:** Two clusters $\{1,2\}$ and $\{3,4\}$ (intra-cluster close, inter-cluster far).
- **Metric $D_2$:** Two different clusters $\{1,3\}$ and $\{2,4\}$ (different grouping, similar structure).

**Applying Richness:**
By richness, there exist metrics $D$ and $D'$ such that:
- Algorithm $A$ on $D$ outputs $\mathcal{C}_1 = \{\{1,2\}, \{3,4\}\}$.
- Algorithm $A$ on $D'$ outputs $\mathcal{C}_2 = \{\{1,3\}, \{2,4\}\}$.

**Applying Scale-Invariance:**
For any $\alpha > 0$, $A$ on $\alpha D$ outputs $\mathcal{C}_1$, and $A$ on $\alpha D'$ outputs $\mathcal{C}_2$. The algorithm cannot distinguish between scaled versions.

**Deriving the Contradiction:**
Via careful construction using consistency: we can modify $D$ by shrinking intra-cluster distances and expanding inter-cluster distances such that the resulting metric approaches (in the limit or after finitely many steps) a metric structurally identical to $D'$ or vice versa. By consistency, the output must remain $\mathcal{C}_1$. But by scale-invariance applied to sufficiently scaled or modified versions, the algorithm must eventually output $\mathcal{C}_2$. This contradiction shows no algorithm can satisfy all three axioms.

**Practical Implication:**
In practice, one must relax at least one axiom: skip scale-invariance (use absolute thresholds), accept that richness fails (not all partitions are achievable), or allow divergence under metric transformation. $\square$
:::
