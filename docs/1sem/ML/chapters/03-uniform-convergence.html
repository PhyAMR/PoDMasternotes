4  Uniform Convergence

Our goal is always to find a predictor that minimizes the true error as much as possible, which is impossible to determine a priori since the distribution \(\mathcal{D}\) is not known to the algorithm.

What we have done so far is to rely on those functions that minimize the empirical risk, with the hope that they are also the ones that minimize the true risk. We have actually managed to quantify the probability with which these errors were effectively “close” to each other, by linking the parameters \(\epsilon\) and \(\delta\) to the number of samples present in the learning set \(\mathcal{S}\): the larger the latter, the greater the representativeness of \(\mathcal{S}\) with respect to the distribution \(\mathcal{D}\).

We now add a further element useful for ensuring that ERM leads to a minimization of the true error: it is sufficient, in fact, that the empirical risk of all predictors \(h\) belonging to the considered class \(\mathcal{H}\) are a good approximation of their true error.

4.1 \(\varepsilon\)-representative set

If we want the empirical risk to be a good measure of the true risk regardless of the chosen predictor, it is necessary that their difference is always limited by an upper bound. In this regard, we define \(\boldsymbol{**\epsilon-**}\)representativeness as follows:

A learning set \(\mathcal{S}\) is said to be \(\boldsymbol{\epsilon-}\)representative with respect to a domain \(\mathcal{Z}\), a hypothesis class \(\mathcal{H}\), a loss function \(l\), and a distribution \(\mathcal{D}\) if

\[ | L_{\mathcal{S}}(h)-L_{\mathcal{D}}(h)|\leqslant\epsilon \ \ \ \forall h\in\mathcal{H} \]

4.2 When ERM is an agnostic PAC learner

The definition just given is useful for the following theorem:

Let \(\mathcal{S}\) be a learning set that is \(\frac{\epsilon}{2}-\)representative with respect to a domain \(\mathcal{Z}\), a hypothesis class \(\mathcal{H}\), a loss function \(l\), and a distribution \(\mathcal{D}\). Then any output of \(\text{ERM}_{\mathcal{H}}(\mathcal{S})\), i.e., \(h_\mathcal{S}\in\underset{h\in\mathcal{H}}{\argmin}L_{\mathcal{S}}(h)\), satisfies:

\[ \boxed{L_{\mathcal{D}}\left(h_\mathcal{S}\right)\leqslant\min_{h\in\mathcal{H}}L_{\mathcal{D}}(h)+\epsilon} \]

An important thing to emphasize is that this result is valid with probability \(1\), so we are sure that the error resulting from the ERM is greater than the smallest possible error by at most \(\epsilon\).

The proof of the theorem is divided into three simple steps:

- We start from the $\frac{\epsilon}{2}-$representativeness of $\mathcal{S}$:

$$
    | L_{\mathcal{S}}\left(h_\mathcal{S}\right)-L_{\mathcal{D}}\left(h_\mathcal{S}\right)}\leqslant\frac{\epsilon}{2} \ \iff \ -\frac{\epsilon}{2}\leqslant L_{\mathcal{S}}\left(h_\mathcal{S}\right)-L_{\mathcal{D}}\left(h_\mathcal{S}\right)\leqslant\frac{\epsilon}{2}
$$

Let's look only at the first inequality:

$$
    -L_{\mathcal{D}}\left(h_\mathcal{S}\right)\geqslant -L_{\mathcal{S}}\left(h_\mathcal{S}\right)-\frac{\epsilon}{2} \ \Longrightarrow \ L_{\mathcal{D}}\left(h_\mathcal{S}\right)\leqslant L_{\mathcal{S}}\left(h_\mathcal{S}\right)+\frac{\epsilon}{2} 
$$

- Since the predictor $h_\mathcal{S}$ is obtained from the ERM, then:

$$
    L_{\mathcal{S}}\left(h_\mathcal{S}\right)\leqslant L_{\mathcal{S}}(h)
$$

Returning to @eq:convergenza-uniforme-teorema-epsilon-dimostrazione-1:

$$
    L_{\mathcal{D}}\left(h_\mathcal{S}\right)\leqslant L_{\mathcal{S}}(h)+\frac{\epsilon}{2} 
$$

- We reuse the $\frac{\epsilon}{2}-$representativeness of $\mathcal{S}$:

$$
    | L_{\mathcal{S}}(h)-L_{\mathcal{D}}(h)|\leqslant\frac{\epsilon}{2} \ \iff \ -\frac{\epsilon}{2}\leqslant L_{\mathcal{S}}(h)-L_{\mathcal{D}}(h)\leqslant\frac{\epsilon}{2}
$$

This time we look at the second inequality and substitute it into @eq:convergenza-uniforme-teorema-epsilon-dimostrazione-2

$$
    L_{\mathcal{S}}(h)\leqslant L_{\mathcal{D}}(h)+\frac{\epsilon}{2} \ \Longrightarrow \ L_{\mathcal{D}}\left(h_\mathcal{S}\right)\leqslant L_{\mathcal{D}}(h)+\frac{\epsilon}{2}+\frac{\epsilon}{2}=L_{\mathcal{D}}(h)+\epsilon
$$

The predictor \(h\) is any predictor within the class \(\mathcal{H}\); in particular, we can choose the one that minimizes \(L_{\mathcal{D}}(h)\) thus obtaining the relation (eq:convergenza-uniforme-teorema-epsilon?).

From this theorem arises an important consequence that links ERM to agnostic PAC learning:

If a learning set \(\mathcal{S}\) has a probability of at least \(1-\delta\) of being \(\epsilon-\)representative, then the ERM applied to this set is an agnostic PAC learner.

4.2.1 Formalization through uniform convergence

Another way to formalize what is written in the last definition is the following:

A hypothesis class \(\mathcal{H}\) has the property of uniform convergence with respect to a domain \(Z\) and a loss function \(l\) if there exists a function such that:

- for every $\epsilon,\delta\in(0,1)$
- for every probability distribution $\mathcal{D}$ on $Z$
- if $S$ is a learning set consisting of $m\geqslant m_{\mathcal{H}}^{UC}(\epsilon,\delta)$ points drawn randomly and i.i.d. according to $\mathcal{D}$

then, with a probability of at least \(1-\delta\), \(S\) is \(\epsilon-\)representative.

It is clear that, as we will see formalized in the next theorem, stating that \(\mathcal{H}\) has the property of uniform convergence with respect to \(S\) automatically also means saying that the ERM applied to the latter is an agnostic PAC learner, since in both cases the hypothesis of \(\epsilon-\)representativeness of \(S\) is satisfied.

4.3 Uniform convergence and PAC learnability

We have finally reached the initial objective of this chapter, which is to link the new concept introduced - that of uniform convergence - with PAC learning. The following theorem, in fact, briefly formalizes this link.

If a hypothesis class \(\mathcal{H}\) has the property of uniform convergence with a function \(m_{\mathcal{H}}^{UC}\), then:

- $\mathcal{H}$ is **agnostically PAC learnable** with sample complexity

$$
    m_{\mathcal{H}}(\epsilon,\delta)\leqslant m_{\mathcal{H}}^{UC}\left(\frac{\epsilon}{2},\delta\right)
$$

- The learning paradigm $ERM_{\mathcal{H}}$ is an **agnostic PAC learner** for $\mathcal{H}$

The second point - as we have already said - follows directly from the definition of uniform convergence for a class. The first point is a direct consequence of the theorem on \(\epsilon-\)representativeness, where - we recall - the \(\frac{\epsilon}{2}-\)representativeness of the learning set is required to have an accuracy of \(\epsilon\) for the error (equation (eq:convergenza-uniforme-teorema-epsilon?)).

4.4 Hoeffding’s Inequality

A fundamental concentration inequality underlies the connection between empirical and true errors for a single hypothesis.

Let \(\theta_1, \theta_2, \ldots, \theta_m\) be independent and identically distributed random variables such that \(a \leq \theta_i \leq b\) for all \(i\). Let \(\mu = \mathbb{E}[\theta_i]\) and \(\hat{\theta} = \frac{1}{m}\sum_{i=1}^m \theta_i\). Then for any \(\varepsilon > 0\):

\[ \mathbb{P}\left(|\hat{\theta} - \mu| \geq \varepsilon\right) \leq 2\exp\left(-\frac{2m\varepsilon^2}{(b-a)^2}\right) \]

Intuition: This inequality bounds how far the empirical average of a random variable can deviate from its true mean, with the bound decreasing exponentially in the sample size \(m\). When applied to a hypothesis \(h\), we can set \(\theta_i = \mathbb{1}[h(x_i) \neq y_i]\) with \(a=0, b=1\), giving us a concentration bound on the training error around the true error.

Application to uniform convergence: If we apply Hoeffding to each hypothesis \(h \in \mathcal{H}\) and use a union bound over all hypotheses, we get that with high probability all hypotheses simultaneously satisfy \(| L_{\mathcal{S}}(h) - L_{\mathcal{D}}(h)| \leq \varepsilon\), provided the sample size satisfies:

\[ m \geq \frac{\log(2|\mathcal{H}|/\delta)}{2\varepsilon^2} \]

4.5 Finite classes are agnostic PAC learnable

Uniform convergence is also useful in reference to finite hypothesis classes. From the corollary on PAC learning, we know that all finite hypothesis classes are PAC learnable, but we have not said anything about agnostic PAC learnability.

Given a finite hypothesis class \(\mathcal{H}\) and a domain \(Z\), consider a loss function . We have:

- $\mathcal{H}$ enjoys the property of **uniform convergence** with a sample complexity

$$
    m_{\mathcal{H}}^{UC}(\epsilon,\delta)\leqslant\frac{\log\left(\frac{2|\mathcal{H}|}{\delta}\right)}{2\epsilon^2}
$$

- $\mathcal{H}$ is **agnostic PAC learnable** using the ERM algorithm with sample complexity

$$
    m_{\mathcal{H}}(\epsilon,\delta)\leqslant m_{\mathcal{H}}^{UC}\left(\frac{\epsilon}{2},\delta\right)\leqslant \frac{2\log\left(\frac{2|\mathcal{H}|}{\delta}\right)}{\epsilon^2}
$$

The second point is easily proved from the first, since once it is established that \(\mathcal{H}\) enjoys uniform convergence, then the previous theorem holds, which assures us of agnostic PAC learnability and also specifies the upper limit of the sample complexity. The proof would therefore be reduced to that of the first point, but it is not required by the course.

It is important to underline that if PAC learnability is always valid for a finite class, agnostic PAC learnability is valid only considering a loss function whose codomain is \([0,1]\).

4.5.1 Discretization of an infinite class

In practice, although the hypothesis class \(\mathcal{H}\) is infinite, it is still possible to apply the theorem just written through the so-called discretization trick.

This is based on the fact that computers do not work with real numbers with infinite precision, but with discrete quantities; each real number, normally, is stored through 64-bit variables (working in double precision) and therefore \(2^{64}\) numbers in total are possible. If the hypothesis class is parameterized by a total of \(d\) real numbers, then at most it will result:

\[ |\mathcal{H}|=\underbrace{2^{64}\cdot2^{64}\cdot\dotsc\cdot2^{64}}_{d \ \text{times}}=2^{64d} \]

which is a very large number, but still finite.

Substituting this result in the upper bound of equation (eq:convergenza-uniforme-classi-finite-pac-learnability?) we obtain: (From this calculation I also discover that the logarithm written so far has always been in base \(2\) although it was never specified by the professor, because otherwise this calculation would not be correct.)

\[ \frac{2\log\left(\frac{2\cdot2^{64d}}{\delta}\right)}{\epsilon^2}=\frac{2\log\left(2^{64d}\right)+2\log\left(\frac{2}{\delta}\right)}{\epsilon^2}=\frac{2\cdot64d+2\log\left(\frac{2}{\delta}\right)}{\epsilon^2}=\frac{128d+2\log\left(\frac{2}{\delta}\right)}{\epsilon^2} \]

A fundamental concentration inequality underlies the connection between empirical and true errors for a single hypothesis.

$$
\mathbb{P}\left(|\hat{\theta} - \mu| \geq \varepsilon\right) \leq 2\exp\left(-\frac{2m\varepsilon^2}{(b-a)^2}\right)
$$
:::

**Intuition:** This inequality bounds how far the empirical average of a random variable can deviate from its true mean, with the bound decreasing exponentially in the sample size $m$. When applied to a hypothesis $h$, we can set $\theta_i = \mathbb{1}[h(x_i) \neq y_i]$ with $a=0, b=1$, giving us a concentration bound on the training error around the true error.

$$
    m \geq \frac{\log(2|\mathcal{H}|/\delta)}{2\varepsilon^2}
$$
