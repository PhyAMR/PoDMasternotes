13  Neural Networks and Convolutional Neural Networks

13.1 Neural Network Architecture

A neural network is a directed graph of interconnected neurons (nodes), where each connection has an associated learnable weight. The network is typically organized in layers, allowing for efficient hierarchical computation.

13.1.1 Feedforward Architecture

In a feedforward neural network, neurons are organized into layers \(\mathcal{L}_0, \mathcal{L}_1, \ldots, \mathcal{L}_T\) where: - Layer 0 is the input layer - Layers 1 to \(T-1\) are hidden layers - Layer \(T\) is the output layer - Each neuron in layer \(t\) receives inputs only from layer \(t-1\)

13.1.2 Layer Computation

For a neuron \(i\) in layer \(t\), the computation is:

  1. Pre-activation (weighted sum): \[z_i^{(t)} = \sum_{j} w_{ij}^{(t)} a_j^{(t-1)} + b_i^{(t)}\] where \(a_j^{(t-1)}\) are activations from the previous layer, \(w_{ij}^{(t)}\) are weights, and \(b_i^{(t)}\) is the bias.

  2. Activation: \[a_i^{(t)} = \sigma(z_i^{(t)})\] where \(\sigma\) is an activation function.

In matrix form for a full layer: \[\mathbf{a}^{(t)} = \sigma(W^{(t)} \mathbf{a}^{(t-1)} + \mathbf{b}^{(t)})\]

13.1.3 Activation Functions

Different activation functions provide different properties:

Function Formula Derivative Properties
Sigmoid \(\sigma(z) = \frac{1}{1+e^{-z}}\) \(\sigma'(z) = \sigma(z)(1-\sigma(z))\) Smooth, bounded \([0,1]\); can saturate (small gradients when \(z\) large/small)
Tanh \(\sigma(z) = \frac{e^z - e^{-z}}{e^z + e^{-z}}\) \(\sigma'(z) = 1 - \sigma^2(z)\) Smooth, centered at 0, bounded \([-1,1]\); similar saturation issues as sigmoid
ReLU \(\sigma(z) = \max(0, z)\) \(\sigma'(z) = \mathbb{1}_{z > 0}\) Simple, fast; sparse activation; dead ReLU problem when \(z < 0\)
Softmax \(\sigma(z_i) = \frac{e^{z_i}}{\sum_j e^{z_j}}\) Element-wise Jacobian For output layer in multiclass; outputs sum to 1

ReLU is the most commonly used activation in modern networks due to computational simplicity and fewer saturation issues.

13.2 Universal Approximation Theorem

Let \(\sigma\) be any non-polynomial activation function (e.g., sigmoid, tanh). Then for any continuous function \(f: [0,1]^d \to \mathbb{R}\) and any \(\varepsilon > 0\), there exists a feedforward network with a single hidden layer such that the network output \(\hat{f}\) satisfies:

\[ |\hat{f}(\mathbf{x}) - f(\mathbf{x})| < \varepsilon \text{ for all } \mathbf{x} \in [0,1]^d \]

This fundamental result guarantees that neural networks with one hidden layer can approximate any continuous function. However, it says nothing about the number of neurons required, which can be exponentially large. In practice, deep networks (many layers) can be much more efficient for certain functions, which is why deep learning is powerful.

13.3 Backpropagation Algorithm

Backpropagation computes gradients of the loss with respect to all weights by applying the chain rule recursively from the output layer backward to the input layer.

13.3.1 Backpropagation Recurrence

Let \(L\) denote the loss function. Define the error signal (or delta) for layer \(t\) as:

\[ \delta_i^{(t)} = \frac{\partial L}{\partial z_i^{(t)}} \]

For the output layer \(T\): \[ \delta_i^{(T)} = \frac{\partial L}{\partial a_i^{(T)}} \cdot \sigma'(z_i^{(T)}) \]

For hidden layer \(t < T\) (recursive): \[ \delta_i^{(t)} = \left(\sum_j w_{ji}^{(t+1)} \delta_j^{(t+1)}\right) \cdot \sigma'(z_i^{(t)}) \]

Weight gradients: \[ \frac{\partial L}{\partial w_{ij}^{(t)}} = \delta_i^{(t)} a_j^{(t-1)} \]

The backpropagation algorithm computes these deltas in reverse topological order, then updates weights using gradient descent or a variant like Adam.

13.3.2 Example: Two-Layer Network

Consider a network: input (\(d\) dims) \(\to\) hidden layer (\(h\) neurons) \(\to\) output (1 neuron, regression).

Forward pass: - \(\mathbf{z}^{(1)} = W^{(1)} \mathbf{x} + \mathbf{b}^{(1)}\) - \(\mathbf{a}^{(1)} = \text{ReLU}(\mathbf{z}^{(1)})\) - \(z^{(2)} = \mathbf{w}^{(2)} \mathbf{a}^{(1)} + b^{(2)}\) - \(\hat{y} = z^{(2)}\) (linear output for regression) - Loss: \(L = \frac{1}{2}(\hat{y} - y)^2\)

Backward pass: - \(\delta^{(2)} = \frac{\partial L}{\partial z^{(2)}} = \hat{y} - y\) - \(\delta_i^{(1)} = \mathbf{w}_i^{(2)} \delta^{(2)} \cdot \mathbb{1}_{z_i^{(1)} > 0}\) (ReLU derivative) - \(\frac{\partial L}{\partial W^{(2)}} = \delta^{(2)} (\mathbf{a}^{(1)})^T\) - \(\frac{\partial L}{\partial W^{(1)}} = \boldsymbol{\delta}^{(1)} \mathbf{x}^T\)

13.4 Practical Training Techniques

Modern neural network training requires several techniques for stability and efficiency:

  • Mini-batch training: Compute gradients on small batches (e.g., 32 samples) instead of the full dataset, reducing memory and adding regularization via noise.
  • Momentum: Accumulate gradients over steps: \(\mathbf{v}_{t+1} = \mu \mathbf{v}_t - \eta \nabla L\), then \(\boldsymbol{\theta}_{t+1} = \boldsymbol{\theta}_t + \mathbf{v}_{t+1}\). Helps escape shallow local minima.
  • Dropout: Randomly zero out activations with probability \(p\) during training to prevent co-adaptation and reduce overfitting.
  • Glorot initialization: Initialize weights from \(U[-\sqrt{6/(n_{\text{in}}+n_{\text{out}})}, \sqrt{6/(n_{\text{in}}+n_{\text{out}})}]\) to maintain stable gradient flow across layers.
  • Weight decay (L2 regularization): Add \(\lambda \|\boldsymbol{\theta}\|^2\) to the loss to prevent large weights.
  • Batch normalization: Normalize layer activations to have zero mean and unit variance within each batch, reducing internal covariate shift and allowing higher learning rates.

13.5 Convolutional Neural Networks (CNNs)

For image and grid-like data, fully connected networks are inefficient (huge parameter count). Convolutional Neural Networks exploit spatial structure through local connectivity and weight sharing.

13.5.1 Key Ideas

  1. Local connectivity: Each neuron connects only to a small local region (receptive field) of the previous layer.
  2. Weight sharing: The same filter (weights) is applied across all spatial positions, detecting the same feature everywhere.
  3. Feature maps: Multiple filters create multiple feature maps, each detecting different patterns.
  4. Pooling: Downsampling layers reduce spatial dimension and aggregate information.

13.5.2 Convolution Operation

A convolution with a filter \(F\) of size \(k \times k\), stride \(s\), and padding \(p\) produces output of size:

\[ \text{output size} = \left\lfloor \frac{\text{input size} - k + 2p}{s} \right\rfloor + 1 \]

Each output element is the dot product of the filter with a corresponding patch of the input:

\[ \text{out}[i, j] = \sum_{u=0}^{k-1} \sum_{v=0}^{k-1} F[u, v] \cdot \text{input}[i \cdot s + u, j \cdot s + v] \]

Stride controls how much the filter shifts (larger stride \(\to\) smaller output); padding adds zeros around the input border.

13.5.3 Pooling

Max pooling downsamples by taking the maximum value in each \(p \times p\) receptive field:

\[ \text{out}[i, j] = \max_{u, v \in \text{field}} \text{input}[i \cdot s + u, j \cdot s + v] \]

Pooling provides translation invariance and reduces computation in deeper layers. A receptive field grows by \(2k-1\) per convolution layer (for stride 1), allowing deep networks to capture large-scale context.

13.5.4 Computational Savings

Convolution is vastly more efficient than fully connected layers: - Fully connected: \(d_{\text{in}} \times d_{\text{out}}\) parameters - Convolution: \(k \times k \times c_{\text{in}} \times c_{\text{out}}\) parameters (independent of spatial size)

For a \(28 \times 28\) image with 32 output channels and \(3 \times 3\) filters: only \(3 \times 3 \times 32 \approx 3000\) parameters per filter, vs. millions for fully connected.

13.5.5 Reference Architectures

  • LeNet (1998): Conv \(\to\) Pool \(\to\) Conv \(\to\) Pool \(\to\) FC \(\to\) FC. Pioneered CNNs for MNIST digit classification.
  • AlexNet (2012): 5 convolutional layers (with ReLU) + 3 fully connected layers. Won ImageNet competition; enabled the deep learning revolution with GPU training.
  • VGG (2014): Simple, deep architecture (up to 19 layers) using \(3 \times 3\) convolutions. Demonstrated that depth is crucial; still widely used as a feature extractor.

Modern architectures (ResNet, EfficientNet, ViT) add residual connections, attention mechanisms, and better optimization to go even deeper while maintaining trainability.

13.6 Summary

Neural networks compose linear transformations and nonlinearities to learn complex mappings. Backpropagation efficiently computes gradients layer-by-layer. Practical techniques (momentum, dropout, batch norm, Glorot init) are essential for successful training. CNNs leverage spatial locality and weight sharing for dramatic efficiency gains on image tasks, making them the standard for computer vision.

## Backpropagation Algorithm

Backpropagation computes gradients of the loss with respect to all weights by applying the chain rule recursively from the output layer backward to the input layer.

### Backpropagation Recurrence

Let $L$ denote the loss function. Define the **error signal** (or delta) for layer $t$ as:

$$
    \delta_i^{(t)} = \frac{\partial L}{\partial z_i^{(t)}}
$$

**For the output layer $T$:**
$$
    \delta_i^{(T)} = \frac{\partial L}{\partial a_i^{(T)}} \cdot \sigma'(z_i^{(T)})
$$

**For hidden layer $t < T$** (recursive):
$$
    \delta_i^{(t)} = \left(\sum_j w_{ji}^{(t+1)} \delta_j^{(t+1)}\right) \cdot \sigma'(z_i^{(t)})
$$

**Weight gradients:**
$$
    \frac{\partial L}{\partial w_{ij}^{(t)}} = \delta_i^{(t)} a_j^{(t-1)}
$$

The backpropagation algorithm computes these deltas in reverse topological order, then updates weights using gradient descent or a variant like Adam.

### Example: Two-Layer Network

:::{.example name="Two-layer MLP"}
Consider a network: input ($d$ dims) $\to$ hidden layer ($h$ neurons) $\to$ output (1 neuron, regression).

**Forward pass:**
- $\mathbf{z}^{(1)} = W^{(1)} \mathbf{x} + \mathbf{b}^{(1)}$
- $\mathbf{a}^{(1)} = \text{ReLU}(\mathbf{z}^{(1)})$
- $z^{(2)} = \mathbf{w}^{(2)} \mathbf{a}^{(1)} + b^{(2)}$
- $\hat{y} = z^{(2)}$ (linear output for regression)
- Loss: $L = \frac{1}{2}(\hat{y} - y)^2$

**Backward pass:**
- $\delta^{(2)} = \frac{\partial L}{\partial z^{(2)}} = \hat{y} - y$
- $\delta_i^{(1)} = \mathbf{w}_i^{(2)} \delta^{(2)} \cdot \mathbb{1}_{z_i^{(1)} > 0}$ (ReLU derivative)
- $\frac{\partial L}{\partial W^{(2)}} = \delta^{(2)} (\mathbf{a}^{(1)})^T$
- $\frac{\partial L}{\partial W^{(1)}} = \boldsymbol{\delta}^{(1)} \mathbf{x}^T$
:::
