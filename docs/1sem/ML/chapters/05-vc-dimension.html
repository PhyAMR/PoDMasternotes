6  VC Dimension

In the last chapter, we discussed the importance of the chosen hypothesis class in finding the function that best labels a data distribution, paying particular attention to its size and how it correlates with the errors associated with the learning algorithm.

Our main goal, however, remains to understand which classes \(\mathcal{H}\) are PAC learnable or, at the limit, agnostically PAC learnable, that is, to know with certainty that the error is bounded by a certain quantity as a function of the parameters \(\epsilon\) and \(\delta\).

Through uniform convergence, we have shown that all finite classes are agnostically PAC learnable, while through the No-Free-Lunch theorem we know that the set of all functions certainly does not enjoy the same property. It therefore remains to be discovered whether infinite classes are, in general, PAC learnable or not.

6.1 The example of the threshold function

To affirm that not only finite classes are PAC learnable (Or, in other words, that the property of a class being finite is a sufficient but not necessary condition for PAC learnability.) a single counterexample is enough, which can be provided by the threshold function.

It is a very simple function, dependent on a single parameter \(a\) which can take any value in the set of real numbers. It is defined as:

\[ h_a(x)= \begin{cases} 1 \ \ \ x<a\\ 0 \ \ \ x\geqslant a \end{cases} \]

which, in short, can also be written as \(\mathds{1}_{[x<a]}\).

We are therefore in the case of binary classification, where a value is classified in one way if it is below the threshold (threshold, precisely) \(a\), and in another if it is above it.

The class \(\mathcal{H}=\{h_a:a\in\mathbb{R}\}\) is evidently infinite, the set \(\mathbb{R}\) being infinite, but it is possible to show that it is PAC learnable with sample complexity

\[ m_{\mathcal{H}}(\epsilon,\delta)=\frac{\log\frac{2}{\delta}}{\epsilon} \]

We will see the proof later, but in reality the reason is quite intuitive: given a training set \(\mathcal{S}\) with a certain number of points divided into \(0\) and \(1\), the parameter \(a\) must necessarily be between the largest point classified as \(1\) and the smallest point classified as \(0\), since otherwise contradictions would be encountered; this fact significantly limits the values that the parameter \(a\) can take and, intuitively, makes \(\abs{\mathcal{H}}<\infty\).

6.2 Restriction of a function

To understand which hypothesis classes are PAC learnable, it is useful to define the restriction of a class \(\mathcal{H}\) to a set \(\mathcal{C}\).

For simplicity, we always remain in the context of binary classifications, so \(\mathcal{H}\) represents a set of functions that go from a domain \(\mathcal{X}\) to \(\{0,1\}\). Let us now consider a subset \(\mathcal{C}\) of the domain \(\mathcal{X}\):

\[ \mathcal{C}=\left\{c_1,\dots,c_m\right\}\subset\mathcal{X} \]

The restriction of \(\mathcal{H}\) to \(\mathcal{C}\) is defined as the set of functions from \(\mathcal{C}\) to \(\{0,1\}\) that can be derived from \(\mathcal{H}\):

\[ \mathcal{H}_{\mathcal{C}}=\left\{h\left(c_1\right),\dots,h\left(c_m\right):h\in\mathcal{H}\right\} \]

In the context of binary classification, each function \(h\left(c_i\right)\) is a vector of length \(m\) composed of \(0\)s and \(1\)s.

To visualize the concept of restriction, let’s go back to the example of the threshold function and imagine selecting three points on the real axis; it is clear that there are only four possible classifications (below threshold, above threshold, etc.). We have thus restricted, precisely, the class \(\mathcal{H}\) to four possible cases, formally represented by four vectors composed of \(0\)s and \(1\)s of size equal to \(|\mathcal{C}|=3\).

6.3 Shattering a set

From the restriction of a hypothesis class, it is possible to define the shattering (In Italian frantumazione.) of a set:

Given a set \(\mathcal{C}\subset\mathcal{X}\), we will say that \(\mathcal{H}\) shatters \(\mathcal{C}\) if its restriction \(\mathcal{H}_{\mathcal{C}}\) contains all \(2^{\abs{\mathcal{C}}}\) functions from \(\mathcal{C}\) to \(\{0,1\}\).

The example proposed above, with the threshold function and \(\mathcal{C}=\left\{c_1,c_2,c_3\right\}\), clearly shows that \(\mathcal{H}\) does not shatter \(\mathcal{C}\), since \(2^{\mathcal{C}}=8\) but the restriction includes only four possible functions.

This concept is useful to us because it leads directly back to overfitting: shattering a set, in fact, is equivalent to saying that any combination of points belonging to that set can be reproduced by a hypothesis of the considered class; this means that the number \(\abs{\mathcal{C}}\) can give us an indication of how valid \(\mathcal{H}\) is for solving a given problem.

6.3.1 Relationship between shattering and No-Free-Lunch

The (non) validity of a hypothesis class due to its size was discussed in the No-Free-Lunch theorem and its related corollary, where it was shown that the class containing all functions is not PAC learnable.

The concept of shattering is important for extending this reasoning to a generic hypothesis class \(\mathcal{H}\). The underlying procedure, in fact, is the same: to identify an algorithm that fails in learning for a given data distribution; for this reason, the following theorem is indicated as another corollary of the No-Free-Lunch theorem:

Let:

item $\mathcal{H}$ be a hypothesis class containing functions that go from $\mathcal{X}$ to $\{0,1\}$

item $m$ be the size of a learning set

item $\mathcal{C}\subset\mathcal{X}$ be a set of size $2m$ that is *shattered* by $\mathcal{H}$

Under these hypotheses, the No-Free-Lunch theorem is valid.

The proof is not required for the course, but it is important to understand intuitively what this corollary is communicating to us: in short, we can say that if \(\mathcal{H}\) is able to shatter a certain set \(\mathcal{C}\), a subset of the domain \(\mathcal{X}\), then a set containing up to half the number of points of \(\mathcal{C}\) is not a good learning set for the algorithm; learning on a set of that type, in fact, is such that it does not provide real information for predicting the classification of further points drawn from the same distribution.

6.4 Definition of VC dimension

Having understood that the limit in the learning of an algorithm is given by the size of \(\abs{\mathcal{C}}\) for which there is shattering by \(\mathcal{H}\), we can formalize this aspect by defining a quantity called VC dimension, which takes its name from the two Russian mathematicians Vapnik and Chervonenkis.

The VC dimension of a hypothesis class \(\mathcal{H}\), denoted by \(VCdim(\mathcal{H})\), is the maximum size of a set \(\mathcal{C}\subset\mathcal{X}\) that can be shattered by \(\mathcal{H}\).

If \(\mathcal{H}\) can shatter sets of arbitrarily large size, we will say that \(\mathcal{H}\) has an infinite VC dimension.

It is easy to verify that if \(VCdim(\mathcal{H})=\infty\) then \(\mathcal{H}\) is certainly PAC learnable, since whatever size \(m\) of the learning set we choose, \(\mathcal{H}\) will be able to shatter a set twice as large, from which the No-Free-Lunch theorem follows.

Another easy result to prove is that a finite hypothesis class certainly has a finite VC dimension: for \(\mathcal{H}\) to be able to shatter \(\mathcal{C}\), in fact, it is necessary that

\[ \abs{\mathcal{H}}\geqslant2^{\abs{\mathcal{C}}} \]

From which, deriving \(\abs{\mathcal{C}}\):

\[ 2^{\abs{\mathcal{C}}}\leqslant\abs{\mathcal{H}} \ \Longrightarrow \ \abs{\mathcal{C}}\leqslant\log_2\abs{\mathcal{H}} \]

If \(\abs{\mathcal{C}}\) is the maximum size of the set that can be shattered by \(\mathcal{H}\), then it will be nothing other than the VC dimension:

\[ VCdim(\mathcal{H})\leqslant\log_2\abs{\mathcal{H}} \]

The reverse, however, is not true, so there can exist infinite classes with a finite VC dimension; we will see some examples in this regard.

6.4.1 Calculation of the VC dimension

To prove that the VC dimension of a class is finite and equal to a certain value \(d\), it is necessary to proceed by following two steps:

item Show that there exists a set $\mathcal{C}$ of size $d$ that can be shattered by $\mathcal{H}$:

$$
    VCdim(\mathcal{H})\geqslant d
$$


item Show that \underline{*all*} sets of size $d+1$ cannot be shattered by $\mathcal{H}$:

$$
    VCdim(\mathcal{H})<d+1
$$

If these two conditions are met, then it is clear that

\[ VCdim(\mathcal{H})=d \]

6.4.1.1 Calculation for the threshold function

With \(\abs{\mathcal{C}}=1\), it is clear that the set can be shattered, since it is sufficient to place the parameter \(a\) to the left or right of \(c_1\) to classify it as \(0\) or \(1\).

With \(|\mathcal{C}|=2\), there is no shattering since it is impossible to have the classification \((0,1)\) and only the remaining three are possible. Following the steps mentioned at the beginning of the paragraph, we will have:

\[ \begin{cases} VCdim(\mathcal{H})\geqslant1\\ VCdim(\mathcal{H})<2 \end{cases} \Longrightarrow \ \ \boxed{VCdim(\mathcal{H})=1} \]

6.4.1.2 Calculation for the interval function

Let’s define the interval function through two parameters \(a\), \(b\) as follows:

\[ h_{a,b}(x)= \begin{cases} 1 \ \ \ a<x<b\\ 0 \ \ \ otherwise \end{cases} \]

The hypothesis class will therefore be:

\[ \mathcal{H}=\left\{h_{a,b}:a,b\in\mathbb{R},a<b\right\} \]

This time a set \(\mathcal{C}\) with \(|\mathcal{C}|=2\) can be shattered by the hypothesis class. Moving on to \(|\mathcal{C}|=3\), the restriction \(\mathcal{H}_{\mathcal{C}}\) contains the functions for all classifications except one, namely \((1,0,1)\), as it is impossible to place the central point outside the interval while simultaneously including the other two inside.

It follows that:

\[ \begin{cases} VCdim(\mathcal{H})\geqslant2\\ VCdim(\mathcal{H})<3 \end{cases} \Longrightarrow \ \ \boxed{VCdim(\mathcal{H})=2} \]

6.4.1.3 Calculation for the rectangular function

Let’s move to \(\mathbb{R}^2\) and consider the following function dependent on four parameters:

\[ h_{a_1,a_2,b_1,b_2}\left(x_1,x_2\right)= \begin{cases} 1 \ \ \ a_1\leqslant x_1\leqslant a_2, \ b_1\leqslant x_2\leqslant b_2\\ 0 \ \ \ otherwise \end{cases} \]

The class is therefore:

\[ \mathcal{H}=\left\{h_{a_1,a_2,b_1,b_2}:a_1,a_2,b_1,b_2\in\mathbb{R},a_1\leqslant a_2,b_1\leqslant b_2\right\} \]

As expected, there are examples up to \(\abs{\mathcal{C}}=4\) in which \(\mathcal{H}\) is able to shatter the set. For \(\abs{\mathcal{C}}=5\), however, any set \(\mathcal{C}\) is no longer shatterable by \(\mathcal{H}\), since the most "internal" of the five cannot be classified as \(0\) while the other four are classified as \(1\).

It will therefore be:

\[ \begin{cases} VCdim(\mathcal{H})\geqslant4\\ VCdim(\mathcal{H})<5 \end{cases} \Longrightarrow \ \ \boxed{VCdim(\mathcal{H})=4} \]

6.4.1.4 Calculation for the sine ceiling function

In the examples seen so far, the VC dimension has coincided with the number of parameters useful for defining the functions within the hypothesis class. This, in reality, is not a general rule, and in this regard we will see an example below in which the two values are different.

Let’s consider the ceiling function applied to the sine function modulated by a factor of \(0.5\) and whose frequency is determined by a free parameter \(\theta\):

\[ h_\theta(x)=\lceil 0.5\sin(\theta x)\rceil \]

We recall that the ceiling function returns the smallest integer greater than or equal to the input value; since the modulated sine oscillates between \(-0.5\) and \(0.5\), then the function \(h_\theta(x)\) will alternately take only the values \(0\) (when the sine is negative) and \(1\) (when the sine is positive). It is possible to show that for any combination of points classified as \(0\) and \(1\), in whatever number they are, there exists a value of \(\theta\) capable of determining a function in \(\mathcal{H}\) that correctly describes this classification. In other words, the VC dimension of this class is infinite, despite the fact that the parameter on which it depends is only one.

6.5 Theorem of statistical learning

What we have arrived at so far has been to state that if the VC dimension of a class is infinite, then it is certainly not PAC learnable, and we have also shown that a finite class (which we know to be PAC learnable) has a finite VC dimension.

There is still one type to explore, namely infinite classes with a finite VC dimension; the question is: what can we say about their PAC learnability?

We will see below an important theorem - not by chance called the fundamental theorem - which guarantees the PAC learnability of all classes with a finite VC dimension. We will thus have managed to reach our initial goal, which was to find a parameter that is directly linked to the PAC learnability of a class: it is, in fact, the VC dimension.

Let \(\mathcal{H}\) be a hypothesis class containing functions that go from a domain \(\mathcal{X}\) to \(\{0,1\}\) and let the loss function be a \(0-1\) objective function.

Then, the following statements are equivalent:

item $\mathcal{H}$ has the property of **uniform convergence**


item Any ERM algorithm is an **agnostic PAC learner** for $\mathcal{H}$


item $\mathcal{H}$ is **agnostic PAC learnable**



item $\mathcal{H}$ is **PAC learnable**



item Any ERM algorithm is a **PAC learner** for $\mathcal{H}$



item $\mathcal{H}$ has **finite VC dimension**

We do not give the proof in detail, also because most of them are obvious or have already been seen previously:

item The implications $1\rightarrow2\rightarrow3$ derive from the theorem on the link between uniform convergence and PAC learning, which we saw in section @sec:convergenza-uniforme-e-pac-learnability


item The implication $3\rightarrow4$ is obvious, since

$$
    \underbrace{\underbrace{L_{\mathcal{D}}(h)\leqslant\epsilon}_{\text{PAC learning}}\leqslant\min_{h'\in\mathcal{H}}L_{\mathcal{D}}\left(h'\right)+\epsilon}_{\text{Agnostic PAC learning}}
$$

and the two properties coincide if the realizability assumption is valid (and therefore the minimum in the last member is zero)


item For the same reason just seen, $2\rightarrow5$ also holds


item By contradiction, if the infinite VC dimension for a hypothesis class implies its non-PAC learnability, then for a class to be PAC learnable it is necessary that it has a finite VC dimension, so we will have $4\rightarrow6$ and $5\rightarrow6$

To close the circle, it is necessary to show that \(6\rightarrow1\). The proof is not required by the course, so it will not be given in detail; suffice it to say that it is divided into two parts:

item Let $d$ be the VC dimension of $\mathcal{H}$, even if $\abs{\mathcal{H}}=\infty$, by applying the restriction of $\mathcal{H}$ to $\mathcal{C}$ it is shown that the "effective size" $\abs{\mathcal{H}_{\mathcal{C}}}$ is $O\left(\abs{\mathcal{C}}^d\right)$, so the size of $\mathcal{H}_{\mathcal{C}}$ grows *polynomially* with respect to $\abs{\mathcal{C}}$. This part of the proof is known as *Sauer's lemma*.


item We know that finite classes enjoy the property of uniform convergence, but it is possible to generalize this result by stating that uniform convergence is valid for all those classes with "effective size" small, that is, with a polynomial and not exponential growth with respect to $\abs{\mathcal{C}}$.

6.5.1 The link between VC dimension and sample complexity

The VC dimension characterizes not only PAC learnability but also the sample complexity associated with the various properties of the considered hypothesis class. This is demonstrated in the quantitative version of the fundamental theorem of statistical learning.

Let \(\mathcal{H}\) be a hypothesis class containing functions that go from a domain \(\mathcal{X}\) to \(\{0,1\}\) and let the loss function be a \(0-1\) objective function. Assume, moreover, that \(VCdim(\mathcal{H})=d<\infty\).

Then, there exist constants \(C_1\) and \(C_2\) such that:

item $\mathcal{H}$ enjoys uniform convergence with sample complexity

$$
    C_1\frac{d+\log\left(\frac{1}{\delta}\right)}{\epsilon^2}\leqslant m_{\mathcal{H}}^{UC}(\epsilon,\delta)\leqslant C_2\frac{d+\log\left(\frac{1}{\delta}\right)}{\epsilon^2}
$$


item $\mathcal{H}$ is agnostically PAC learnable with sample complexity

$$
    C_1\frac{d+\log\left(\frac{1}{\delta}\right)}{\epsilon^2}\leqslant m_{\mathcal{H}}(\epsilon,\delta)\leqslant C_2\frac{d+\log\left(\frac{1}{\delta}\right)}{\epsilon^2}
$$


item $\mathcal{H}$ is PAC learnable with sample complexity

$$
    C_1\frac{d+\log\left(\frac{1}{\delta}\right)}{\epsilon}\leqslant m_{\mathcal{H}}(\epsilon,\delta)\leqslant C_2\frac{d\log\left(\frac{1}{\epsilon}\right)+\log\left(\frac{1}{\delta}\right)}{\epsilon}
$$

The proof is not required for this theorem either, but it is interesting to note how the ends of the interval are similar to the upper bound of the sample complexity seen in equation (eq:convergenza-uniforme-sample-complexity?). The numerator, in particular, can be rewritten as

\[ \log\left(\frac{2\abs{\mathcal{H}}}{\delta}\right)=\log(2\abs{\mathcal{H}})+\log\left(\frac{1}{\delta}\right) \]

So it is clear that, in this context, it is as if the VC dimension of \(\mathcal{H}\), i.e., \(d\), were playing the role of \(\log(2\abs{\mathcal{H}})\).

$$
    \mathcal{C}=\left\{c_1,\dots,c_m\right\}\subset\mathcal{X}
$$

The **restriction** of $\mathcal{H}$ to $\mathcal{C}$ is defined as the set of functions from $\mathcal{C}$ to $\{0,1\}$ that can be derived from $\mathcal{H}$:

$$
    \mathcal{H}_{\mathcal{C}}=\left\{h\left(c_1\right),\dots,h\left(c_m\right):h\in\mathcal{H}\right\}
$$

In the context of binary classification, each function $h\left(c_i\right)$ is a vector of length $m$ composed of $0$s and $1$s.

From the restriction of a hypothesis class, it is possible to define the **shattering** *(In Italian *frantumazione*.)* of a set:

:::{.definition} 
Given a set $\mathcal{C}\subset\mathcal{X}$, we will say that $\mathcal{H}$ **shatters** $\mathcal{C}$ if its restriction $\mathcal{H}_{\mathcal{C}}$ contains *all* $2^{\abs{\mathcal{C}}}$ functions from $\mathcal{C}$ to $\{0,1\}$.
:::

The example proposed above, with the threshold function and $\mathcal{C}=\left\{c_1,c_2,c_3\right\}$, clearly shows that $\mathcal{H}$ does not shatter $\mathcal{C}$, since $2^{\mathcal{C}}=8$ but the restriction includes only four possible functions.

This concept is useful to us because it leads directly back to overfitting: shattering a set, in fact, is equivalent to saying that any combination of points belonging to that set can be reproduced by a hypothesis of the considered class; this means that the number $\abs{\mathcal{C}}$ can give us an indication of how valid $\mathcal{H}$ is for solving a given problem.

The (non) validity of a hypothesis class due to its size was discussed in the No-Free-Lunch theorem and its related corollary, where it was shown that the class containing all functions is not PAC learnable.

The underlying procedure, in fact, is the same: to identify an algorithm that fails in learning for a given data distribution; for this reason, the following theorem is indicated as another corollary of the No-Free-Lunch theorem:

::: {.theorem name="No-Free-Lunch Corollary (with shattering)"}
Let:


    
    item $\mathcal{H}$ be a hypothesis class containing functions that go from $\mathcal{X}$ to $\{0,1\}$
    
    item $m$ be the size of a learning set
    
    item $\mathcal{C}\subset\mathcal{X}$ be a set of size $2m$ that is *shattered* by $\mathcal{H}$


Under these hypotheses, ***the No-Free-Lunch theorem is valid***.
:::

The proof is not required for the course, but it is important to understand intuitively what this corollary is communicating to us: in short, we can say that if $\mathcal{H}$ is able to shatter a certain set $\mathcal{C}$, a subset of the domain $\mathcal{X}$, then a set containing up to half the number of points of $\mathcal{C}$ is not a good learning set for the algorithm; learning on a set of that type, in fact, is such that it does not provide real information for predicting the classification of further points drawn from the same distribution.

## Definition of VC dimension

Having understood that the limit in the learning of an algorithm is given by the size of $\abs{\mathcal{C}}$ for which there is shattering by $\mathcal{H}$, we can formalize this aspect by defining a quantity called **VC dimension**, which takes its name from the two Russian mathematicians Vapnik and Chervonenkis.

:::{.definition} 
The **VC dimension** of a hypothesis class $\mathcal{H}$, denoted by $VCdim(\mathcal{H})$, is the ***maximum size*** of a set $\mathcal{C}\subset\mathcal{X}$ that can be shattered by $\mathcal{H}$.
:::

If $\mathcal{H}$ can shatter sets of arbitrarily large size, we will say that $\mathcal{H}$ has an *infinite* VC dimension.

It is easy to verify that if $VCdim(\mathcal{H})=\infty$ then $\mathcal{H}$ is certainly \underline{*not*} PAC learnable, since whatever size $m$ of the learning set we choose, $\mathcal{H}$ will be able to shatter a set twice as large, from which the No-Free-Lunch theorem follows.

Another easy result to prove is that a finite hypothesis class certainly has a finite VC dimension: for $\mathcal{H}$ to be able to shatter $\mathcal{C}$, in fact, it is necessary that

$$
    \abs{\mathcal{H}}\geqslant2^{\abs{\mathcal{C}}}
$$

From which, deriving $\abs{\mathcal{C}}$:

$$
    2^{\abs{\mathcal{C}}}\leqslant\abs{\mathcal{H}} \ \Longrightarrow \ \abs{\mathcal{C}}\leqslant\log_2\abs{\mathcal{H}}
$$

If $\abs{\mathcal{C}}$ is the maximum size of the set that can be shattered by $\mathcal{H}$, then it will be nothing other than the VC dimension:

$$
    VCdim(\mathcal{H})\leqslant\log_2\abs{\mathcal{H}}
$$

The reverse, however, is not true, so there can exist infinite classes with a finite VC dimension; we will see some examples in this regard.

### Calculation of the VC dimension

To prove that the VC dimension of a class is finite and equal to a certain value $d$, it is necessary to proceed by following two steps:

    
    item Show that there exists a set $\mathcal{C}$ of size $d$ that can be shattered by $\mathcal{H}$:
    
    $$
        VCdim(\mathcal{H})\geqslant d
    $$

    
    item Show that \underline{*all*} sets of size $d+1$ cannot be shattered by $\mathcal{H}$:
    
    $$
        VCdim(\mathcal{H})<d+1
    $$

If these two conditions are met, then it is clear that

$$
    VCdim(\mathcal{H})=d
$$

#### Calculation for the threshold function

With $\abs{\mathcal{C}}=1$, it is clear that the set can be shattered, since it is sufficient to place the parameter $a$ to the left or right of $c_1$ to classify it as $0$ or $1$.

With $|\mathcal{C}|=2$, there is no shattering since it is impossible to have the classification $(0,1)$ and only the remaining three are possible. Following the steps mentioned at the beginning of the paragraph, we will have:

$$
    \begin{cases}
        VCdim(\mathcal{H})\geqslant1\\
        VCdim(\mathcal{H})<2
    \end{cases}
    \Longrightarrow \ \ \boxed{VCdim(\mathcal{H})=1}
$$

#### Calculation for the interval function

Let's define the interval function through two parameters $a$, $b$ as follows:

$$
    h_{a,b}(x)=
    \begin{cases}
        1 \ \ \ a<x<b\\
        0 \ \ \ otherwise
    \end{cases}
$$

The hypothesis class will therefore be:

$$
    \mathcal{H}=\left\{h_{a,b}:a,b\in\mathbb{R},a<b\right\}
$$

This time a set $\mathcal{C}$ with $|\mathcal{C}|=2$ can be shattered by the hypothesis class. Moving on to $|\mathcal{C}|=3$, the restriction $\mathcal{H}_{\mathcal{C}}$ contains the functions for all classifications except one, namely $(1,0,1)$, as it is impossible to place the central point outside the interval while simultaneously including the other two inside.

It follows that:

$$
    \begin{cases}
        VCdim(\mathcal{H})\geqslant2\\
        VCdim(\mathcal{H})<3
    \end{cases}
    \Longrightarrow \ \ \boxed{VCdim(\mathcal{H})=2}
$$

#### Calculation for the rectangular function

Let's move to $\mathbb{R}^2$ and consider the following function dependent on four parameters:

$$
    h_{a_1,a_2,b_1,b_2}\left(x_1,x_2\right)=
    \begin{cases}
        1 \ \ \ a_1\leqslant x_1\leqslant a_2, \ b_1\leqslant x_2\leqslant b_2\\
        0 \ \ \ otherwise
    \end{cases}
$$

The class is therefore:

$$
    \mathcal{H}=\left\{h_{a_1,a_2,b_1,b_2}:a_1,a_2,b_1,b_2\in\mathbb{R},a_1\leqslant a_2,b_1\leqslant b_2\right\}
$$

As expected, there are examples up to $\abs{\mathcal{C}}=4$ in which $\mathcal{H}$ is able to shatter the set. For $\abs{\mathcal{C}}=5$, however, any set $\mathcal{C}$ is no longer shatterable by $\mathcal{H}$, since the most \"internal\" of the five cannot be classified as $0$ while the other four are classified as $1$.

It will therefore be:

$$
    \begin{cases}
        VCdim(\mathcal{H})\geqslant4\\
        VCdim(\mathcal{H})<5
    \end{cases}
    \Longrightarrow \ \ \boxed{VCdim(\mathcal{H})=4}
$$

#### Calculation for the sine ceiling function

In the examples seen so far, the VC dimension has coincided with the number of parameters useful for defining the functions within the hypothesis class. This, in reality, is not a general rule, and in this regard we will see an example below in which the two values are different.

Let's consider the ceiling function applied to the sine function modulated by a factor of $0.5$ and whose frequency is determined by a free parameter $\theta$:

$$
    h_\theta(x)=\lceil 0.5\sin(\theta x)\rceil
$$

We recall that the ceiling function returns the smallest integer greater than or equal to the input value; since the modulated sine oscillates between $-0.5$ and $0.5$, then the function $h_\theta(x)$ will alternately take only the values $0$ (when the sine is negative) and $1$ (when the sine is positive). It is possible to show that for any combination of points classified as $0$ and $1$, in whatever number they are, there exists a value of $\theta$ capable of determining a function in $\mathcal{H}$ that correctly describes this classification. In other words, the VC dimension of this class is infinite, despite the fact that the parameter on which it depends is only one.

## Theorem of statistical learning

What we have arrived at so far has been to state that if the VC dimension of a class is infinite, then it is certainly not PAC learnable, and we have also shown that a finite class (which we know to be PAC learnable) has a finite VC dimension.

There is still one type to explore, namely infinite classes with a finite VC dimension; the question is: what can we say about their PAC learnability?

We will see below an important theorem - not by chance called the *fundamental theorem* - which guarantees the PAC learnability of all classes with a finite VC dimension. We will thus have managed to reach our initial goal, which was to find a parameter that is directly linked to the PAC learnability of a class: it is, in fact, the VC dimension.

::: {.theorem name="Fundamental Theorem of Statistical Learning"}
Let $\mathcal{H}$ be a hypothesis class containing functions that go from a domain $\mathcal{X}$ to $\{0,1\}$ and let the loss function be a $0-1$ objective function.

Then, the following statements are equivalent:


    
    item $\mathcal{H}$ has the property of **uniform convergence**
    
    
    item Any ERM algorithm is an **agnostic PAC learner** for $\mathcal{H}$
    
    
    item $\mathcal{H}$ is **agnostic PAC learnable**
    
    
    
    item $\mathcal{H}$ is **PAC learnable**
    
    
    
    item Any ERM algorithm is a **PAC learner** for $\mathcal{H}$
    
    
    
    item $\mathcal{H}$ has **finite VC dimension**

:::

We do not give the proof in detail, also because most of them are obvious or have already been seen previously:

    
    item The implications $1\rightarrow2\rightarrow3$ derive from the theorem on the link between uniform convergence and PAC learning, which we saw in section @sec:convergenza-uniforme-e-pac-learnability
    
    
    item The implication $3\rightarrow4$ is obvious, since
    
    $$
        \underbrace{\underbrace{L_{\mathcal{D}}(h)\leqslant\epsilon}_{\text{PAC learning}}\leqslant\min_{h'\in\mathcal{H}}L_{\mathcal{D}}\left(h'\right)+\epsilon}_{\text{Agnostic PAC learning}}
    $$

    and the two properties coincide if the realizability assumption is valid (and therefore the minimum in the last member is zero)
    
    
    item For the same reason just seen, $2\rightarrow5$ also holds
    
    
    item By contradiction, if the infinite VC dimension for a hypothesis class implies its non-PAC learnability, then for a class to be PAC learnable it is necessary that it has a finite VC dimension, so we will have $4\rightarrow6$ and $5\rightarrow6$

To close the circle, it is necessary to show that $6\rightarrow1$. The proof is not required by the course, so it will not be given in detail; suffice it to say that it is divided into two parts:

    
    item Let $d$ be the VC dimension of $\mathcal{H}$, even if $\abs{\mathcal{H}}=\infty$, by applying the restriction of $\mathcal{H}$ to $\mathcal{C}$ it is shown that the "effective size" $\abs{\mathcal{H}_{\mathcal{C}}}$ is $O\left(\abs{\mathcal{C}}^d\right)$, so the size of $\mathcal{H}_{\mathcal{C}}$ grows *polynomially* with respect to $\abs{\mathcal{C}}$. This part of the proof is known as *Sauer's lemma*.
    
    
    item We know that finite classes enjoy the property of uniform convergence, but it is possible to generalize this result by stating that uniform convergence is valid for all those classes with "effective size" small, that is, with a polynomial and not exponential growth with respect to $\abs{\mathcal{C}}$.

### The link between VC dimension and sample complexity

The VC dimension characterizes not only PAC learnability but also the sample complexity associated with the various properties of the considered hypothesis class. This is demonstrated in the quantitative version of the fundamental theorem of statistical learning.

::: {.theorem name="Fundamental Theorem of Statistical Learning\ (quantitative version)"}
Let $\mathcal{H}$ be a hypothesis class containing functions that go from a domain $\mathcal{X}$ to $\{0,1\}$ and let the loss function be a $0-1$ objective function. Assume, moreover, that $VCdim(\mathcal{H})=d<\infty$.

Then, there exist constants $C_1$ and $C_2$ such that:


    
    item $\mathcal{H}$ enjoys uniform convergence with sample complexity
    
    $$
        C_1\frac{d+\log\left(\frac{1}{\delta}\right)}{\epsilon^2}\leqslant m_{\mathcal{H}}^{UC}(\epsilon,\delta)\leqslant C_2\frac{d+\log\left(\frac{1}{\delta}\right)}{\epsilon^2}
    $$

    
    item $\mathcal{H}$ is agnostically PAC learnable with sample complexity
    
    $$
        C_1\frac{d+\log\left(\frac{1}{\delta}\right)}{\epsilon^2}\leqslant m_{\mathcal{H}}(\epsilon,\delta)\leqslant C_2\frac{d+\log\left(\frac{1}{\delta}\right)}{\epsilon^2}
    $$

    
    item $\mathcal{H}$ is PAC learnable with sample complexity
    
    $$
        C_1\frac{d+\log\left(\frac{1}{\delta}\right)}{\epsilon}\leqslant m_{\mathcal{H}}(\epsilon,\delta)\leqslant C_2\frac{d\log\left(\frac{1}{\epsilon}\right)+\log\left(\frac{1}{\delta}\right)}{\epsilon}
    $$

:::

The proof is not required for this theorem either, but it is interesting to note how the ends of the interval are similar to the upper bound of the sample complexity seen in equation @eq:convergenza-uniforme-sample-complexity. The numerator, in particular, can be rewritten as

$$
    \log\left(\frac{2\abs{\mathcal{H}}}{\delta}\right)=\log(2\abs{\mathcal{H}})+\log\left(\frac{1}{\delta}\right)
$$

So it is clear that, in this context, it is as if the VC dimension of $\mathcal{H}$, i.e., $d$, were playing the role of $\log(2\abs{\mathcal{H}})$.
